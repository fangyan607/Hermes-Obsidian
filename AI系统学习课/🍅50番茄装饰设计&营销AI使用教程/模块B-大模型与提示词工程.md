## 🍅 番茄11：大模型工作原理（通俗版）

### 📖 费曼三句话

1. **大模型像一个"超级搜索+超级组合"引擎**——它不是真的"思考"，而是在它训练过的海量数据里，找到最符合你要求的文字组合。你问"北欧风客厅怎么设计"，它把所有关于北欧风客厅的文本碎片组合成一段通顺的回答。
2. **训练大模型就像教一个天才婴儿**：给它看互联网上几乎所有公开的文字（读书、看文章、看设计论坛），然后让它做"接龙游戏"——根据上文猜下文，猜错了就调整，反复千亿次，直到它学会"像人一样说话"。
3. **大模型的三个局限**：①它不知道"不知道"——胡说八道时也很自信（这叫"幻觉"）；②它没有审美——你觉得好看的设计它不一定懂；③它记不住最近的事——除非你告诉它。**知道它的局限，才能更好地用它**。

### 🛠 刻意练习

1. 给AI一个"陷阱问题"，观察它怎么反应：
   ```
   请推荐一种不存在的装修材料"碳纤维纳米墙布"的优缺点。
   ```
2. 再给它一个明确的问题：
   ```
   请推荐3种常见的客厅地面材料，对比优缺点和价格。
   ```
3. 对比两个回答，总结AI什么时候可靠、什么时候不可靠

### ✅ 完成标准

- [ ] 知道什么时候该相信AI、什么时候该查证
- [ ] 写下你对AI可靠性的判断标准

## 🍅 番茄13：提示词黄金框架——角色 + 任务 + 细节 + 格式

### 📖 费曼三句话

1. **好的提示词 = 给实习生写的任务说明书**——越具体越好。不要说"设计个客厅"，要说"一个20平米、采光一般、三口之家、预算3万的北欧风客厅改造方案"。
2. **黄金四要素缺一不可**：
   - 🎭 **角色**：你是谁？（"你是一个有10年经验的装修设计师"）
   - 📋 **任务**：你要AI做什么？（"请设计一个改造方案"）
   - 🔍 **细节**：关键约束条件（面积/预算/风格/人群）
   - 📤 **输出格式**：怎么呈现（表格/清单/分段/多少字）
3. **最常见的错误是"只给任务不给细节"**——就像跟设计师说"帮我设计个好看的家"，设计师无从下手。细节越多，AI少猜，结果越好。

### 🛠 刻意练习

**练习1**：把下面这个模糊提示词改写出4个版本，每个加一个新细节

```
原始：帮我写一个装修方案。
```

| 版本 | 追加细节 | 效果预期 |
|:----|:---------|:--------:|
| A | +角色（有10年经验的设计师） | 回答更专业 |
| B | +面积（80平米老房） | 回答更具体 |
| C | +预算（15万以内） | 更贴合实际 |
| D | +格式（用表格列出项目+费用） | 更清晰可用 |

**练习2**：用你改写的最好版本发给AI，看输出质量

### ✅ 完成标准

- [ ] 对比A和D的输出质量，写出你的观察
- [ ] 记住黄金四要素框架

## 🍅 番茄15：装饰行业提示词库建设（下）——预算与材料类

### 📖 费曼三句话

1. **预算编制是AI最擅长的**——只要你给出面积、项目和材料等级，AI能按市场均价快速生成预算表。关键是你告诉它"本地市场价"而非"全国均价"。
2. **材料选配提示词的关键**：给AI"约束条件"比给"想要的"更重要。"环保等级E0级，每平米不超过200元，耐污易清洁"比"我要好材料"精准10倍。
3. **AI预算的"人机协作"模式**：AI出初版 → 你根据本地实际价格调整 → 保存调整后的版本作为模板 → 下次AI在模板基础上微调。循环迭代，预算越来越准。

### 🛠 刻意练习

1. 用提示词模板生成一份预算：
   ```
   你是一个装修预算专家。请为以下项目编制预算：

   - 中等装修档次
   - 包含：拆除、水电、墙面、地面、厨卫、门窗
   - 请按项目分项列出人工费和材料费
   - 格式：表格，含项目/说明/数量/单价/总价
   ```
2. 手动调整至少3项价格（改成你了解的本地价格）
3. 保存调整后的版本为"预算模板"

## 🍅 番茄16：提示词高级技巧——少样本 + 链式思考

### 📖 费曼三句话

1. **少样本提示（Few-shot） = 给AI一个"参考答案"**——想让AI按你的风格回答，先给它2-3个例子。比如先给一个你写过的案例帖，再说"按这个风格再写一个"。AI会模仿你给的例子。
2. **链式思考（Chain-of-Thought） = 让AI"先想再说"**——在提示词里加一句"请先一步步分析，再给出答案"。这会让AI减少胡说八道，因为它在"思考"过程中会自我纠错。
3. **组合技最强大**：少样本保证风格一致 + 链式思考保证逻辑严谨 + 细节约束保证内容准确 = 90分的AI输出。

### 🛠 刻意练习

**对比实验**——同一个问题，三种提示词：

```
版本A（普通）：
写一个旧房翻新的案例帖。

版本B（少样本）：
参考以下案例帖风格，写一篇新的旧房翻新案例：
【案例】60平米老房改造，预算12万，北欧风格...
（你的真实或模拟案例）
请按这个风格再写一篇80平米老房翻新。

版本C（少样本+链式思考）：
参考以上案例风格，写一篇80平米老房翻新。
请先分析：1.这套房子的核心问题 2.改造重点 3.预算分配
然后输出完整案例帖。
```

**对比三个输出质量，写下你的发现**

## 🍅 番茄17：方案生成实战——从客户需求到初步方案

### 📖 费曼三句话

1. **用AI出方案的正确顺序**：先让AI帮你"问问题"而不是直接"给答案"——"我客户有20万预算翻新80平米老房，一对退休夫妇，我应该问他们什么问题才能给出好方案？"AI的问题清单会帮你想到你忽略的细节。
2. **AI方案是"初稿"，不是"终稿"**——告诉AI"输出一个能直接给客户看的版本"，然后你花10分钟做三件事：改掉AI说错的本地信息、加你的个人风格、删掉太空泛的套话。
3. **方案的核心价值不是"设计"，而是"沟通"**——客户看不懂设计图，但看得懂"这里改成这样，你早上起来会更方便"。AI写的专业术语要翻译成客户能听懂的话。

### 🛠 刻意练习

**模拟一个完整客户需求**：

```
客户：李先生，45岁
房子：90平米，20年房龄老房
家庭：夫妻+一个初中生儿子
预算：18-20万
需求：现代简约风格，重点改造客厅和厨房，
      儿子房间需要学习区，主卧增加储物。
```

1. 先用AI生成"问题清单"（问客户的问题）
2. 然后用AI生成初步方案
3. 最后用AI把方案改写成"给客户看的版本"

### ✅ 完成标准

- [ ] 跑通了"提问→方案→客户版"三步骤
- [ ] 方案包含至少3个具体的改造建议
- [ ] 客户版语言通俗易懂，没有专业术语

## 🍅 番茄18：预算编制实战——从方案到报价

### 📖 费曼三句话

1. **装饰报价的两个关键数字**：单价×数量=总价。AI擅长给你"市场参考单价"，但数量（面积/米数）你必须自己量或估算。AI的数量是猜的，你的数量是准的。
2. **报价的"三级精度"**：AI估算（±30%准）→ 你调整后（±15%准）→ 供应商报价后（±5%准）。第一次报价用AI足够了，但签合同前必须拿到供应商真实报价。
3. **最有价值的AI报价技巧**：告诉AI"请按低/中/高三档分别报价"——这样你能给客户三种选择。大部分客户会选中档，但你给他选择权比只给一个价好10倍。

### 🛠 刻意练习

**用番茄17的方案做预算**：

1. 把番茄17生成的方案发给AI，要求做三档预算：
   ```
   根据以下改造方案，请按经济档/标准档/品质档
   分别编制预算。
   [粘贴方案]
   ```
2. 检查：AI的三档差价是否合理（通常每档差30-50%）
3. 调整至少2项你觉得和本地价格不符的

### ✅ 完成标准

- [ ] AI生成了三档预算（经济/标准/品质）
- [ ] 三档之间有合理的梯度
- [ ] 你手动调整了至少2项本地化价格

## 🍅 番茄19：AI辅助客户沟通——话术与跟进

### 📖 费曼三句话

1. **AI话术的核心不是"替你说"，而是"帮你准备说"**——客户问"为什么比别家贵"，AI给你3种回答角度：讲材料差异、讲服务差异、讲长期价值。你选一个最适合这个客户的。
2. **常见客户异议的AI应对库**——提前把10个常见问题输入AI，让AI生成回答话术。整理成一个文档"客户常见问题应答手册"，需要时直接翻。
3. **AI跟进提醒 + 个性化消息**——CRM里记下客户关注的点和上次聊到的内容，让AI帮你写跟进消息。"王姐，上次聊到您喜欢北欧风，我最近刚做了一个类似案例，发您参考一下"——有温度、有针对性。

## 🍅 番茄20：模块B综合考核——从客户咨询到方案交付

### 📖 费曼三句话（模块B总结）

1. **提示词工程 = 装饰设计师的"手艺"**——同样的工具，不同的提示词水平，产出质量天差地别。这个手艺会越来越值钱。
2. **黄金四要素 + 少样本 + 链式思考 = 你的提示词工具箱**——不同场景用不同工具。方案生成用"角色+细节"，文案创作用"少样本"，分析类用"链式思考"。
3. **AI是你的"实习生"，你是"主案设计师"**——AI出初稿，你定方向、做决策、加细节。永远不要让AI直接给客户，你审核修改过的才是你的作品。
