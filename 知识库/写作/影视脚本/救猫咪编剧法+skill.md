# 救猫咪编剧法

> 用普通话语写的编剧指南——从高度概念化故事线到10种题材、15节拍结构表的实战编剧方法论。

## 核心理念

本书由卖出多部剧本并获百万美元稿酬的职业编剧布莱克·斯奈德撰写，核心观点：好剧本始于"电影讲什么"。通过高度概念化的故事线（含意外、心理影像、观众成本、杀伤力片名四要素）、10种题材分类和15节拍结构表，将编剧从艺术直觉转化为可操作、可验证的系统工程。"救猫咪"场面——让主角做一件好事使观众喜欢他——是编剧的基本常识。

---

## 触发条件

当用户请求以下任务时自动激活：
- 构思电影故事创意
- 写故事线（logline）
- 确定故事的题材类型
- 应用节拍表规划剧本结构
- 解决剧本创作中的常见问题

## 执行流程

### Phase 1: 构思故事线

**目标**：创作有杀伤力的故事线

**输入**：
- 初步创意或想法

**步骤**：

#### Step 1: 提炼意外元素

**动作**：在故事中找到意外（irony）元素

**标准**：
- 意外必须有情绪上的吸引力
- 像"鱼钩"一样勾起兴趣
- 例子：《虎胆龙威》——一个警察来看分居妻子，妻子办公室被恐怖分子占领

**输出**：`意外元素：[故事中的意外冲突]`

#### Step 2: 创造心理影像

**动作**：确保故事线能引发完整的心理影像

**标准**：
- 听到故事线就能"看到"整部电影
- 包含时间范围
- 引发连锁反应式的灵感

**输出**：`心理影像：[故事线引发的视觉想象]`

#### Step 3: 界定观众和成本

**动作**：明确目标观众和大致制片成本

**标准**：
- 四象限影片（面向所有观众）
- 中等成本 vs 高成本
- 从故事线元素推断成本

**输出**：`观众与成本：[目标观众 + 成本级别]`

#### Step 4: 设计杀伤力片名

**动作**：创作体现故事实质的片名

**标准**：
- 片名必须体现故事实质
- 包含意外元素
- 避免过于一般的名称

**输出**：`片名：[有杀伤力的片名]`

**检查点**：
- [ ] 意外元素已提炼
- [ ] 心理影像已创造
- [ ] 观众和成本已界定
- [ ] 杀伤力片名已设计

**输出**：完整故事线

---

### Phase 2: 市场调研

**目标**：验证故事线的市场吸引力

**输入**：
- 完整故事线

**步骤**：

#### Step 1: 执行街头测试

**动作**：向陌生人推销故事线

**标准**：
- 找各年龄段、各种场合的人
- 盯着对方眼睛观察反应
- 对方眼神飘忽=故事线有问题

**输出**：`测试结果：[正面/负面/需改进]`

#### Step 2: 分析反馈

**动作**：根据反馈调整故事线

**标准**：
- 识别不适当的场景
- 消除令人困惑的元素
- 强化吸引人的部分

**输出**：`调整方案：[基于反馈的修改]`

**检查点**：
- [ ] 已测试≥5个陌生人
- [ ] 能始终吸引注意力
- [ ] 对方想更多了解故事

**输出**：验证后的故事线

---

### Phase 3: 题材分类

**目标**：将故事归入10种题材之一

**输入**：
- 验证后的故事线

**步骤**：

#### Step 1: 识别题材类型

**动作**：从10种题材中选择最匹配的一种

**标准**——10种题材：
1. **"鬼怪屋"**：怪兽+封闭空间+原罪（《大白鲨》《异形》）
2. **"金羊毛"**：旅程+寻宝+自我发现（《星球大战》《绿野仙踪》）
3. **"如愿以偿"**：魔法/奇迹+变化+教训（《大话王》《冒牌天神》）
4. **"麻烦家伙"**：普通人+突发危机+生存（《虎胆龙威》《泰坦尼克号》）
5. **"变迁仪式"**：生活变迁+成长烦恼（《克莱默夫妇》《普通人》）
6. **"伙伴之情"**：搭档关系+爱情/友情（《雨人》《当哈利遇上莎莉》）
7. **"推理侦探"**：侦探+罪恶+真相（《唐人街》《冰血暴》）
8. **"愚者成功"**：凡人+跳龙门（《阿甘正传》《律政俏佳人》）
9. **"被制度化"**：个人vs集体（《飞越疯人院》《教父》）
10. **"超级英雄"**：特殊生物+小人国（《超人》《角斗士》）

**输出**：`题材类型：[10种之一]`

#### Step 2: 研究同类影片

**动作**：观看同题材经典影片

**标准**：
- 了解该题材的规则
- 学习成功之处
- 汲取失败教训
- 找到"似而不同"的纠葛

**输出**：`同类影片分析：[成功要素 + 可改进点]`

**检查点**：
- [ ] 题材已分类
- [ ] 同类影片已研究
- [ ] 题材规则已理解

**输出**：题材定位报告

---

### Phase 4: 应用15节拍结构表

**目标**：用15节拍表规划剧本结构

**输入**：
- 题材定位报告
- 故事线

**步骤**：

#### Step 1: 填写节拍表

**动作**：按15个节拍规划剧本

**标准**——15节拍（基于110页剧本）：

| 节拍 | 页码 | 功能 |
|------|------|------|
| 1. 开场画面 | 1 | 基调、类型铺垫 |
| 2. 主题呈现 | 5 | 某人告诉主人公主题 |
| 3. 铺垫 | 1-10 | 介绍A故事所有角色 |
| 4. 推动（催化剂） | 12 | 震撼主人公的事件 |
| 5. 争论 | 12-25 | 主人公犹豫不决 |
| 6. 第二幕衔接点 | 25 | 进入第二幕，旅程开始 |
| 7. B故事 | 30 | 爱情故事/主题讨论 |
| 8. 游戏 | 30-55 | "影片许下的诺言" |
| 9. 中点 | 55 | 气氛紧张，倒计时开始 |
| 10. 坏人逼近 | 55-75 | 内外压力增大 |
| 11. 一无所有 | 75 | 人为失败，死亡气息 |
| 12. 灵魂黑夜 | 75-85 | 失去所有希望 |
| 13. 第三幕衔接点 | 85 | B故事带来新灵感 |
| 14. 结局 | 85-110 | 两个世界的综合 |
| 15. 终场画面 | 110 | 与开场呼应，变化已发生 |

**输出**：完整节拍表

#### Step 2: 填充具体内容

**动作**：将故事内容填入节拍表

**标准**：每个节拍必须有具体的情节内容

**输出**：`填充后的节拍表：[具体内容]`

**检查点**：
- [ ] 15个节拍已填写
- [ ] 页码分配合理
- [ ] 每个节拍有具体内容
- [ ] 开场与终场呼应

**输出**：剧本结构蓝图

## 决策规则

| 条件 | 行动 | 说明 |
|------|------|------|
| 故事线太平淡 | 加入意外元素 | 意外是故事线的鱼钩 |
| 不确定题材 | 研究最接近的同类影片 | 10种题材覆盖所有故事 |
| 结构卡住 | 检查节拍表 | 通常是某个节拍缺失或错位 |
| 观众不喜欢主角 | 加"救猫咪"场面 | 让主角做一件好事 |
| 静止=死亡 | 给主人公变化的理由 | 没有变化就没有故事 |

## 示例

### 示例：构思一个"麻烦家伙"型故事

**输入**：构思一个关于普通人在危机中生存的故事

**执行过程**：
1. Phase 1: 故事线→"一个单亲父亲在航班上发现恐怖分子计划劫机，他必须在不让任何人发现的情况下阻止灾难"
2. Phase 2: 测试→陌生人反应积极，要求更多细节
3. Phase 3: 题材→"麻烦家伙"型，同类影片《虎胆龙威》《天地大冲撞》
4. Phase 4: 节拍表→15节拍填充完整结构

**输出**：
```
片名：《万米高空》
故事线：一个单亲父亲在跨洋航班上发现恐怖分子计划劫机，他必须在不让任何人发现的情况下，在燃料耗尽前阻止灾难。
题材：麻烦家伙型
核心节拍：
- 开场：父亲送女儿上学，承诺不缺席她的演出
- 推动：在洗手间听到恐怖分子密谈
- 中点：被发现，成为目标
- 一无所有：女儿在另一架航班上
- 结局：拯救全机乘客，赶上女儿演出
```

## 局限性

- **不适用于**：艺术片、实验电影、非叙事性作品
- **依赖条件**：需要创作者愿意接受结构化方法
- **可能失效**：当过度依赖模板而忽视故事独特性时
- **信息截止**：基于原书内容，核心理念和结构方法具有持久价值
