# 镜头语法体系

> "说"全球通用的视觉语言，把故事传递给世界观众——汇集近百种镜头语法细则，辅以近150幅插图。

## 执行流程

### Phase 1: 景别选择

**目标**：为每个镜头选择最合适的景别

**输入**：
- 场景描述
- 叙事需求

**步骤**：

#### Step 1: 确定信息需求

**动作**：判断观众需要看到什么信息

**标准**——9种景别的功能：
- **超大远景**：表现地点、时间、规模
- **全远景**：定场镜头，人物清晰但环境为主
- **远景/广角**：全身镜头，人物突出但保留环境
- **中远景**：膝部镜头，牛仔镜头
- **中景**：腰部镜头，人物交流
- **中近景**：两粒扣子，最常用
- **特写**：头像镜头，面部细节
- **大特写**：面部充满画幅
- **极特写**：纯细节镜头

**输出**：`景别选择：[景别] + [信息需求理由]`

#### Step 2: 考虑叙事节奏

**动作**：根据叙事节奏选择景别序列

**标准**：
- 远景→中景→特写：渐进式聚焦
- 特写→中景→远景：渐进式展开
- 混合使用创造节奏变化

**输出**：`景别序列：[景别 progression]`

**检查点**：
- [ ] 信息需求已确定
- [ ] 景别已选择
- [ ] 叙事节奏已考虑

**输出**：景别方案

---

### Phase 2: 构图设计

**目标**：应用构图法则优化画面

**输入**：
- 景别方案
- 被摄对象信息

**步骤**：

#### Step 1: 应用基本构图法则

**动作**：应用四大基本构图法则

**标准**：
- **头顶留白**：紧镜头不留太多头顶空间，松镜头适中留白
- **视线留白**：人物视线方向留更多空间
- **九宫格定律**：将重要元素放在交叉点上
- **拍摄方式**：主观（直视镜头）vs 客观（不直视）

**输出**：`构图方案：[头顶留白 + 视线留白 + 九宫格位置]`

#### Step 2: 设计摄影角度

**动作**：选择摄影机角度

**标准**：
- **水平角度**：正面、侧面、3/4角度
- **垂直角度**：平视（中性）、高角度（弱化）、低角度（强化）、鸟瞰/虫视

**输出**：`摄影角度：[水平 + 垂直角度]`

#### Step 3: 设计双人镜头

**动作**：为两人场景设计构图

**标准**：
- **侧面双人镜头**：两人并排侧对摄影机
- **直接面向摄影机**：两人面对镜头
- **过肩双人镜头**：一人肩部入画

**输出**：`双人构图：[构图类型]`

**检查点**：
- [ ] 头顶留白已检查
- [ ] 视线留白已安排
- [ ] 九宫格已应用
- [ ] 摄影角度已选择
- [ ] 双人构图已设计

**输出**：构图方案

---

### Phase 3: 构图进阶

**目标**：应用进阶构图技巧

**输入**：
- 构图方案
- 场景复杂度

**步骤**：

#### Step 1: 创造第三维

**动作**：在二维画面中创造深度感

**标准**：
- **地平线**：控制地平线位置
- **荷兰角**：倾斜画面制造不安
- **斜线法**：用斜线引导视线
- **前景/中景/后景**：利用电影空间深度

**输出**：`深度设计：[深度线索]`

#### Step 2: 选择镜头和焦点

**动作**：选择镜头焦距和焦点策略

**标准**：
- **广角镜头**：夸张深度，适合环境
- **标准镜头**：接近人眼视角
- **远摄镜头**：压缩深度，适合特写
- **焦点引导**：用焦点引导观众眼睛

**输出**：`镜头选择：[焦距 + 焦点策略]`

#### Step 3: 设计光线

**动作**：设计场景光线方案

**标准**——三点布光法：
- **主光**：主要照明，塑造形态
- **补光**：减弱阴影，控制对比度
- **轮廓光**：分离主体与背景

**其他要素**：
- 色温（暖/冷）
- 自然光或人工光
- 光的性质（硬光vs柔光）
- 对比度控制

**输出**：
```
光线方案：
- 主光：[方向 + 强度 + 色温]
- 补光：[方向 + 强度]
- 轮廓光：[方向 + 强度]
- 对比度：[高/中/低]
```

**检查点**：
- [ ] 第三维已创造
- [ ] 镜头焦距已选择
- [ ] 三点布光已设计
- [ ] 色温和对比度已确定

**输出**：进阶构图方案

### Phase 4: 连续性与运动

**目标**：确保镜头间的连续性并设计运动

**输入**：
- 进阶构图方案
- 场景动作需求

**步骤**：

#### Step 1: 应用连续性规则

**动作**：确保镜头间的视觉连续性

**标准**：
- **180°规则**：摄影机保持在动作轴线一侧
- **30°定律**：相邻镜头角度差≥30°
- **匹配角度**：相似角度需足够差异
- **视线匹配**：角色视线方向一致
- **动作连贯**：动作在剪辑点匹配

**输出**：`连续性检查：[规则应用结果]`

#### Step 2: 设计摄影机运动

**动作**：设计运动镜头

**标准**：
- **手持摄影**：纪实感、紧张感
- **横摇和俯仰摇摄**：跟随动作、重新构图
- **三脚架**：稳定画面
- **移动摄影车**：平滑运动
- **升降车**：垂直运动

**输出**：`运动设计：[运动类型 + 设备 + 路径]`

**检查点**：
- [ ] 180°规则已遵守
- [ ] 30°定律已遵守
- [ ] 视线匹配已检查
- [ ] 动作连贯已检查
- [ ] 运动镜头已设计

**输出**：连续性与运动方案

## 决策规则

| 条件 | 行动 | 说明 |
|------|------|------|
| 需要建立场景 | 使用超大远景/全远景 | 定场镜头建立时空 |
| 需要情感亲密 | 使用特写/大特写 | 拉近与观众的距离 |
| 需要表现交流 | 使用中景/中近景 | 最接近日常观看距离 |
| 需要不安感 | 使用荷兰角 | 倾斜画面制造不安 |
| 需要强化角色 | 使用低角度 | 低角度使角色显得强大 |
| 需要弱化角色 | 使用高角度 | 高角度使角色显得弱小 |
| 越轴需求 | 用中性镜头过渡 | 不可直接跳轴 |

## 示例

### 示例：设计一个紧张对峙场景

**输入**：设计两个角色在走廊对峙的场景

**执行过程**：
1. Phase 1: 选择中远景（建立关系）→特写（情感）→极特写（紧张细节）
2. Phase 2: 应用低角度强化角色、荷兰角制造不安、视线留白指向对方
3. Phase 3: 高对比度硬光、深景深保持双方清晰
4. Phase 4: 遵守180°规则、设计缓慢推轨镜头逼近
5. Phase 5: 应用重叠动作拍摄、多留时间、保证眼神光

**输出**：
```
对峙场景镜头设计：
1. 中远景（固定）：建立空间关系，低角度，硬光高对比
2. 角色A特写（固定）：低角度，眼神光，视线留白向右
3. 角色B特写（固定）：低角度，眼神光，视线留白向左
4. 极特写A的手（固定）：紧张细节
5. 推轨镜头：缓慢推进至双人中景，荷兰角5°
```

- **可能失效**：当现场条件限制（光线、空间、设备）无法执行理想方案时
