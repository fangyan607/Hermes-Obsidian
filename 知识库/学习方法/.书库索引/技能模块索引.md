## 二、核心技能详解

### Skill1: 时间管理技能

**触发指令**：时间管理、日程规划、时间分配、创造时间

**调取书籍**：《创造时间》《极简时间》《番茄工作法》

**核心能力**：

| 能力 | 说明 |
|------|------|
| 时间审计 | 分析时间去向，识别时间黑洞 |
| 优先级排序 | 区分重要/紧急，聚焦高价值任务 |
| 时间块规划 | 固定时段专注，创造深度工作时间 |
| 番茄钟执行 | 25分钟专注+5分钟休息的节奏 |

**时间分配公式**：

```
每日可用时间 = 24小时 - 睡眠(7h) - 必需活动(3h) = 14小时

学习时间分配建议：
├── 深度学习（高认知负荷）：2-4小时/天
├── 浅层学习（复习/阅读）：2-3小时/天
├── 实践练习：2-3小时/天
└── 休息/运动：2-3小时/天
```

**用户时间窗口适配**：

| 时间段 | 时长 | 适用任务类型 |
|--------|------|--------------|
| 上午 8:00-10:00 | 2小时 | 深度学习（新知识吸收） |
| 中午 12:00-16:00 | 4小时 | 实践练习 + 项目开发 |
| 晚上 18:00-21:00 | 3小时 | 复习巩固 + 轻量学习 |
| **每日总计** | **9小时** | 完整学习周期 |

### Skill2: 刻意练习技能

**触发指令**：刻意练习、技能学习、技能提升、高效训练

**调取书籍**：《刻意学习》

**核心能力**：

| 能力 | 说明 |
|------|------|
| 目标设定 | 明确具体、可衡量的学习目标 |
| 专注练习 | 高度专注的单点突破 |
| 即时反馈 | 练习后立即获取反馈 |
| 纠正迭代 | 根据反馈调整，持续改进 |
| 舒适区突破 | 在学习区练习，避免舒适区/恐慌区 |

**刻意练习四要素**：

```
┌─────────────────────────────────────────────────────────────┐
│                    刻意练习四要素                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 明确目标 ──→ "学会Hermes Agent技能系统"                  │
│       ↓                                                     │
│  2. 专注练习 ──→ 每个技能点单独练习（如Skill安装、记忆配置）  │
│       ↓                                                     │
│  3. 即时反馈 ──→ 运行命令→看结果→判断是否成功                │
│       ↓                                                     │
│  4. 纠正迭代 ──→ 失败→查文档→修正→再练习                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**学习进度追踪模板**：

```markdown
## [技能名称] 学习进度

### 已完成
- [ ] 基础概念理解
- [ ] 第一次实践
- [ ] 第一次成功执行

### 进行中
- [ ] 当前任务

### 待改进
- [ ] 发现的问题
- [ ] 需要深入的点
```

### Skill3: 深度工作技能

**触发指令**：深度工作、专注力、心流状态、全神贯注

**调取书籍**：《全神贯注的方法》《精英都是细节控》

**核心能力**：

| 能力 | 说明 |
|------|------|
| 专注环境设计 | 减少干扰，创造深度工作空间 |
| 专注力训练 | 冥想、单任务练习 |
| 心流触发 | 找到挑战与能力的平衡点 |
| 精力管理 | 高峰时段做高价值任务 |

**深度工作四原则**：

```
原则1: 工作要深入
       → 固定时间、固定地点、固定仪式

原则2: 拥抱无聊
       → 训练注意力，减少分心习惯

原则3: 远离社交媒体
       → 设定使用边界，避免碎片化

原则4: 摒弃浅薄
       → 优先深度工作，压缩浅层事务
```

### Skill5: 高维思考技能

**触发指令**：高维思考、发现问题、系统思维、认知升级

**调取书籍**：《高维度思考法》《聪明人是如何思考的》

**核心能力**：

| 能力 | 说明 |
|------|------|
| 问题重构 | 从"如何解决"到"问题是什么" |
| 维度升级 | 跳出当前维度，更高视角审视 |
| 模式识别 | 发现重复模式，提炼规律 |
| 第一性原理 | 追问本质，从根本出发 |

**思考维度升级**：

```
Level 1: 解决问题 ──→ "Hermes怎么安装？"
Level 2: 优化方案 ──→ "哪种安装方式最适合我？"
Level 3: 发现问题 ──→ "我真正需要Hermes解决什么问题？"
Level 4: 重构问题 ──→ "有没有更好的工具/方法解决我的问题？"
Level 5: 系统整合 ──→ "Hermes如何融入我的知识管理系统？"
```

## 三、技能组合应用

### 完整学习流程

```
┌─────────────────────────────────────────────────────────────┐
│                    学习方法组合应用                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Step1: 高维思考 ──→ 明确学习目标与真正需求                  │
│          ↓                                                  │
│  Step2: 时间管理 ──→ 规划学习时间窗口                        │
│          ↓                                                  │
│  Step3: 清单系统 ──→ 拆解学习任务，制作清单                  │
│          ↓                                                  │
│  Step4: 刻意练习 ──→ 专注练习，获取反馈，迭代改进            │
│          ↓                                                  │
│  Step5: 深度工作 ──→ 在固定时段进入心流状态                  │
│          ↓                                                  │
│  Step6: 金字塔原理 ──→ 结构化输出学习成果                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 场景 | 推荐技能组合 |
|------|--------------|
| 新技能学习 | 高维思考 + 刻意练习 + 金字塔原理 |
| 项目规划 | 时间管理 + 清单系统 + 高维思考 |
| 深度研究 | 深度工作 + 金字塔原理 + 刻意练习 |
| 日常学习 | 时间管理 + 清单系统 + 刻意练习 |
