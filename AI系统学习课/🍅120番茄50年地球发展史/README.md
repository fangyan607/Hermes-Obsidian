# 🌍 120番茄·50年地球发展史

> **从中国的视角，看世界如何走到今天，又将走向何方。**
> 融合「番茄工作法×费曼学习法×刻意练习」，以「人类命运共同体」为蓝图，以「世界村民」为人设，系统学习2001-2050年全球格局演变。

---

## 📖 课程总览

### 为什么要学这门课？

当今世界正经历**百年未有之大变局**。美伊战争、中美战略稳定框架、俄乌和谈窗口、AI革命、全球南方崛起……每一件大事都在重塑我们生活的世界。

但大多数人对世界的理解是**碎片化**的——刷新闻得到的是"点"，看评论得到的是"情绪"，真正缺少的是**一条贯穿50年的叙事线**。

这门课要做的事：**用一个框架、一套方法、一群角色，帮你把50年的世界发展史装进脑子里。**

### 三个核心设计

| 设计 | 说明 |
|:----|:------|
| **中国视角** | 以"人类命运共同体"为价值坐标，以中国为观察原点，不偏袒但不假装中立 |
| **世界村民** | 每个国家都是性格鲜明的村民——美国是"前村长"，俄罗斯是"熊脾气猎户"，欧洲是"体面的乡绅"……记住人设就记住了国家逻辑 |
| **番茄×费曼×刻意练习** | 时间结构（番茄钟）+ 概念内化（费曼自测）+ 技能固化（刻意练习）= 真正的认知升级 |

### 学习成果

完成120🍅（24天）后，你将获得：

1. **一张50年的世界认知地图**——2001到2050，重大事件、关键转折、底层逻辑，尽在掌握
2. **一套分析框架**——地缘政治、科技变革、发展模式、文明叙事四维分析模型
3. **一个可迁移的学习方法论**——番茄×费曼×刻意练习的组合技能，可复用于任何领域
4. **对"人类命运共同体"的深度理解**——知其然，知其所以然，知其将然

## 🍅 每日学习节奏（5番茄 = 约2.5小时）

每个Day的结构完全相同，遵循"悬疑导入→核心叙事→角色演绎→刻意练习→费曼输出"的五步学习法：

```
===== 🍅 T1（25分钟）：悬疑钩子与导入 =====
  ⏱ 0-5min   悬疑开场："你知道吗？……"（制造认知缺口）
  ⏱ 5-20min  核心概念讲解（带着问题学）
  ⏱ 20-25min 费曼自测#1："用一句话回答今天的核心问题"

===== 🍅 T2（25分钟）：核心叙事 =====
  ⏱ 0-20min  深度叙事（事件、逻辑、数据、地图）
  ⏱ 20-25min 费曼自测#2："画出/说出今天的因果链"

===== 🍅 T3（25分钟）：世界村民角色演绎 =====
  ⏱ 0-15min  村民角色深度刻画（动机·底牌·软肋·盘算）
  ⏱ 15-25min 角色代入练习："如果你是[角色]，你会怎么做？"

===== 🍅 T4（25分钟）：刻意练习 =====
  ⏱ 0-20min  三层循环练习（模仿→变式→创造）
  ⏱ 20-25min 自检清单 + 费曼自测#3

===== 🍅 T5（25分钟）：费曼输出 × 预习钩子 =====
  ⏱ 0-10min  费曼输出："用最简单的语言向别人讲今天学到的内容"
  ⏱ 10-20min 认知升级笔记（今天的知识如何连接已有认知）
  ⏱ 20-25min 预习钩子："明天将看到一个更惊人的真相……"
```

## 🛠️ 学习方法指南

### 番茄工作法（Pomodoro）在本课程中的应用

```
┌─────────────────────────────────────────┐
│           1个番茄 = 25分钟               │
│  ├─ 专注学习：20分钟                    │
│  └─ 休息复盘：5分钟（含费曼自测）        │
│                                         │
│           5个番茄 = 1个Day              │
│  ├─ 每番茄后休息5分钟                    │
│  ├─ 第3番茄后休息15分钟（大休息）        │
│  └─ 完成后 + 今日总结                    │
│                                         │
│           24天 = 120番茄                │
│  ├─ 每天约2.5小时                        │
│  └─ 建议节奏：每周6天，4周完成           │
└─────────────────────────────────────────┘
```

### 费曼学习法（Feynman Technique）在本课程中的应用

每番茄末尾的**费曼自测**是课程设计的核心创新。四种自测类型：

| 类型 | 占比 | 示例 | 出现位置 |
|:-----|:----|:-----|:--------|
| **复述型** | 40% | "用一句话解释什么是人类命运共同体" | T1/T5末尾 |
| **对比型** | 25% | "G7和金砖机制有什么区别？" | T2末尾 |
| **演绎型** | 25% | "如果你是印度总理，在中美之间怎么选？" | T3末尾 |
| **预测型** | 10% | "基于现有趋势，2035年世界格局会怎样？" | T4末尾 |

### 刻意练习（Deliberate Practice）在本课程中的应用

每个Day的T4是刻意练习模块，采用"三循环"设计：

```
===== 循环1（模仿）：=====
  照着地图/数据/框架走一遍
  → 目标：建立基线认知
  
===== 循环2（变式）：=====
  换个角度/角色/假设再分析一次
  → 目标：理解可迁移性
  
===== 循环3（创造）：=====
  用学到的框架预测或解释一个真实事件
  → 目标：综合运用
```
