# 英雄之旅

> **一句话定义**：一个跨文化的叙事原型模式，描述英雄从平凡世界进入非凡世界，经历考验后带着"灵药"回归的旅程。

## 12个阶段

### 第一幕：出发

| 阶段 | 描述 | 电影示例 |
|:---|:----|:--------|
| 1. 平凡世界 | 英雄在日常环境中 | 弗罗多在夏尔 |
| 2. 冒险召唤 | 打破平静的事件 | 甘道夫通知魔戒 |
| 3. 拒绝召唤 | 英雄犹豫不决 | 弗罗多想逃避 |
| 4.  mentor） | 导师出现 | 甘道夫引导 |

### 第二幕：入门

| 阶段 | 描述 | 电影示例 |
|:---|:----|:--------|
| 5. 跨越门槛 | 进入非凡世界 | 离开夏尔 |
| 6. 考验、盟友、敌人 | 在陌生世界摸索 | 遇到阿拉贡等 |
| 7. 接近深穴 | 面对最大挑战前夕 | 接近魔多 |
| 8. 严峻考验 | 核心危机 | 在末日火山前 |

### 第三幕：回归

| 阶段 | 描述 | 电影示例 |
|:---|:----|:--------|
| 9. 报酬 | 获得"灵药" | 摧毁魔戒 |
| 10. 回归之路 | 返回平凡世界 | 回家之路 |
| 11. 复活 | 最后考验 | 与萨鲁曼对决 |
| 12. 带着灵药回归 | 英雄归来并分享智慧 | 离开中土 |

## 核心模型

```
平凡世界 ──→ 冒险召唤 ──→ 拒绝 ──→ 导师
                │
            mentor 引导
                │
                ▼
        跨越门槛 ──→ 考验 ──→ 深穴 ──→ 严峻考验
                │
                ▼
        报酬 ──→ 回归 ──→ 复活 ──→ 携灵药回归
```

## 人物原型
- **英雄**：推动故事的主角
- **导师**：指引英雄的智者（可去世推动英雄成长）
- **信使**：带来召唤
- **门槛守护者**：阻止英雄进入新世界
- **变形者**：立场不确定的角色
- **阴影**：反派/反派的内在投射
- **捣蛋鬼**：打破平衡的喜剧角色

## 创作技巧
1. **反转结构**：打破12步顺序，制造惊喜
2. **缺失步骤**：有意省略某些阶段（如反对导师）
3. **双重旅程**：两条英雄之旅并行
4. **内心旅程**：外部事件映射内在成长

## 费曼解释
> "英雄之旅就是'离开舒适圈→经历困难→成长后回来'的故事版。每个人的人生都有英雄之旅——搬家、换工作、恋爱分手，都是小型英雄之旅。"

## 应用场景
- 长篇小说结构
- 电影剧本（特别是类型片）
- 游戏剧情设计
- 个人品牌故事
```

# AIDA模型

## 定义
**AIDA**是广告文案的核心公式：**Attention（注意）→ Interest（兴趣）→ Desire（欲望）→ Action（行动）**

## 四步详解

### A - 吸引注意 (Attention)
头部标题和第一句话必须在**3秒内**抓住读者。

**技巧**：
- 反常识："你知道吗，90%的人..."
- 强力宣言："2025年最被低估的技能"
- 具体数字："3个方法让你..."
- 引发好奇："我用这个方法赚到了第一个100万"

### I - 激发兴趣 (Interest)
让读者觉得"这和我有关"。

**技巧**：
- 讲一个简短的故事或场景
- 描述读者正在经历的痛苦
- 展示一个令人向往的结果

### D - 激发欲望 (Desire)
让读者"想要"。

**技巧**：
- 展示具体收益（不是"更健康"而是"腰围减5cm"）
- 社会证明（"已有10万人..."）
- 稀缺性（"仅限今天"）

### A - 促使行动 (Action)
一个清晰的CTA。

**技巧**：
- 明确的动词："点击"、"订阅"、"购买"
- 降低门槛："免费试用7天"
- 消除风险："不满意全额退款"

## 应用模板
```
【A】震撼性标题
第一段：场景切入，让读者感觉"这就是我"
【I】详细描述问题+解决方案
列举3个核心价值点
【D】客户见证/案例数据
展示预期效果
【A】CTA按钮/链接
```

## 费曼解释
> "AIDA就是'看我→感兴趣→想要→行动'的节奏。好的广告像在约会——先引起注意，再建立吸引，最后约出去。"
```

  # 核心能力
  1. **故事构思**：从灵感出发，发展出完整的故事框架
  2. **结构设计**：运用三幕剧、英雄之旅等结构搭建故事骨架
  3. **人物塑造**：设计有深度、有弧光的人物
  4. **情节编排**：编织冲突、悬念和高潮
  5. **风格模仿**：根据要求调整文风和语气
  
  # 协作模式
  你是**共创者**而非替代者——你提供建议、框架和优化，但核心创意由用户决定。
  
  # 工作流程
  ## 阶段一：构思（头脑风暴）
  - 引导用户确定：
    - 类型（悬疑/奇幻/科幻/文学/...）

    - 主题（成长/复仇/救赎/...）
    - 目标读者
  
  ## 阶段二：结构搭建
  - 推荐适合的故事结构
  - 设计关键情节点
  - 创建章节/场景大纲
  
  ## 阶段三：人物设计
  - 核心人物档案（动机+缺陷+弧光）

  ## 阶段四：创作与修改

  [提供2-3个选择，让用户决策]

### 25.1 创作工作流设计

```
完整创作流程：

用户需求 → 故事Agent结构设计 → 剧本Agent场景拆分
                                              ↓
                         文案Agent  ← 短内容创作  ← 完成初稿
                              ↓
                         推广文案 + 社交媒体内容
```

## 场景1：游戏叙事设计

- **script-writer**：对话树 + 过场CG脚本
