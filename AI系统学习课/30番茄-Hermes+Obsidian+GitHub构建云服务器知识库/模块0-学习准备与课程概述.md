# 🍅 模块0：学习准备与课程概述

> ⏱ **预估时间**：25分钟（0.5番茄）
> 📖 **学习目标**：了解课程全貌、确认前置条件、准备所需工具
> 🧠 **费曼核心**：能用一句话回答"这门课能帮我解决什么问题？"

---

## 1️⃣ 这门课能解决什么问题？

**核心痛点**：
- ❌ 本地笔记无法随时随地访问
- ❌ 想用AI知识库但不会部署服务器
- ❌ 多个设备间笔记同步混乱
- ❌ 想搭建专属AI Agent但无从下手

**解决方案**：
```
本地 Obsidian ✕ GitHub 同步 ✕ 云服务器 Hermes = 
✅ 随时随地访问的AI知识库
✅ 多个设备无缝同步
✅ 专属AI Agent 7×24小时在线
```

---

## 2️⃣ 课程核心理念

### 三层架构

```
┌─────────────────────────────────┐
│         Layer 3: AI Agent       │
│     Hermes 心理咨询/法律/写作    │
├─────────────────────────────────┤
│         Layer 2: 知识库          │
│    Obsidian LLM-WIKI 结构仓库    │
├─────────────────────────────────┤
│         Layer 1: 同步层          │
│ 本地Obsidian ↔ GitHub ↔ 服务器  │
├─────────────────────────────────┤
│         Layer 0: 基础设施        │
│    阿里云 Ubuntu 24.04 + Git    │
└─────────────────────────────────┘
```

### 数据流

```
你写笔记 → Obsidian保存 → Git提交 → push到GitHub
                                   ↓
服务器 pull ← GitHub ←（其他设备也可pull）
    ↓
Hermes 读取笔记 → 用户提问 → AI回答（基于你的知识库）
```

## 5️⃣ 学习建议

### 费曼准备

**课前预热**：用你自己的话回答以下问题（写下来或说出来）：
1. "AI知识库是什么？它和普通搜索引擎有什么区别？"
2. "为什么要用Obsidian管理知识库？而不是Notion或Confluence？"
3. "Agent是什么？它和ChatGPT有什么不同？"

### 刻意练习计划

**基础路径**：按顺序学完30个番茄即可
**进阶路径**：学完基础后，选择1个实战模块深入（心理/法律/写作三选一）
**创作路径**：学完所有内容后，尝试构建你自己的专业知识库（投资、编程、医学等）
