# 模块0：基础入门与界面

> **预估时间**：5个番茄钟（每个25分钟）
> **目标**：从零安装 Blender 到独立完成一个低多边形模型
> **版本**：Blender 5.1

---

## 🍅1 安装部署与配置（25分钟）

### 费曼输入（概念解释）

Blender 是目前最强大的**免费开源 3D 创作套件**，支持建模、雕刻、材质、动画、渲染、合成、视频编辑全流程。5.1 版本于 2026 年 3 月发布，基于 Python 3.13，几何节点新增 Bone Info / String to Curves / UV Unwrap with SLIM，Grease Pencil 重构了渲染管线，Winter of Quality 修复了 350+ 问题。

**.blend 文件**是 Blender 的"快照式"保存格式——场景中的所有物体、材质、动画数据、甚至操作历史都打包在一个文件里（类似 Photoshop 的 .psd）。

### 刻意练习（具体操作步骤 + 代码/命令）

**Step 1：下载 Blender 5.1**
- 官网：https://www.blender.org/download/
- 根据操作系统选择对应版本

**Step 2：安装流程**

| 系统 | 安装方式 | 说明 |
|------|----------|------|
| Windows | 运行 `.msi` 安装包 | 建议默认路径，自动关联 .blend 文件 |
| macOS | 打开 `.dmg`，拖入 Applications | 首次打开可能需 Gatekeeper 放行 |
| Linux | 下载 `.tar.xz`，解压即用 | 解压到 `/opt/blender`，添加软链接到 `/usr/local/bin` |

```bash
# Linux 解压示例
tar -xf blender-5.1.0-linux-x64.tar.xz
sudo mv blender-5.1.0-linux-x64 /opt/blender
sudo ln -s /opt/blender/blender /usr/local/bin/blender
```

**Step 3：首次启动配置**
1. 启动 Blender → 自动弹出 Splash 屏幕
2. 语言设置：左下角可以选择语言（简体中文）
3. 主题：Edit → Preferences → Themes → 切换暗色/亮色
4. 选择键位预设：Blender 默认（右键选择）或 Industry Compatible（左键选择，类似 Maya/Max）

**Step 4：插件管理**
- Edit → Preferences → Add-ons
- 搜索框中输入插件名称
- 勾选启用，例如：`Node Wrangler`（材质节点必备）、`3D-Print Toolbox`
- 插件目录：Windows: `%APPDATA%\Blender Foundation\Blender\5.1\scripts\addons`

**Step 5：项目文件操作**
- `Ctrl + N` = 新建项目
- `Ctrl + S` = 保存（第一次保存会自动以 Blender 5.1 格式存储）
- 文件格式：`.blend`（自动压缩，可设置压缩级别）
- 打开 .blend：File → Open 或直接拖入

- [ ] 完成语言和主题配置
- [ ] 启用了 Node Wrangler 插件
- [ ] 成功新建并保存了一个 .blend 文件
- [ ] 理解了 .blend 文件的作用

### 📖 费曼三句话

1. Blender 是免费开源的 3D 创作套件，5.1 版本在渲染和几何节点上有重大更新。
2. 安装后建议先配置语言、主题和常用插件（尤其是 Node Wrangler）。
3. .blend 文件是场景的"快照"，包含所有数据——一个文件管全部。

## 🍅2 界面介绍与导航（25分钟）

### 费曼输入（概念解释）

Blender 的界面不是传统"菜单栏 + 工具栏"的单窗口，而是**编辑器区域组合**。每个区域可以独立切换为不同类型的编辑器（3D 视口、时间线、着色器编辑器等）。这种设计叫做**完全可定制界面**。

核心布局结构：
- **3D视口** (中心) —— 场景的"画布"
- **属性编辑器** (右侧) —— 物体的"控制面板"
- **大纲** (右上) —— 场景的"目录树"
- **时间线** (底部) —— 动画的"时间轴"

### 刻意练习（具体操作步骤 + 代码/命令）

**Step 1：认识界面区域**
1. 启动 Blender，默认出现一个立方体、一盏灯、一台摄像机
2. 观察五大区域：
   - 最上方：菜单栏 + 工作区标签（Layout / Modeling / Sculpting / UV Editing / Texture Paint / Shading / Animation / Rendering / Compositing / Geometry Nodes / Scripting）
   - 中央：3D 视口（最大的区域）
   - 右上：大纲（Outliner）—— 显示场景中的所有物体
   - 右侧：属性编辑器（Properties）—— 扳手/曲线/材质/纹理等图标
   - 底部：时间线（Timeline）—— 动画面板

**Step 2：切换编辑器类型**
1. 将鼠标移到任意区域的**左上角**，点击下拉图标
2. 尝试将底部区域切换为：`Python Console`（控制台）、`Text Editor`（文本编辑器）
3. 恢复默认：顶部菜单 → File → Defaults → Load Factory Settings

**Step 3：3D 视口导航操控**

| 操作 | 鼠标/键盘 | 说明 |
|------|-----------|------|
| 旋转视图 | 按住**中键**拖动 | 围绕场景中心旋转 |
| 平移视图 | `Shift + 中键` 拖动 | 上下左右平移 |
| 缩放视图 | **滚轮**前后滚动 | 拉近拉远 |
| 正交前视图 | 小键盘 `1` | 正面 |
| 正交右视图 | 小键盘 `3` | 右侧 |
| 正交顶视图 | 小键盘 `7` | 俯视 |
| 正交底视图 | `Ctrl + 7` | 仰视 |
| 相机视图 | 小键盘 `0` | 从当前相机看 |
| 聚焦选中 | 小键盘 `.` （句号） | 将物体居中到屏幕 |
| 四视图 | `Ctrl + Alt + Q` | 切换三分屏/单视图 |

**Step 4：视口着色方式**

按 `Z` 打开视口菜单，四种着色方式：

| 方式 | 快捷键 | 用途 |
|------|--------|------|
| 线框 | `Z → Wireframe` 或 `Shift+Z` | 看拓扑结构 |
| 实体 | `Z → Solid` | 日常建模 |
| 材质 | `Z → Material Preview` | 看材质效果（带环境光） |
| 渲染 | `Z → Rendered` | 最终渲染效果 |

**Step 5：属性编辑器认识各选项卡**
属性编辑器右侧有一排图标选项卡，从左到右依次：

| 图标 | 名称 | 作用 |
|------|------|------|
| 🔴 | 场景(Scene) | 渲染引擎、帧率、单位设置 |
| 🌐 | 世界(World) | 环境背景色/纹理 |
| 📦 | 物体(Object) | 位置/旋转/缩放、显示设置 |
| 🔧 | 修改器(Modifiers) | 非破坏性修改（后文重点） |
| 🏷️ | 粒子(Particles) | 粒子系统 |
| ⚡ | 物理(Physics) | 刚体/软体/布料/流体 |
| 🎨 | 材质(Materials) | 材质球（Principled BSDF） |
| 📐 | 纹理(Textures) | 纹理贴图 |
| ⛓️ | 约束(Constraints) | 物体约束关系 |
| 📄 | 数据(Data) | 网格顶点数、UV 映射 |

### 📖 费曼三句话

1. Blender 界面是分区域编辑器组成的"瑞士军刀"——每个区域都能独立切换功能。
2. 3D 视口用鼠标中键操控视角：中键旋转、Shift+中键平移、滚轮缩放。
3. 属性编辑器是物体的"控制面板"，每个图标对应一个方面的设置。

## 🍅3 基础操作（移动/旋转/缩放/选择）（25分钟）

### 费曼输入（概念解释）

Blender 的核心操作围绕**变换**和**选择**两大体系。

**Object Mode**（物体模式）和 **Edit Mode**（编辑模式）是 Blender 最常用的两个模式。物体模式下操作的是完整物体；编辑模式下操作的是物体的**顶点/边/面**。

变换操作的核心理念：**G/R/S + 锁定轴向 + 精确控制**。

### 刻意练习（具体操作步骤 + 代码/命令）

**Step 1：模式切换练习**
1. 选中场景中的立方体（点击它）
2. 按 `Tab` 切换 Object Mode ↔ Edit Mode
3. 观察：Edit Mode 时物体的顶点会高亮显示
4. 注意：再次按 `Tab` 回到 Object Mode

**Step 2：基础变换（G/R/S）**

| 操作 | 快捷键 | 技巧 |
|------|--------|------|
| 移动 | `G` (Grab) | 按 G 后自由移动 |
| 旋转 | `R` (Rotate) | 按 R 后旋转 |
| 缩放 | `S` (Scale) | 按 S 后缩放 |

**轴向锁定练习**：
1. 选中立方体，按 `G` → 再按 `X`：只能在 X 轴上移动
2. 按 `R` → 再按 `Z`：绕 Z 轴旋转
3. 按 `S` → 再按 `Y`：只在 Y 轴方向缩放
4. 按 `G` → 按住 `Shift` + `X`：排除 X 轴，在 YZ 平面移动

**精确控制**：
- 按 `G` → 输入 `2` → `Enter`：沿鼠标方向移动 2 个单位
- 按 `G` → `X` → 输入 `1.5` → `Enter`：沿 X 轴移动 1.5 单位
- 按 `R` → `Z` → 输入 `45` → `Enter`：绕 Z 轴旋转 45 度
- 按 `S` → 输入 `2` → `Enter`：整体放大 2 倍

**Step 3：选择操作**

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 选择物体 | 左键/右键（取决于键位设置） | 点击选中 |
| 全选/全不选 | `A` | 按一下全选，再按全不选 |
| 框选 | `B` | 拖出矩形框选择 |
| 刷选 | `C` | 圆圈刷选，滚轮调大小，ESC 退出 |
| 加选 | `Shift + 点击` | 追加选择 |
| 减选 | `Shift + 点击` 已选物体 | 取消选择 |

**Step 4：添加基本物体**
1. 确保在 Object Mode
2. 按 `Shift + A` 打开添加菜单
3. 选择 `Mesh` → 浏览各基本体
4. 依次添加：Cube / UV Sphere / Cylinder / Cone / Plane / Torus
5. 观察每个物体的默认位置和外形

**Step 5：删除与隐藏**
- 选中物体 → `X` 或 `Delete` → 确认删除
- 选中物体 → `H` 隐藏 / `Alt + H` 显示全部
- 选中物体 → `Shift + H` 隐藏未选中

### ✅ 完成标准

- [ ] 能用 G/R/S 对物体进行变换操作
- [ ] 能锁定 X/Y/Z 轴进行精确变换
- [ ] 能用 B（框选）/ C（刷选）/ A（全选）选择物体
- [ ] 能用 Shift+A 添加至少 4 种基本体
- [ ] 能熟练用 Tab 切换 Object/Edit Mode

### 📖 费曼三句话

1. G/R/S 是 Blender 的"三套马车"——移动/旋转/缩放，是一切操作的基础。
2. 按 X/Y/Z 锁定轴向 + 输入数字精确控制，是 3D 操作精确又高效的秘诀。
3. Tab 键在 Object Mode 和 Edit Mode 之间切换，决定你操作的是整个物体还是它的顶点/边/面。

## 🍅4 快捷键体系（核心快捷键速记）（25分钟）

### 费曼输入（概念解释）

Blender 以**快捷键密集**著称，但好消息是：**30 个核心快捷键覆盖 90% 的操作**。不需要一次性背完，应该按场景分批记忆。

快捷键学习的最佳策略：**"5个一组，每次学一组，练熟了再加新组"**——也叫"渐进式快捷键积累法"。

### 刻意练习（具体操作步骤 + 代码/命令）

**Step 1：通用快捷键（第一组）**

| 快捷键 | 功能 | 记忆联想 |
|--------|------|----------|
| `Ctrl + S` | 保存文件 | Save |
| `Ctrl + Z` | 撤销 | 后悔药 |
| `Ctrl + Shift + Z` | 重做 | 又后悔了 |
| `Ctrl + N` | 新建场景 | New |
| `Ctrl + Q` | 退出 Blender | Quit |

👉 **现在就练**：按3遍 `Ctrl+Z`，再按3遍 `Ctrl+Shift+Z`，感受撤销堆栈。

**Step 2：编辑模式快捷键（第二组）**

| 快捷键 | 功能 | 记忆联想 |
|--------|------|----------|
| `E` | **挤出**（Extrude） | 最常用的建模命令 |
| `I` | **内插面**（Inset Faces） | Inset |
| `K` | **切刀**（Knife） | 像用刀切 |
| `Ctrl + R` | **环切**（Loop Cut） | 加一圈线 |
| `Ctrl + B` | **倒角**（Bevel） | 切角/倒角 |
| `O` | **衰减编辑**（Proportional Editing） | 球型影响区域 |

👉 **现在就练**：进入 Edit Mode，选一个面，分别按 `E` `I` `K` `Ctrl+R` `Ctrl+B` 各试一次。

**Step 3：视图快捷键（第三组）**

| 快捷键 | 功能 | 记忆联想 |
|--------|------|----------|
| `/` | 孤立显示选中物体 | 专注模式 |
| `.`(句号) | 聚焦选中物体 | 立即居中 |
| `Home` | 显示全部物体 | 回到全景 |
| `Ctrl + Alt + Q` | 切换四视图/单视图 | 四个窗口 |
| `Z` | 视口着色方式菜单 | 渲染方式切换 |

👉 **现在就练**：选中物体，按 `/` 孤立 → 按 `.` 聚焦 → 按 `Home` 显示全部。

**Step 4：对齐与父子级（第四组）**

| 快捷键 | 功能 |
|--------|------|
| `Shift + S` | 吸附菜单（光标到世界原点/选中到光标等） |
| `Alt + G` | 清除位置变换（归零） |
| `Alt + R` | 清除旋转（归零） |
| `Alt + S` | 清除缩放（回到 1:1） |
| `Ctrl + P` | 设置父级（父子绑定） |
| `Alt + P` | 清除父级 |
| `Ctrl + A` | 应用变换/修改器 |

👉 **现在就练**：移动一个物体后，按 `Alt+G` 看它回到原点。

**Step 5：渲染与其它（第五组）**

| 快捷键 | 功能 |
|--------|------|
| `F12` | 渲染当前帧（静态图） |
| `Ctrl + F12` | 渲染动画 |
| `F3` | 搜索菜单（万能搜索框） |
| `Ctrl + Alt + S` | 切换已保存版本（File History） |

**Step 6：自定义快捷键**
1. Edit → Preferences → Keymap
2. 搜索要修改的命令
3. 点击右侧快捷键 → 按新的组合键
4. 建议：不要急于自定义，先用默认键位两周

### ✅ 完成标准

- [ ] 能说出通用组 5 个快捷键
- [ ] 能说出编辑模式组 6 个快捷键
- [ ] 能说出视图组 5 个快捷键
- [ ] 能在 Edit → Preferences → Keymap 中找到并修改快捷键
- [ ] 明确了快捷键分批学习的策略

### 📖 费曼三句话

1. 掌握 30 个核心快捷键就能覆盖 90% 的操作——E(挤出)、Ctrl+R(环切)、Ctrl+B(倒角) 是建模三剑客。
2. 分批学习比一次性硬背更有效：通用组 → 编辑组 → 视图组 → 对齐组 → 渲染组，逐个击破。
3. Blender 支持自定义快捷键，但建议先适应默认键位两周再改。

## 🍅5 基础建模实战（案例：低多边形小屋）（25分钟）

### 费曼输入（概念解释）

**低多边形（Low Poly）** 风格是用少量多边形构建物体轮廓的艺术风格。其核心思想：**基础几何体的组合 + 变形 = 任何物体**。这种风格非常适合初学者，因为：
- 不需要完美拓扑
- 出错容易修复
- 视觉效果好，有成就感
- 为后续复杂建模打基础

### 刻意练习（具体操作步骤 + 代码/命令）

**Step 1：建小屋主体**
1. `Ctrl + N` → General → 新建场景
2. 场景中已有一个默认立方体，这就是**房子主体**
3. 按 `S` → 缩放，让房子扁一点（更像小屋比例）

**Step 2：做尖顶屋顶**

```mermaid
flowchart LR
    A[选中顶面] --> B[E挤出] --> C[S缩放] --> D[G上移]
```

1. 选中主体，按 `Tab` 进入 Edit Mode
2. 按 `3`（面选择模式），点击顶面（或 `A` 全选再按 `Ctrl` + 减选底面）
3. 按 `E` → `Z` → 向上挤出一点
4. 按 `S` 缩放顶面让它变尖（在顶视图中扁平）
5. 按 `G` → `Z` → 向上移动形成尖顶

**Step 3：添加烟囱**
1. 按 `Tab` 回到 Object Mode
2. `Shift + A` → Mesh → Cylinder
3. `S` 缩放（压扁成矮圆柱样式）→ `G` 移动到屋顶侧面
4. 如果烟囱需要穿过屋顶 → 在 Edit Mode 调整顶点位置

**Step 4：建一棵树**
- **树冠**：`Shift + A` → UV Sphere，`S` 缩放到合适大小
- **树干**：`Shift + A` → Cylinder，`S` 调细，`G` 移到球下方
- 选中树干 → `Shift + 点击` 树冠 → `Ctrl + P` → Object（将树干设为树冠的子级，整体移动方便）

**Step 5：布置场景**
1. `Shift + A` → Plane（作为地面）
2. `S` 放大地面（比如 10 倍）
3. 选中房子、树 → `G` 摆放到地面上
4. `Shift + D`（复制）树 → `G` 移动到其他位置
5. 重复几次，造一片"小树林"

**Step 6：加细节**
1. 选中房子 → Tab 进入 Edit Mode
2. 按 `Ctrl + R` 环切加几条线
3. 选中新切的线 → `G` 稍微移动，形成木板缝的效果
4. 选中屋顶 → 按 `E` 再挤出一些边缘形成屋檐

**Step 7：拍照留念**
1. 用鼠标中键调整视角，找到最佳角度
2. 按 `F12` 渲染（如果没装 GPU 渲染器，默认 Eevee 很快）
3. 渲染完成后 `Alt + S` 保存图像

**可选进阶**：
- 给不同物体上不同的材质颜色（属性编辑器 → 材质选项卡 → New → Base Color）
- 给场景加一个背景 Plane，用蓝天颜色

### ✅ 完成标准

- [ ] 成功建立了小屋主体 + 尖顶屋顶
