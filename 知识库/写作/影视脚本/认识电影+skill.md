# 认识电影

> 了解电影是如何制作的，会使得观影经历更加愉快。

## 核心理念

本书由路易斯·贾内梯撰写，核心方法论围绕**从多个维度系统认识电影艺术**。涵盖摄影、场面调度、运动、剪辑、声音、表演、剧本、意识形态等全方位电影语言要素。

## 执行流程

### Phase 1: 视觉语言分析
摄影（镜头/景别/角度）、场面调度（布景/道具/走位）、运动（摄影机运动）

### Phase 2: 听觉与时间语言
剪辑（连续性/蒙太奇/节奏）、声音（对白/音效/音乐）

### Phase 3: 表演与故事
表演风格（现实主义/古典/形式主义）、剧本结构、类型分析

### Phase 4: 理论批评
作者论、意识形态批评、女性主义、符号学

## 局限性

- **适用于**：电影入门者全面了解电影艺术
- **不适用于**：高端学术研究和创作实操技能
