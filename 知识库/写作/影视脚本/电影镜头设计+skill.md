# 电影镜头设计法

> 从构思到银幕——将故事逻辑转化为影像逻辑的分镜设计权威方法。全球发行25万册，被翻译成七种语言的电影分镜设计经典教材。

## 核心理念

本书由史蒂文·卡茨撰写，是电影镜头设计的核心方法论。核心观点：分镜设计是导演将剧本意念视觉化的关键工艺。从前期筹备（剧本分析、美术设计、故事板）到拍摄阶段（机位调度、动作轴线、对话场景），再到后期制作（剪辑风格），形成完整的影像化工作流。书中涵盖希区柯克、斯皮尔伯格等经典影片的故事板案例。

---

## 触发条件

当用户请求以下任务时自动激活：
- 设计分镜头脚本或故事板
- 规划对话场景的机位调度
- 学习动作轴线与三角机位系统
- 设计运动镜头（摇镜、推轨、升降）
- 将剧本转化为视觉化方案

---

## 执行流程

### Phase 1: 影像化准备

**目标**：将剧本转化为视觉化方案

**输入**：
- 剧本或场景描述

**步骤**：

#### Step 1: 剧本分析

**动作**：分解剧本，识别视觉化需求

**标准**：
- 识别场景类型（对话/动作/过渡）
- 标注关键情绪和情感转折点
- 确定场景的戏剧焦点

**输出**：`剧本分解：[场景类型 + 情绪标注 + 戏剧焦点]`

#### Step 2: 美术设计规划

**动作**：与美术设计师协作规划场景视觉

**标准**：
- 场景设计图（概念图→最后设计图）
- 建筑图（平面图、剖面图、立面图）
- 效果图和模型

**输出**：`美术设计：[场景视觉方案]`

#### Step 3: 故事板制作

**动作**：绘制分镜头草图和故事板

**标准**：
- 导演职责：确定叙事意图和视觉风格
- 故事板画家技巧：透视、构图、箭头标注
- 材料选择：铅笔、墨水、马克笔、图解摄影

**输出**：`故事板：[分镜头视觉方案]`

**检查点**：
- [ ] 剧本已分解
- [ ] 美术设计已规划
- [ ] 故事板已制作
- [ ] 已确定画幅制式（1.5:1/1.85:1/2.35:1）

**输出**：影像化方案

---

### Phase 2: 镜头构图设计

**目标**：设计单个镜头的空间关系

**输入**：
- 影像化方案
- 场景需求

**步骤**：

#### Step 1: 选择景别

**动作**：根据叙事需求选择合适景别

**标准**：
- **特写**：强调面部表情、细节、情感
- **中景**：展现人物交流、肢体动作
- **全景**：建立空间关系、环境信息

**输出**：`景别选择：[特写/中景/全景] + [理由]`

#### Step 2: 建立动作轴线

**动作**：确定场景的动作轴线（180°规则）

**标准**：
- 动作轴线原则：摄影机保持在轴线一侧
- 三角机位系统：在灰色半圆区内布置机位
- 越轴处理：通过角色移动或中性镜头建立新轴线

**输出**：`动作轴线：[轴线方向] + [机位工作区间]`

#### Step 3: 设计机位布局

**动作**：在动作轴线范围内布置具体机位

**标准**：
- 根据新视线建立新轴线
- 利用切出镜头和过渡镜头平滑越轴
- 运动物体和动作的动作轴线跟随运动方向

**输出**：
```
机位布局：
- 机位A：[景别 + 角度 + 覆盖内容]
- 机位B：[景别 + 角度 + 覆盖内容]
...
```

**检查点**：
- [ ] 景别已选择
- [ ] 动作轴线已建立
- [ ] 机位布局已完成
- [ ] 30°规则已遵守（相邻镜头角度差≥30°）

**输出**：镜头构图方案

---

### Phase 3: 对话场景调度

**目标**：设计对话场景的完整调度方案

**输入**：
- 镜头构图方案
- 对话内容

**步骤**：

#### Step 1: 选择调度系统

**动作**：根据对话人数和场景复杂度选择调度模式

**标准**：
- **双人对话**：正拍/反拍镜头模式、过肩镜头、侧拍镜头
- **三人对话**：A形态（三角形）或L形态
- **四人以上**：单一机位正面位置、纵深调度、群众场景

**输出**：`调度系统：[模式名称]`

#### Step 2: 设计主镜头

**动作**：确定主镜头（master shot）覆盖整个场景

**标准**：
- 主镜头建立空间关系
- 长镜头提供剪辑灵活性
- 景别和距离覆盖所有角色

**输出**：`主镜头：[全景/中景] + [覆盖范围]`

#### Step 3: 设计覆盖镜头

**动作**：在主镜头基础上设计覆盖镜头

**标准**：
- 正拍镜头与反拍镜头模式
- 过肩镜头建立空间关系
- 特写镜头强调情感反应

**输出**：
```
覆盖镜头：
- 角色A正拍：[景别 + 角度]
- 角色B反拍：[景别 + 角度]
- 过肩镜头A→B：[...]
- 过肩镜头B→A：[...]
- 特写A：[...]
- 特写B：[...]
```

**检查点**：
- [ ] 调度系统已选择
- [ ] 主镜头已设计
- [ ] 覆盖镜头已设计
- [ ] 演员视线已匹配
- [ ] 银幕方向已一致

**输出**：对话调度方案

---

### Phase 4: 运动镜头设计

**目标**：设计运动镜头的调度方案

**输入**：
- 对话调度方案
- 运动需求

**步骤**：

#### Step 1: 选择运动类型

**动作**：根据叙事需求选择运动镜头类型

**标准**：
- **摇镜**：左右横摇、上下竖摇、重新构图、交代动作
- **升降镜头**：运动范围、视点变化
- **推轨镜头**：介绍主体、跟随运动、走近/远离、环绕主体

**输出**：`运动类型：[摇镜/升降/推轨]`

#### Step 2: 设计运动路径

**动作**：规划摄影机运动的具体路径

**标准**：
- 建构单位：将运动分解为可执行的段落
- 解放调度风格：用演员运动取代剪辑
- 移动兴趣中心：在运动中改变焦点

**输出**：
```
运动路径：
- 起始位置：[机位 + 景别]
- 运动轨迹：[描述运动方向]
- 终止位置：[机位 + 景别]
- 运动动机：[叙事理由]
```

#### Step 3: 设计转场

**动作**：设计镜头间的转场方式

**标准**：
- 剪切：瞬时转换
- 叠化：时间流逝或空间转换
- 淡入淡出：场景开始/结束
- 动作划变：利用动作匹配转场
- 聚焦-失焦：焦点转移

**输出**：`转场方式：[剪切/叠化/淡入淡出/其他]`

**检查点**：
- [ ] 运动类型已选择
- [ ] 运动路径已设计
- [ ] 运动动机已明确
- [ ] 转场方式已设计
- [ ] 观众位置已考虑

**输出**：运动镜头方案

---

## 决策规则

| 条件 | 行动 | 说明 |
|------|------|------|
| 双人对话场景 | 使用正拍/反拍模式 | 最经典的对话调度方式 |
| 三人对话场景 | 选择A形态或L形态 | 根据角色位置关系选择 |
| 需要建立空间 | 使用全景主镜头 | 先建立空间再切近景 |
| 需要越轴 | 通过角色移动或中性镜头 | 不可直接跳轴 |
| 运动镜头 | 必须有明确动机 | 运动需服务叙事 |
| 宽银幕制式 | 选择1.85:1或2.35:1 | 根据场景复杂度选择 |

## 示例

### 示例：双人对话场景分镜设计

**输入**：设计一个两人在咖啡馆对话的场景

**执行过程**：
1. Phase 1: 剧本分析→场景类型为对话，情绪为紧张对峙
2. Phase 2: 选择中景为主，建立动作轴线，布置三角机位
3. Phase 3: 设计正拍/反拍模式，主镜头覆盖全景，覆盖镜头包括过肩和特写
4. Phase 4: 设计一个推轨镜头在对话高潮时缓慢推进

**输出**：
```
分镜头设计：咖啡馆对峙
1. 全景主镜头（固定）：建立空间关系 → 切
2. 中景A侧拍（固定）：角色A发言 → 切
3. 中景B侧拍（固定）：角色B回应 → 切
4. 过肩A→B（固定）：A看着B → 切
5. 过肩B→A（固定）：B看着A → 切
6. 特写A（固定）：A的表情反应 → 叠化
7. 推轨镜头：缓慢推进至双人中景 → 切
8. 特写B（固定）：B的反应 → 淡出
```

## 局限性

- **不适用于**：纪录片即兴拍摄、实验电影非传统叙事
- **依赖条件**：需要剧本或明确的场景描述作为起点
- **可能失效**：当场景过于复杂（大量角色、复杂运动）时需要拆分设计
- **信息截止**：基于原书插图第2版，更新了数字时代工具但核心方法不变
