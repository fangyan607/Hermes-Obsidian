# 🍅 50番茄装饰设计&营销AI使用教程

> **从零到一，掌握装饰行业的全栈 AI 工具链**
> 适用人群：装饰行业从业者 / 独立设计师 / 装修公司转型者

## 📐 教程定位

本教程基于 [[Script/装饰行业五年发展规划]] 的 AI 技术需求设计。规划文档明确指出六大要素中的**AI设计能力**和**内容获客体系**是轻资产转型的核心驱动力：

| 规划需求 | 本教程对应模块 | 番茄数 |
|:---------|:--------------|:------:|
| AI方案生成（大模型+提示词） | 模块A+B | 20🍅 |
| AI效果图（Diffusion模型） | 模块C | 10🍅 |
| AI内容创作（案例帖/知识帖） | 模块D | 10🍅 |
| 全流程整合（从获客到交付） | 模块E | 10🍅 |

## 🧠 核心教学法

本教程使用**番茄工作法 × 费曼学习法 × 刻意练习**三体合一：

### 番茄工作法（节奏）
- 每个🍅 = 25分钟专注学习 + 5分钟休息
- 4个🍅一组，之后15分钟长休息
- 50🍅 = 约12.5个学习日（每天4🍅）

### 费曼学习法（理解）
每个番茄末尾的**费曼三句话**要求你：
1. 用最简单的语言解释核心概念
2. 去掉所有术语，像讲给小学生听
3. 如果说不清楚，说明还没真懂

### 刻意练习（技能）
- 每个番茄都有**具体可操作的练习任务**
- 任务有明确的**完成标准**
- 难度**循序渐进**，从模仿到创造
