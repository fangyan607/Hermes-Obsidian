# 世界模型：AI从语言走向物理世界

> 2026年6月，北京智源人工智能研究院（BAAI）发布全球首个通用世界基础模型 Physis-v0.1，标志着AI从"生成内容"向"理解物理世界"的范式迁移

## 概述

世界模型（World Model）是2026年AI领域最重要的范式转型——AI系统不再满足于处理文本和图像，而是试图学习、理解并预测物理世界的运行规律。不同于大型语言模型（LLM）从海量文本中学习统计模式，世界模型的核心在于对物理空间、时间、因果关系的深层理解。

## 核心突破：Physis-v0.1

2026年6月14日，BAAI在2026智源大会上正式发布了**Physis-v0.1**，这是全球首个通用世界基础模型。其四大核心能力包括：

| 能力 | 说明 |
|------|------|
| **物理一致性** | 模型输出遵循真实世界物理规律，不出现违背常识的结果 |
| **行动因果性** | 理解动作与结果之间的因果链，能预测"如果...会怎样" |
| **长期连贯性** | 在长时间序列预测中保持逻辑一致性，不出现"忘记上下文"的问题 |
| **跨场景泛化** | 训练数据未覆盖的新场景也能合理推理 |

同时发布的 **RoboBrain Orca** 是面向具身机器人的世界模型版本，能够预测世界的"下一物理状态"，使机器人不仅能识别物体，还能推理出"杯子在桌边→掉下来会碎"这样的物理因果关系。

## 产业影响

世界模型被视为通往AGI的关键路径，产业前景极为广阔：

- **摩根士丹利**预测：世界模型将在2035年前催生 **$10万亿** 产业规模
- **国元证券**预测：基于世界模型的机器人产业在2030年前将超过 **$3万亿**
- 应用场景：自动驾驶、机器人、气象预测、工业仿真、游戏引擎

## 与现有AI的差异

| 维度 | 传统LLM | 世界模型 |
|------|---------|---------|
| 输入 | 文本/代码 | 多模态（视觉+触觉+听觉+文本） |
| 输出 | 文本生成 | 物理状态预测 + 行动规划 |
| 核心能力 | 语言理解 | 物理世界理解 |
| 典型任务 | 写作、翻译、编程 | 机器人控制、场景模拟、因果推理 |
| 局限 | 缺乏物理常识 | 算力需求极高 |
