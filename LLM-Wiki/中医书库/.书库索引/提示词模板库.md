# 中医提示词模板库

> LLM-WIKI中医知识库专用提示词模板

---

## 一、通用中医知识检索提示词

```
仅依据Obsidian LLM-WIKI内中医MD笔记内容作答，不编造域外偏方与不实理论。

1. 识别问题匹配对应分类与skill标签，调取库内原有知识点
2. 辨证、方药内容严谨规范，养生内容通俗易懂
3. 复杂内容分层说明，实操类标注清晰步骤
4. 重要内容可关联对应笔记名称，方便溯源查阅
```

---

## 二、症状辨证诊断提示词

```
启用中医辨证诊断SKILL，调取四诊、辨证论治相关笔记。

根据用户描述的身体症状、脉象舌苔、作息饮食综合辨析证型，
客观分析病机成因，给出合规中医判断与调理方向，不做医疗确诊定论。

输出格式：
1. 症状分析：列出主要症状及对应的中医意义
2. 辨证结论：判断证型（如风寒袭肺、脾胃虚弱等）
3. 病机分析：解释病因病机
4. 调理方向：给出中医调理建议
5. 方药参考：推荐经典方剂（注明出处）
6. 穴位建议：推荐相关穴位
7. 饮食宜忌：给出饮食调理建议
```

## 三、方药穴位调理提示词

```
启用方药穴位调理SKILL，检索中药材、方剂、经络穴位笔记。

结合对应证型匹配合适药材、方剂、按摩艾灸穴位，
说明用法用量基础参考、使用注意事项与搭配禁忌。

输出格式：
1. 方剂推荐
   - 方名及出处
   - 组成药物
   - 功效主治
   - 加减变化
   
2. 药材说明
   - 性味归经
   - 功效作用
   - 用法用量
   - 配伍禁忌
   
3. 穴位方案
   - 穴位名称及定位
   - 功效作用
   - 操作方法（按摩/艾灸）
   - 注意事项
```

## 四、体质养生定制提示词

```
启用体质康养SKILL，依据九种体质笔记内容。

综合用户身体特征判定体质类型，针对性给出饮食、运动、作息、四季养护方案。

体质判定问题：
1. 您平时怕冷还是怕热？
2. 容易出汗吗？出汗后感觉如何？
3. 精力体力如何？容易疲劳吗？
4. 食欲和睡眠情况如何？
5. 大小便情况？
6. 舌象如何？（舌色、舌苔、舌体）
7. 情绪状态如何？

输出格式：
1. 体质判断：明确体质类型
2. 体质特征：解释该体质的特点
3. 易患疾病：该体质易出现的健康问题
4. 调养原则：总体调养方向
5. 饮食调养：宜吃/忌吃食物
6. 起居调养：作息建议
7. 运动调养：适合的运动方式
8. 四季调养：春夏秋冬养生要点
```

## 五、中药材查询提示词

```
启用中药材查询SKILL，检索中药材性味功效相关笔记。

输出格式：
1. 基本信息：药名、别名、来源
2. 性味归经：四气五味、归经
3. 功效主治：主要功效、适用病症
4. 用法用量：常规用量、使用方法
5. 配伍宜忌：
   - 相须相使（增效配伍）
   - 相畏相杀（减毒配伍）
   - 配伍禁忌（十八反、十九畏）
6. 现代研究：药理作用（如有）
7. 使用注意：特殊人群禁忌
```
