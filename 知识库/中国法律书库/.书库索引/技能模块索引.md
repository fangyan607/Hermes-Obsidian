## 二、技能详细说明

### Skill1：法律条文查询

**触发指令**：
- 法律条文查询
- 法条检索
- 法律依据查找
- 条文搜索

**核心能力**：
| 能力 | 说明 |
|------|------|
| 自然语言查询 | 支持口语化问题输入，如"民间借贷利息上限" |
| 精准条文检索 | 按法律名称、法条编号、发布时间检索 |
| 关联法条拓展 | 自动关联展示配套法规、司法解释 |
| 时效性甄别 | 自动标注有效/废止/修订状态 |

**数据来源**：
- 中国法律库（719部）
- 行政法规库（819部）
- 司法解释库（877部）

**输出内容**：
- 核心法律条文原文
- 条文释义与适用条件
- 关联法规链接
- 新旧对比
- 实务注意事项

**使用示例**：
```
用户：查询民间借贷利息的法律规定
AI：调用Skill1，检索民法典、最高人民法院关于审理民间借贷案件适用法律若干问题的规定等，输出利息上限（LPR的4倍）、相关条文原文、司法解释要点。
```

### Skill2：案例检索

**触发指令**：
- 案例检索
- 查找案例
- 搜索类似案例
- 年度案例查询

**核心能力**：
| 能力 | 说明 |
|------|------|
| 案由分类检索 | 按民事、刑事、行政、执行等分类检索 |
| 关键词检索 | 按裁判要旨、争议焦点关键词检索 |
| 相似案例匹配 | 根据案情要素匹配相似案例 |
| 裁判规则提取 | 提取案例的裁判要旨和法官后语 |

**数据来源**：
- 年度案例库（2025年度案例23册）
- 买卖合同纠纷、合同纠纷、民间借贷纠纷等民商事案例
- 刑事案例一~四
- 行政纠纷、执行案例

**输出内容**：
- 案例基本信息（案由、当事人、审理法院）
- 裁判要旨（一句话概括）
- 争议焦点
- 裁判观点
- 法官后语（指导价值提炼）
- 关联法条

**案例分册导航**：

| 类型 | 分册 | 适用场景 |
|------|------|----------|
| 合同类 | 买卖合同纠纷、合同纠纷、民间借贷纠纷 | 合同起草、纠纷解决 |
| 物权类 | 房屋买卖合同纠纷、物权纠纷 | 房产交易、物权争议 |
| 家事类 | 婚姻家庭与继承纠纷 | 婚姻继承问题 |
| 金融类 | 保险纠纷、金融纠纷 | 金融业务、保险理赔 |
| 劳动类 | 劳动纠纷 | 劳动争议处理 |
| 刑事类 | 刑事案例一~四 | 刑事辩护、量刑参考 |
| 其他 | 知产、土地、行政、执行 | 专业领域案件 |

**使用示例**：
```
用户：查找关于民间借贷利息上限的案例
AI：调用Skill2，检索民间借贷纠纷分册，找到相关案例，输出裁判要旨（利息超过LPR4倍部分不予支持）、争议焦点、法官后语、关联法条。
```

### Skill3：案例分析

**触发指令**：
- 案例分析
- 案例比对
- 相似案例查询
- 案例拆解

**核心能力**：
| 能力 | 说明 |
|------|------|
| 案情要素抽取 | 自动提取主体、行为、后果、争议焦点 |
| 相似案例匹配 | 在案例库中检索相似度最高的案例 |
| 案例深度拆解 | 输出基本案情、争议焦点、裁判依据、裁判观点、判决结果 |
| 多案例对比 | 分析同案同判、类案不同判原因 |

**数据来源**：
- 年度案例库（2025年度案例23册）
- 司法解释库中的案例规则
- 人民检察院办案规则
- 人民法院诉讼规则

**输出内容**：
- 基本案情提炼
- 争议焦点梳理
- 裁判依据列明
- 裁判观点提取
- 判决结果明确
- 实务启示总结

**使用示例**：
```
用户：分析一个民间借贷纠纷案例，借款10万，约定年利率24%，已还款5万
AI：调用Skill3，提取案情要素，匹配相似案例，分析利率是否超过法定上限（LPR的4倍），计算合法本息，输出裁判预判。
```

### Skill4：案件研判

**触发指令**：
- 案件研判
- 判案辅助
- 法律适用分析
- 裁判预判

**核心能力**：
| 能力 | 说明 |
|------|------|
| 案情要素抽取 | 自动清洗无效信息，抽取核心要素 |
| 法律适用判定 | 判定适用法律门类、具体法条、司法解释 |
| 裁判结果预判 | 基于同类案例预判责任划分、赔偿区间、量刑范围 |
| 风险提示 | 识别法律漏洞、证据瑕疵、争议隐患 |

**数据来源**：
- 全库检索（法律+法规+解释+案例）

**输出内容**：
- 案情概述
- 法律依据
- 类案参考
- 裁判预判
- 风险提示
- 办案建议

**案件类型预判**：

| 案件类型 | 预判内容 |
|----------|----------|
| 民事案件 | 责任划分比例、赔偿金额区间、胜诉概率 |
| 行政案件 | 行政行为是否合法、处罚是否适当 |
| 刑事案件 | 罪名认定、量刑区间、缓刑适用概率 |
| 合规案件 | 违规等级、处罚标准、整改要求 |

**使用示例**：
```
用户：研判案件：甲借乙10万元，约定年利率36%，已还款8万元，乙起诉要求还款本息共15万元
AI：调用Skill4，分析利率超过法定上限（LPR的4倍约13.8%），超出部分无效，重新计算合法本息，预判乙的诉讼请求部分支持，输出风险提示和办案建议。
```

### Skill5：合规审查

**触发指令**：
- 合规审查
- 风险识别
- 合同审查
- 条款筛查

**核心能力**：
| 能力 | 说明 |
|------|------|
| 合同合规审查 | 筛查违规条款、不合规表述 |
| 法律风险提示 | 识别潜在法律风险 |
| 修改建议 | 给出合规修改建议 |
| 合规依据 | 列明合规/违规法律依据 |

**数据来源**：
- 中国法律库（民法、合同法、公司法等）
- 行政法规库（行业监管条例）

**审查领域**：

| 领域 | 审查要点 |
|------|----------|
| 合同审查 | 条款合法性、权利义务平衡、违约责任 |
| 劳动合同 | 劳动法合规、社保缴纳、解除条件 |
| 公司章程 | 公司法合规、股东权利、治理结构 |
| 数据合规 | 个人信息保护法、数据安全法 |
| 金融合规 | 证券法、银行法、监管条例 |

**使用示例**：
```
用户：审查这份借款合同的主要条款是否合规
AI：调用Skill5，逐条审查合同条款，标注利率是否超过法定上限、违约金是否过高、格式条款是否有效，给出修改建议和合规依据。
```

- 法律意见书

**核心能力**：

|------|------|

| 法律意见书 | 整合法律依据、分析意见、风险提示 |
| 合规意见书 | 整合合规分析、整改建议 |

**数据来源**：

**文书类型**：

| 文书类型 | 适用场景 |
|----------|----------|

| 合规意见书 | 企业合规审查 |

**使用示例**：
```
用户：生成一份民间借贷纠纷起诉状，原告张三，被告李四，借款10万元，约定年利率12%，到期未还
AI：调用Skill6，依据民事诉讼法、民法典，生成起诉状，包含当事人信息、诉讼请求（本金+利息）、事实与理由、法律依据。
```

**触发指令**：
- 法律经济学分析

- 成本收益分析

**核心能力**：
| 能力 | 说明 |
|------|------|
| 经济学视角分析 | 运用经济学理论分析法律制度 |
| 成本-收益分析 | 评估法律规则的经济效率 |
| 激励效应分析 | 分析法律规则对行为的影响 |
| 制度效率比较 | 比较不同法律规则的效率 |

| 侵权法 | 责任规则效率分析（汉德公式） |
| 刑法 | 犯罪威慑与刑罚效率 |
| 反垄断 | 市场力量与竞争政策分析 |
| 程序法 | 诉讼成本与和解分析 |

**技能文档**：

```
用户：分析环境污染应适用何种责任规则
AI：调用Skill8，运用经济学分析：企业预防成本低、受害人预防成本高、信息不对称，结论：应适用严格责任激励企业最优预防。
```

## 三、技能调用流程

```
用户提问 → 意图识别 → 技能匹配 → 数据检索 → 结果生成 → 输出展示

意图识别规则：
├── 查询法条 → Skill1
├── 检索案例 → Skill2
├── 分析案例 → Skill3
├── 案件研判 → Skill4
├── 审查合规 → Skill5
├── 生成文书 → Skill6
├── 法律咨询 → Skill7
├── 经济学分析 → Skill8（新增）
└── 犯罪心理分析 → Skill9（新增）
```
