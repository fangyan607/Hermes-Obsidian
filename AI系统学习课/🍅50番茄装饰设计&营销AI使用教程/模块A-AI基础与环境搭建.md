# 🍅 模块A：AI基础与环境搭建（10🍅）

> 🎯 目标：搭建完整AI工具链，能独立跑通每个工具
> ⏱ 预计：10天（每天1🍅）或 2.5天（每天4🍅）

## 🍅 番茄1：AI认知与装饰行业应用全景

### 📖 费曼三句话

1. **AI不是魔法，是"有经验的设计助理"**——你给它清晰的指令，它在海量数据里找到最优方案。就像给一个看过百万张装修图的助理说"我要一个日式简约客厅"，它立刻知道长什么样。
2. **装饰行业用AI有三大场景**：出方案（大模型写设计说明）、出效果图（AI画图）、出内容（AI写案例帖）。这三个场景覆盖了从获客到交付的完整链条。
3. **AI不会取代设计师，但会用AI的设计师会取代不会用的**——就像计算器没有取代会计，但让会计效率提升了10倍。

### 🛠 刻意练习（25分钟）

1. **打开一个AI大模型**（推荐 DeepSeek 网页版，免费）
2. **输入第一个装饰行业问题**：
   ```
   我是一个装修设计师，想了解2026年装饰行业的三大趋势。
   请用100字以内回答，适合发朋友圈的简洁风格。
   ```
3. **观察输出**：它回答了什么？语气怎么样？你觉得哪些可以用？

## 🍅 番茄2：主流AI大模型对比与选择

### 📖 费曼三句话

1. **大模型分三类**：对话型（DeepSeek/Claude/ChatGPT——适合写方案、对话）、绘图型（Midjourney/Stable Diffusion——适合画效果图）、专用型（剪映AI/CanvaAI——适合做内容）。
2. **装饰行业最优组合**：DeepSeek（免费，中文好，写方案+预算）+ Midjourney（效果好，画效果图）+ 剪映（免费，剪短视频）——零成本启动，总月费≤200元。
3. **不用纠结"哪个最好"**——每个工具都有擅长和不擅长的。核心能力不是选工具，而是**知道什么时候用哪个工具**。

### 🛠 刻意练习

|:----|:----|:--------|:----:|:---------|

| Claude | 深度分析 | ⭐⭐ | ¥0-200 | 注册+提问1次 |
| 通义千问 | 中文绘图 | ⭐ | ¥0 | 注册+尝试 |

## 🍅 番茄3：AI绘图工具安装与配置

### 📖 费曼三句话

1. **AI绘图有两条路**：云端的Midjourney（像租车，交钱就能开，效果好但贵）和本地的Stable Diffusion（像买车，一次投入，自由度高但需要配置）。
2. **对装饰行业，推荐先用云端工具**——通义万相（阿里出品，免费额度）、文心一格（百度出品，中文理解好）、Midjourney（效果最好，月费$10-30）。零成本启动，之后按需升级。
3. **安装AI绘图工具的核心就三步**：注册账号 → 找到"输入描述词"的输入框 → 输入"一个北欧风客厅，浅色木地板，大白墙，绿植点缀"看它画出什么。

### 🛠 刻意练习

3. 保存生成的图片到本地
4. 用同样的提示词在**不同工具**生成一次，对比效果

## 🍅 番茄4：AI内容创作工具搭建

### 📖 费曼三句话

1. **AI内容创作是"一人编辑部"**——你不需要文案、摄影师、设计师团队。AI 写初稿、你改一改、AI 配图、你选一选——一个人完成以前3个人的工作。
2. **核心工具链就三样**：AI大模型（写文案）+ Canva/稿定设计（排版配图）+ 剪映（剪视频）。三者打通，一篇文章从构思到发布不超过1小时。
3. **关键不是工具多先进，而是你的"编辑能力"**——AI产出的是70分的初稿，用你的行业经验把它改到90分。这20分的差距，就是你的专业价值。

### 🛠 刻意练习

1. 注册**Canva**（canva.cn）或**稿定设计**（gaoding.com）
2. 搜索模板"装修"/"家居"，选择一个喜欢的
3. 用AI生成的文案替换模板中的占位文字
4. 导出为一张社交媒体图片

### ✅ 完成标准

- [ ] 注册Canva或稿定设计
- [ ] 用模板+AI文案完成1张社交媒体图
- [ ] 导出发到手机上看看效果

## 🍅 番茄5：短视频工具链搭建

### 📖 费曼三句话

1. **装饰行业的短视频 = 改造前后对比 + 干货知识**——爆款公式就两个：要么让人惊叹"变化太大了"，要么让人感觉"学到了"。不需要专业拍摄，手机+剪映就够了。
2. **剪映的核心功能就三个**：剪辑（剪掉废话）、配音（AI朗读文字）、字幕（自动识别说话）。这三个功能就能做出80分的短视频。
3. **发布频率比质量更重要**——在起步阶段，每周2条"70分的视频"比每月1条"90分的视频"更能建立信任。内容先完成，再完美。

### 🛠 刻意练习

1. 打开**剪映**
2. 找一段装修现场手机视频（或用手机拍一段10秒的空房间）
3. 练习：剪辑到15秒 + 加AI配音（文字转语音） + 加自动字幕
4. 导出看看效果

### ✅ 完成标准

- [ ] 剪映完成一个15秒短视频
- [ ] 包含：剪辑 + 配音 + 字幕
- [ ] 视频导出成功

## 🍅 番茄6：客户管理系统（轻量CRM）搭建

### 📖 费曼三句话

1. **装饰行业的CRM不是软件，是"客户记忆系统"**——客户什么时候问过价、喜欢什么风格、家里几口人、预算多少。你不记下来，下次聊天就忘了，客户就觉得你不专业。
2. **最低成本的CRM**：一张Excel表格或者飞书/钉钉文档。列名：客户名、电话、小区、风格偏好、预算、跟进状态、下次联系时间。每天花5分钟更新，胜过任何昂贵软件。
3. **AI时代CRM的升级**：用AI自动记录通话摘要、AI分析客户偏好、AI提醒你跟进。但这些是加分项——先做好"记录"这个基本功。

### 🛠 刻意练习

1. 打开飞书/钉钉/腾讯文档 或 Excel
2. 创建一张客户管理表格，包含以下字段：
   - 客户姓名 | 电话 | 小区 | 面积 | 风格偏好 | 预算 | 跟进状态 | 下次联系
3. 录入3个虚拟客户数据（或用真实客户脱敏后录入）

### ✅ 完成标准

- [ ] 客户管理表格创建完成
- [ ] 至少录入3条客户记录
- [ ] 表格格式清晰，方便每天更新

## 🍅 番茄7：AI获客矩阵规划

### 📖 费曼三句话

1. **AI获客不是"用AI发广告"，而是"用AI批量生产有价值的内容"**——每篇内容都是在回答客户的一个问题。客户搜"老房翻新多少钱"看到你的文章，觉得有用，就关注你了。
2. **装饰行业的内容矩阵 = 三个平台 × 三种内容**：
   - 小红书：改造案例 + 避坑指南（种草）
   - 抖音：改造过程 + 知识干货（曝光）
   - 公众号：深度案例 + 行业分析（信任）
3. **AI的生产力体现在"批量"和"变体"**——一个案例，AI可以改写成小红书版本、抖音脚本版本、公众号版本。一次整理，三个平台分发。

### 🛠 刻意练习

1. 用AI规划你的内容矩阵：
   ```
   我是一个装修设计师，主要做旧房翻新和适老化改造。
   请帮我规划一个月的内容发布计划：
   - 小红书：每周3篇（改造案例、避坑指南、风格推荐各1篇）
   - 抖音：每周2条（施工过程、知识干货各1条）
   - 公众号：每周1篇（深度案例）
   请输出一个表格。
   ```
2. 根据AI的输出，调整成你真正能执行的版本

### ✅ 完成标准

- [ ] AI生成了一份内容规划
- [ ] 你手动调整了至少3处（让计划更符合实际）
- [ ] 确定了本周要发的第一篇内容主题

## 🍅 番茄8：提示词入门——让AI听懂你的话

### 📖 费曼三句话

1. **提示词（Prompt）就是"给AI的任务说明书"**——说明书越清晰，AI做得越好。"写一个装修方案"和"写一个80平米老房翻新的方案，现代简约风格，三口之家，预算15万以内，重点改造厨房和卫生间"的差距，就是60分和90分的差距。
2. **提示词黄金公式 = 角色 + 任务 + 细节 + 输出格式**：
   - 角色："你是一个有10年经验的装修设计师"
   - 任务："帮我设计一个客厅改造方案"
   - 细节："20平米，采光一般，一家三口，预算3万"
   - 输出格式："请用表格形式列出改造项目和预算"
3. **提示词要迭代，不要追求一次完美**——第一次输出不满意，就加细节："再详细一点""换成更便宜的材料""考虑老人使用"。AI会记住上下文，越改越好。

### 🛠 刻意练习

1. 打开 DeepSeek/Claude
2. 用"角色+任务+细节+输出格式"公式写一个提示词：
   ```
   你是一个有10年经验的室内设计师（角色）。
   请帮我设计一个15平米的卧室改造方案（任务）。
   居住者是一对退休老夫妇，需要双人床、充足的储物空间，
   采光一般，预算1.5万以内（细节）。
   请用清单格式列出改造项目和预估费用（输出格式）。
   ```
3. 不满意就迭代一次，直到你基本满意

### ✅ 完成标准

- [ ] 用黄金公式写了一个完整提示词
- [ ] AI输出了一个可用的方案
- [ ] 你迭代了至少1次

## 🍅 番茄9：图片素材管理与AI增强

### 📖 费曼三句话

1. **装饰行业的图片素材 = 你的"视觉案例库"**——每张完工照片、每张效果图、每张改造对比图，都是获客素材。关键是要分类管理，用时找得到。
2. **AI可以帮你做三件事**：低分辨率图变高清（AI放大）、黑白照片上色（AI上色）、普通照片变不同风格（AI风格迁移）——这些都能用免费在线工具完成。
3. **素材管理的黄金法则**：按"风格/空间/状态"三级分类。风格（北欧/日式/现代）、空间（客厅/卧室/厨卫）、状态（改造前/改造中/改造后）。一个文件归一个类，找的时候3秒定位。

### 🛠 刻意练习

1. 在电脑上建立你的素材文件夹结构：
   ```
   装饰案例/
   ├── 北欧风格/
   │   ├── 客厅/
   │   │   ├── 改造前/
   │   │   ├── 改造中/
   │   │   └── 改造后/
   │   ├── 卧室/
   │   └── 厨卫/
   ├── 日式风格/
   ├── 现代简约/
   └── 适老化改造/
   ```
2. 把番茄3生成的AI效果图存入对应文件夹
3. 用AI把其中一张图"换个风格"试试

### ✅ 完成标准

- [ ] 素材文件夹结构建立完成
- [ ] 至少存入3张案例图片
- [ ] 尝试了AI风格迁移

## 🍅 番茄10：AI工具链整合——从输入到输出跑通全流程

### 📖 费曼三句话

1. **工具链整合 = 一个需求走通所有工具**——客户说"我想看看北欧风客厅改造效果"，你的工作流应该是：大模型写方案描述 → AI绘图生成效果图 → Canva排版方案说明 → 微信发给客户。全程不超过30分钟。
2. **关键不是工具多，是"无缝衔接"**——每个工具的产出是下一个工具的输入。大模型的文案可以直接复制到Canva，Canva的图片可以直接存到素材库。减少"搬运"时间，就是效率提升。
3. **第一次跑通全流程会有点磕绊，但跑通一次就通了**——就像第一次骑自行车。之后的每次使用，只是微调和优化。

### 🛠 刻意练习——模块A综合考核

**任务**：跑通一个迷你全流程（30分钟内完成）

| 步骤 | 工具 | 产出 | 时间 |
|:----|:----|:----|:----:|
| 1. 写方案描述 | DeepSeek/Claude | 200字方案说明 | 5min |
| 2. 生成效果图 | 通义万相/文心一格 | 2张效果图 | 10min |
| 3. 排版方案 | Canva | 1页方案展示图 | 10min |
| 4. 保存归档 | 素材文件夹 | 文件归入对应目录 | 5min |

### ✅ 完成标准
