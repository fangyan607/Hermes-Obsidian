## 书籍概览

### 作者背景

南怀瑾先生对《孟子·离娄篇》的深入讲解，揭示了孟子关于规矩、仁政、修身等方面的核心思想。

### 核心主题

《离娄篇》以"规矩"开篇，讨论了仁政、礼义、修身等重要主题，是理解孟子政治哲学和伦理思想的关键篇章。

### 学习目标

完成本周学习后，你将能够：
- 理解"规矩"的深层含义
- 掌握孟子仁政思想的核心
- 用自己的话复述修身的方法
- 将孟子的智慧应用于现代生活

## Day 1：规矩与方圆

### 番茄钟1：不以规矩不能成方圆（25分钟）

#### 核心原文

> 孟子曰：「离娄之明，公输子之巧，不以规矩，不能成方圆。」

#### 南师解读要点

**1. 离娄与公输子**

南师解释，离娄是古代视力极好的人，公输子即鲁班，是古代最巧的工匠。

**2. 规矩的重要性**

```
离娄之明 → 天赋资质
公输子之巧 → 技术能力
规矩 → 方法与标准
```

南师强调，即使有天资和技术，没有规矩（方法与标准），也无法成就大事。

**3. 规矩的现代意义**

| 古代 | 现代 |
|------|------|
| 规矩圆规 | 方法论、流程 |
| 方圆 | 成果、目标 |

> **费曼自测**
>
> 请举一个现代职场中"不以规矩不能成方圆"的例子。

---

### 番茄钟2：规矩的本质（25分钟）

#### 核心原文

> 「师旷之聪，不以六律，不能正五音。」

#### 南师解读要点

**1. 六律与五音**

南师解释，六律是古代音乐的定音标准，五音是宫商角徵羽。

**2. 标准的作用**

```
六律 → 音准标准
    ↓
正五音 → 演奏和谐
    ↓
比喻：任何领域都需要标准
```

**3. 现代应用**

南师认为，现代社会的"规矩"包括法律、制度、职业道德等。

> **费曼自测**
>
> 请解释"规矩"与创新的关系。

## Day 2：仁政之道

### 番茄钟3：不忍人之心（25分钟）

#### 核心原文

> 孟子曰：「人皆有不忍人之心。先王有不忍人之心，斯有不忍人之政矣。」

#### 南师解读要点

**1. 不忍人之心**

南师解释，"不忍人之心"就是恻隐之心——看到别人痛苦，自己心里也不忍。

**2. 仁政的基础**

```
不忍人之心 → 个人道德
    ↓
不忍人之政 → 政策制度
    ↓
核心：政治源于人性
```

**3. 政治与道德**

南师强调，好的政治必须建立在道德基础之上。

> **费曼自测**
>
> 请用"不忍人之心"分析一个现代政策。

---

### 番茄钟4：仁政的方法（25分钟）

#### 核心原文

> 「以不忍人之心，行不忍人之政，治天下可运之掌上。」

#### 南师解读要点

**1. 治天下如运掌**

南师指出，仁政的核心是"推己及人"——将自己不忍心的事推广到政治。

**2. 仁政的具体措施**

| 措施 | 含义 |
|------|------|
| 省刑罚 | 减少刑罚 |
| 薄税敛 | 减轻税负 |
| 深耕易耨 | 发展农业 |

**3. 现代启示**

```
企业管理 → 关心员工
社会治理 → 以人为本
```

> **费曼自测**
>
> 请举例说明"推己及人"在管理中的应用。

## Day 3：礼义之本

### 番茄钟5：礼的本质（25分钟）

#### 核心原文

> 孟子曰：「礼之实，节文斯二者是也。」

#### 南师解读要点

**1. 礼的内涵**

南师解释，"节文"即调节和修饰。礼是对仁义之心的调节和表达。

**2. 礼与仁义的关系**

```
仁 → 内心之爱
义 → 行为之当
礼 → 表达方式
```

**3. 礼的作用**

| 作用 | 说明 |
|------|------|
| 节 | 节制，不过分 |
| 文 | 文饰，更优美 |

> **费曼自测**
>
> 请解释"礼"在现代社交中的作用。

---

### 番茄钟6：义的精神（25分钟）

#### 核心原文

> 「义之实，从兄是也。」

#### 南师解读要点

**1. 义的来源**

南师指出，"从兄"指尊敬兄长，引申为对长上、对秩序的尊重。

**2. 义的层次**

```
家庭 → 敬兄长
社会 → 尊长上
政治 → 守秩序
```

**3. 义的现代意义**

南师认为，"义"在现代社会表现为职业道德、公民责任。

> **费曼自测**
>
> 请举例说明"义"在职场中的表现。

## Day 4：修身养性

### 番茄钟7：反求诸己（25分钟）

#### 核心原文

> 孟子曰：「爱人不亲，反其仁；治人不治，反其智；礼人不答，反其敬。」

#### 南师解读要点

**1. 反求诸己的方法**

南师强调，遇到问题时，首先要反省自己。

| 问题 | 反省 |
|------|------|
| 爱人不亲 | 反其仁——我的仁心够吗？ |
| 治人不治 | 反其智——我的方法对吗？ |
| 礼人不答 | 反其敬——我的敬意足吗？ |

**2. 内省的智慧**

```
问题出现 → 不责怪他人
    ↓
反求诸己 → 找自己的原因
    ↓
改进提升 → 问题自然解决
```

**3. 现代应用**

南师认为，这是解决人际关系问题的根本方法。

> **费曼自测**
>
> 请用"反求诸己"分析一个你最近遇到的人际问题。

---

### 番茄钟8：修身次第（25分钟）

#### 核心原文

> 「行有不得者，皆反求诸己。」

#### 南师解读要点

**1. 修身的根本**

南师指出，一切外在的问题，根源都在自己。

**2. 修身的次第**

```
修身 → 正心诚意
    ↓
齐家 → 治理家庭
    ↓
治国 → 治理国家
    ↓
平天下 → 天下太平
```

**3. 实践方法**

南师建议每天反省：今天的事有没有做好？问题出在哪里？

> **费曼自测**
>
> 请设计一个适合自己的每日反省方法。
