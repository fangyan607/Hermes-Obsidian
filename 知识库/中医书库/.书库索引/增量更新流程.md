# 增量更新流程

> 中医书库新书入库与维护规范

---

## 一、更新流程概览

```
新书入库 → 格式标准化 → 元数据标注 → 分类入库 → 技能绑定 → 刷新索引
```

---

## 二、详细操作步骤

### Step 1：新书入库
1. 将MD文件放入 `书库/中医书库/待入库/` 目录
2. 确保文件编码为UTF-8
3. 检查文件是否完整可读

### Step 2：格式标准化
按照《中医知识库AI应用方案》处理：

```yaml
---
title: 书籍/笔记名称
category: [分类编号]-[分类名称]
skill_tag: [Skill编号]
keywords: [3-8个关键词]
level: [入门/进阶/临床实操]
type: tcm
---
```

### Step 3：分类入库
确定书籍所属分类：

| 分类编号 | 分类名称 | 适用内容 |
|----------|----------|----------|
| 01 | 中医基础理论 | 阴阳五行、脏腑经络、病因病机 |
| 02 | 中医四诊 | 望闻问切、舌诊脉诊 |
| 03 | 辨证论治 | 八纲辨证、脏腑辨证、六经辨证 |
| 04 | 中药材 | 本草、药性、配伍 |
| 05 | 方剂配伍 | 经典方剂、时方、验方 |
| 06 | 经络针灸 | 经络、穴位、针灸推拿 |
| 07 | 体质辨识 | 九种体质、体质调理 |
| 08 | 各科病症 | 内外妇儿各科 |
| 09 | 食疗药膳 | 药膳食疗、饮食调理 |
| 10 | 养生康养 | 四季养生、起居调摄 |

### Step 4：技能绑定
确定书籍关联的技能模块：

| 技能 | 绑定书籍类型 |
|------|-------------|
| Skill1 | 中医基础、内经类 |
| Skill2 | 四诊、辨证类 |
| Skill3 | 本草、药材类 |
| Skill4 | 方剂类 |
| Skill5 | 针灸、经络、推拿类 |
| Skill6 | 体质类 |
| Skill7 | 临床各科类 |
| Skill8 | 食疗、养生类 |

### Step 5：更新索引
1. 更新 [[书籍元数据索引]] 添加新书记录
2. 如有新技能需求，更新 [[技能模块索引]]
3. 刷新LLM-WIKI向量索引

---

## 三、内容切片规范

对于大型书籍，按知识点拆分：

### 切片原则
- 单切片控制300-800字
- 按病机、症状、药方、取穴、调理方案独立模块
- 保持内容完整性和语义连贯

### 切片示例
```markdown
## 风寒感冒辨证要点

### 病因病机
风寒之邪外袭，肺气失宣

### 主要症状
- 恶寒重，发热轻
- 无汗，头痛
- 鼻塞流清涕
- 咳嗽痰白

### 舌脉
舌淡苔薄白，脉浮紧

### 治法
辛温解表，宣肺散寒

### 方药
荆防败毒散加减

### 穴位
风池、列缺、合谷

---
来源：《中医内科名著集成》第X章
```
