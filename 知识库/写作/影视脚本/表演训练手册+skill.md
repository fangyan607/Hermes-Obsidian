# 表演训练手册：实践美学表演法

> 表演是在舞台上和其他演员一起追求一个具体目标并真实地反应。

## 核心理念

本书由梅利莎·布鲁德等根据大卫·马梅和威廉·梅西的表演大师班笔记整理而成。核心方法论围绕**通过身体行动（Physical Action）驱动真实表演**。三个核心原则：
1. **行动即表演**——演员在舞台上总要有具体的动作
2. **此刻的真实**——在既定情境下对其他演员做出真实反应
3. **工具即习惯**——表演技能像木工手艺一样，通过练习成为习惯

## 触发条件

当用户请求以下任务时自动激活：
- 学习表演技法或准备角色
- 分析剧本中的场景和动作
- 提升即兴表演能力

## 执行流程

### Phase 1: 分析场景

**目标**：找到场景中可表演的戏剧动作

**动作**：
1. 阅读场景，确定基本情境
2. 找到角色的目标——演员要"做什么"
3. 确认动作满足9个条件：身体可做到、有趣、具体、可通过对戏演员的考验、无任务性、不预设身体或情感状态、不操控其他演员、有"结尾"、符合编剧意图

**输出**：`场景动作分析：[目标] + [动作清单]`

### Phase 2: 此刻的真实

**目标**：在场景中真实地活在当下

**动作**：
1. 基于其他演员的行为做出即时反应
2. 在既定情境框架内即兴表演
3. 不预设情感结果，信任动作本身

**输出**：`即兴准备方案：[情境框架] + [反应策略]`

### Phase 3: 外部修饰与准备

**目标**：通过外部手段增强表演

**动作**：
1. 外部修饰——声音、身体、方言
2. 准备一场戏——热身和专注
3. 解决困难——识别并克服表演障碍

**输出**：`表演准备清单：[热身] + [修饰] + [问题预案]`

### Phase 4: 持续练习

**目标**：将表演工具内化为习惯

**关键工具**：身体行动、此刻真实、外部修饰、表演准备

**检查点**：
- [ ] 动作是否身体可做？
- [ ] 动作是否有趣且具体？
- [ ] 是否在此时此刻真实反应？
- [ ] 外部修饰是否服务于内在行动？

## 局限性与来源

- **不适用于**：方法派的情感记忆训练
- **依赖条件**：需要基本的表演经验和剧本理解
