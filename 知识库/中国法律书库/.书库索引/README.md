## 一、书库概览

本法律书库为**本地离线法律法规资料库 + 法律学习书籍库**，归集各类法律、行政法规、司法解释等合规文件，以及法律教材、律师实务、法律思想等学习书籍，支持AI智能查询、案例分析、辅助判案等功能。

### 书库规模统计

| 类型 | 数量 | 占比 |
|------|------|------|
| 中国法律 | 719 | 28.8% |
| 行政法规 | 819 | 32.8% |
| 司法解释 | 877 | 35.1% |
| 法律学习书籍 | 63 | 2.5% |
| 年度案例 | 23册 | 0.9% |
