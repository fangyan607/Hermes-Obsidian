# 增量更新流程

> 新增心理书籍的入库流程与规范

## 一、更新流程概览

```
新书入库 → 格式标准化 → 元数据标注 → 分类入库 → 案例提取 → 技能绑定 → 发布更新
```

---

## 二、详细操作步骤

### Step 1：新书入库
1. 将MD文件放入 `书库/心理学书库/待入库/` 目录
2. 确保文件编码为UTF-8
3. 检查文件是否完整可读

### Step 2：格式标准化
按照《文档标准化统一规范》处理：

```markdown
# 书名：XXX

---
分类：[分类编号]-[分类名称]
作者：[作者名]
适用技能：Skill[X]
关键词：[3-8个核心词]
难度：[入门/进阶/专业临床]
权限：[基础/进阶/专业临床]
入库日期：YYYY-MM-DD
---

[原文内容...]
```

### Step 3：内容切片拆分（可选）
对于大型书籍，按知识点拆分：
- 单切片控制300-800字
- 按章节/主题/案例划分
- 便于向量检索

### Step 4：分类入库
1. 确定书籍所属分类（参考 [[书籍元数据索引]]）
2. 更新 [[书籍元数据索引]] 添加新书记录
3. 移动文件到对应分类文件夹（如需要）

### Step 5：案例提取
如果书籍包含经典案例：
1. 按 [[案例库索引]] 模板提取
2. 分配案例ID
3. 打标签入库
4. 更新案例库索引

### Step 6：技能绑定
1. 确定书籍关联的技能模块
2. 更新 [[技能模块索引]] 添加书籍引用
3. 如需新增细分技能，创建技能文档

### Step 7：发布更新
1. 更新 [[README|书库索引中心]] 的书籍总数
2. 记录更新日志
3. 如使用向量库，重新建立索引
