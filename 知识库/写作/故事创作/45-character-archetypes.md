# 经典人物原型45种

> 原型是构造栩栩如生人物的蓝图，不管是主角还是配角、正面形象还是反派角色

## 核心理念

本书基于荣格心理学和希腊神话，提炼出45种角色原型（32个男女主角原型+13个配角原型），帮助创作者探索最普遍的男性和女性原型——这些神秘的、跨文化的人物模型就是所有角色的源头。

核心价值：
- **避免30页障碍**：用原型给角色注入足够养料
- **创造立体人物**：原型提供动机、恐惧、关心的事物
- **设计性格弧线**：角色在故事中学习和成长
- **理解角色天性**：如蝎子的天性是扎青蛙

## 写作流程

### Phase 1: 选择主角原型

**目标**：确定主人公的核心原型

**步骤**：

#### Step 1: 想象你的角色
- 脸：饱满还是瘦削？
- 皮肤：娇嫩还是粗糙？
- 头发：长/短，直/卷？
- 年龄：最佳挣扎年龄？
- 体型：丰满还是精瘦？
- 风格：时髦还是土气？

#### Step 2: 回答原型识别问题
- 内向还是外向？
- 依靠本能、逻辑还是情绪？
- 想改变世界吗？
- 卧室什么样？
- 对自我外貌满意吗？
- 对家庭和孩子什么看法？

#### Step 3: 选择最吻合的原型
- 原型应该符合角色性格
- 原型应该帮助发展新方向
- 注意之前设置的不符合元素

**输出**：主角原型选择

**输出**：人物旅程文档

---

### Phase 6: 非传统运用与原型结合

**目标**：创新使用原型

**步骤**：

#### Step 13: 非传统式运用
- 原型只是向导，要有创新
- 喜剧变形：国王原型也可以是喜剧角色
- 类型混搭：亚马逊女子也可以是选美选手

#### Step 14: 结合不同原型
- 每个人都有多种原型，但有一种主导
- 压力之下主导原型接管
- 设计原型组合增加复杂性

#### Step 15: 测试原型选择
- 为三个原型各写一页大纲
- 注意每个原型如何改变情节
- 选择面对障碍最有成长空间的原型

**输出**：创新角色方案

## 原型选择决策规则

### 主角原型选择

| 角色特征 | 推荐原型 |
|---------|---------|
| 独立坚强、热爱自然 | 亚马逊女子 |
| 追求成功、注重秩序 | 商人/国王 |
| 感性浪漫、追求爱 | 诱人缪斯/妇女之友 |
| 知识渊博、理性思考 | 父亲的女儿/隐士 |
| 照顾他人、富有同情 | 养育者/照顾者 |
| 追求自由、喜欢冒险 | 愚者/追寻者 |
| 拯救他人、正义感强 | 救世主/保护者 |
| 创造表达、情感丰富 | 艺术家/创造者 |

### 配角原型选择

| 功能需求 | 推荐配角 |
|---------|---------|
| 需要帮助主角 | 忠诚伙伴/智慧导师 |
| 需要制造冲突 | 强大反派/竞争同僚 |
| 需要调节节奏 | 快乐丑角 |
| 需要体现主题 | 象征角色 |
| 需要推动成长 | 挑战者/考验者 |
