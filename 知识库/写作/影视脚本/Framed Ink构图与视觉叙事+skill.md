# Framed Ink：构图与视觉叙事

> 图像的强度应该能够传达整个故事的意图——不是告诉观众，而是让观众感受到。

## 核心理念

本书由Marcos Mateu-Mestre撰写，核心方法论围绕**通过构图传达叙事信息**展开。三个核心原则：
1. **氛围优先**：每一帧图像首先要确定情感基调和叙事意图
2. **构图即叙事**：画面元素的选择和排列本身就在讲述故事
3. **运动有目的**：镜头的每一次移动都应有叙事动机

## 触发条件

当用户请求以下任务时自动激活：
- 设计故事板或分镜
- 分析画面的视觉叙事效果
- 学习构图技巧和镜头语言
- 规划运动镜头的叙事逻辑

## 执行流程

### Phase 1: 确定叙事意图

**目标**：明确每一帧画面要传达的核心情感和信息

**动作**：回答三个基本问题

**标准问题**：
1. 这个场景的目的是什么？（信息传达/情感渲染/动作推进）
2. 主要的情感基调是什么？（喜悦/压抑/紧张/兴奋）
3. 观众应该感受到什么？

**输出**：`叙事意图陈述：[场景目的] + [情感基调] + [观众感受]`

### Phase 2: 单幅构图设计

**目标**：运用构图元素传达叙事信息

**关键构图要素**：
- **布光与明暗**：用光影塑造氛围和焦点
- **正负空间**：利用负空间创造张力和暗示
- **有序与无序**：用秩序感传递稳定或混乱
- **复杂与简单**：用复杂度控制视觉负荷
- **大与小**：用对比强调重要元素
- **折角与曲线**：用线条引导视线和创造动感
- **数量效应**：单木与丛林的视觉差异

**决策规则**：

| 条件 | 行动 | 说明 |
|------|------|------|
| 需要压抑感 | 使用俯视镜头 | 让观众俯视角色 |
| 需要霸气感 | 使用仰视镜头 | 让角色显得强大 |
| 需要情绪暗示 | 使用负空间 | 留白引发想象 |
| 需要强调焦点 | 使用超常关注点调整 | 引导观众视线 |

**输出**：`构图方案：[元素清单] + [布局说明]`

### Phase 3: 运动镜头设计

**目标**：设计有叙事动机的摄影机运动

**运动类型与叙事功能**：

| 运动类型 | 叙事功能 | 示例场景 |
|----------|----------|----------|
| 水平平移 | 展示环境、建立空间关系 | 火车站场景 |
| 垂直平移 | 表现高度、权力关系 | 史诗场景 |
| 情绪变化移动 | 反映角色内心变化 | 拜访场景 |
| 视角转换 | 揭示新信息 | 发现新大陆场景 |
| 镜头锁定 | 固定视角强调关键 | 马路餐馆对话 |

**输出**：`镜头运动方案：[运动类型] + [叙事动机] + [节奏控制]`

### Phase 4: 连续性设计

**目标**：确保多个画面之间的叙事连贯性

**关键原则**：
1. **同一视点原则**：保持视点一致性
2. **叙事高潮营造**：逐步加强到高潮
3. **连线规则**：呆在连线同一侧（180°规则）
4. **氛围递增**：通过细节积累加强氛围

**检查点**：
- [ ] 每个画面是否服务于叙事目的？
- [ ] 构图是否有效传达了情感？
- [ ] 镜头运动是否有叙事动机？
- [ ] 画面之间是否保持连续性？

**输出**：`连续性方案：[镜头序列] + [转场逻辑]`

## 决策规则

| 条件 | 行动 | 说明 |
|------|------|------|
| 角色情绪强烈 | 使用近距离构图 | 增强代入感 |
| 需要建立空间 | 使用远景/全景 | 展示环境关系 |
| 节奏需要加快 | 使用更多切分镜头 | 增加视觉节奏 |
| 需要神秘感 | 使用部分遮蔽/阴影 | 隐藏部分信息 |
| 需要揭示真相 | 使用视角转换 | 从限制视角到全知视角 |

## 检查清单

### 构图检查

**执行时机**：完成每帧构图后

- [ ] 是否有明确的视觉焦点？
- [ ] 光影是否支持叙事情绪？
- [ ] 正负空间是否合理利用？
- [ ] 画面元素是否支持叙事目的？

### 连续性检查

**执行时机**：完成镜头序列后

- [ ] 是否遵守180°规则？
- [ ] 视点变化是否有叙事动机？
- [ ] 信息递增是否合理？
- [ ] 情感曲线是否连贯？

## 局限性

- **不适用于**：纯文字剧本写作、声音设计、表演指导
- **依赖条件**：需要基本的绘画或视觉设计能力
- **可能失效**：当叙事需要故意打破视觉规则时
- **信息截止**：基于2010年原版内容
