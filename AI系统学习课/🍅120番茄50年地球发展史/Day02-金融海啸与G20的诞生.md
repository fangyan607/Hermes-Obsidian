# 🍅 Day02：金融海啸——华尔街的雪崩与G20的诞生

> 2008年，全球最大经济体差点崩溃。救它的不是市场，是政府——而且是中国政府。资本主义的心脏骤停了，东方伸出了一只手。

---

## 🍅 T1（25min）：悬疑钩子与导入

⏱ 0-5min **悬疑开场**：2008年9月15日，雷曼兄弟——一家有158年历史的华尔街巨头——宣布破产。这是美国历史上最大规模的破产案，涉及资产6390亿美元。但在雷曼倒下之前，贝尔斯登被收购了、房地美和房利美被国有化了、美林被卖了。人们以为"雷曼是最后一个"——错了，一个月后，AIG（美国国际集团）也濒临崩溃，美国政府被迫注资1820亿美元。

但最反常识的事是这个：**救美国的，不仅有美国政府，还有中国政府。** 2008年11月，中国出台了震惊世界的"四万亿"计划。这不是普通的刺激方案——它是在全球需求断崖式下跌时，中国用自己的财政空间"托住"了世界经济的底部。如果没有中国的四万亿，2009年的全球衰退可能变成第二次大萧条。

⏱ 5-20min **核心概念**：带着问题学——一场美国房地产市场的"次贷危机"怎么变成了全球金融海啸？

**传导链：五个环节，一个错误接一个错误：**

```
美国低利率(2001-2004) → 次级贷款爆炸 → 房贷证券化(MBS/CDO) → 评级机构失灵 → 雷曼倒闭 → 全球信贷冻结 → 实体经济崩溃
```

**第一环：低利率的"燃料"。** 9/11之后，美联储为了刺激经济，将联邦基金利率从6.5%一路降至1%（2003-2004）。低利率意味着借钱便宜——银行拼命放贷，百姓拼命买房。房价从2000年到2006年涨了约60%。

**第二环：次贷的"炸弹"。** 银行开始向信用记录差的人放贷——"次级贷款"。为什么敢放？因为银行不打算持有这些贷款：它们被打包成复杂的金融产品（MBS房贷抵押债券、CDO债务担保凭证）卖给了全球投资者。

**第三环：评级机构的"盲人"。** 穆迪、标普、惠誉——三大评级机构给这些垃圾债券打上了"AAA"评级（最高安全等级）。不是因为他们傻，是因为他们拿着银行的佣金。利益冲突？是的。违法？当时不违法。

**第四环：雷曼的"倒台"。** 2007年房价开始下跌，次贷违约率飙升。持有大量次贷产品的金融机构开始血亏。2008年3月贝尔斯登被摩根大通收购（美联储提供了300亿美元担保），9月7日房地美和房利美被政府接管，9月15日雷曼——美国政府决定"不救了"。

**第五环：全球"传染"。** 为什么美国的房贷危机变成了全球衰退？因为全球的银行、养老基金、保险公司都买了美国的MBS/CDO。欧洲银行持有大量美国次贷资产——英国北岩银行挤兑、德国工业银行巨亏、冰岛三大银行全部破产。**全球化的本质就是风险共享——你吃别人的肉，别人生你的病。**

⏱ 20-25min ✋ **费曼自测#1**
> **用一句话解释："为什么美国穷人买房违约，会导致冰岛国家破产？"（提示：全球化金融链条）**

## 🍅 T2（25min）：核心叙事

⏱ 0-20min **深度叙事**：以事件+数据+因果链展开

**关键事件时间线：**

```
2007.4 — 新世纪金融公司破产（次贷危机第一枪）
2008.3 — 贝尔斯登被收购（摩根大通，每股2美元——从30美元到2美元）
2008.9.7 — 房利美+房地美被政府接管（"两房"持有或担保5.4万亿美元房贷）
2008.9.15 — 雷曼兄弟破产（158年老店，灰飞烟灭）
2008.9.16 — AIG被国有化（美国政府注资1820亿美元）
2008.9.29 — 国会否决7000亿救市方案（道琼斯暴跌777点，史上最大单日跌幅）
2008.10.3 — 国会通过TARP救市方案（7000亿美元）
2008.11 — 中国宣布四万亿计划
2008.11.15 — 首届G20领导人峰会（华盛顿）
2009.4 — G20伦敦峰会（1.1万亿美元救市承诺）
```

**最深刻的转折：从G7到G20**

2008年之前，全球经济治理是"富人俱乐部"说了算——G7（美国、日本、德国、英国、法国、意大利、加拿大）。它只占全球GDP的约50%了，却决定100%的规则。

2008年11月15日，G20领导人首次聚首华盛顿。注意"首次"——这个成立于1999年的"财长级会议"在2008年升级为"首脑级峰会"。从那一刻起，**世界经济的"治理权"从G7转移到了G20。**

这意味着什么？**中国、印度、巴西、俄罗斯、南非——全球最大的发展中国家们——第一次坐到了全球经济治理的圆桌前。** 这不是西方施舍的座位，是金融危机逼出来的座位——没有这些国家的参与，危机根本解决不了。

**中国的四万亿：为什么这么重要？**

2008年11月5日，中国宣布"四万亿"（约5860亿美元）经济刺激计划，占2008年中国GDP的约13%。这个规模在历史上是空前的：

- 投向：保障性安居工程（4000亿）、农村基础设施（3700亿）、交通基建（1.5万亿）、灾后重建（1万亿）、自主创新（1600亿）……
- 效果：2009年中国GDP保住了9.4%的增长，而世界GDP下降了0.1%
- 溢出效应：中国对全球经济增长的贡献率在2009年达到惊人的50%以上——**全世界都在衰退，只有中国在扩张，而且扩张得足以拉动全球经济。**

⏱ 20-25min ✋ **费曼自测#2**
> **对比型问题：G7和G20有什么本质区别？请从"成员构成"、"决策逻辑"、"合法性"三个维度对比。为什么2008年之前G7说了算，2008年之后G20必须登场？**

## 🍅 T3（25min）：世界村民角色演绎

⏱ 0-15min **角色刻画**：今天三个核心角色

### 美国（村长家）——华尔街的雪崩

2008年的美国，就像一个表面风光但满屁股债的富翁——开着豪车、住着豪宅，但车是租的、房是按揭的、信用卡额度全用完了。然后市场（银行）突然告诉他："你的信用不够了，还钱。"

他的三波崩溃：
1. **金融系统的神经断裂**——信贷市场冻结，连通用电气这样的蓝筹公司都借不到短期资金（因为它有金融子公司）。一个不能借钱的经济体，等于一个人没有血液。
2. **从"救不救"到"必须救"**——雷曼不救，全球恐慌；AIG不救，全球保险体系崩塌；花旗不救，全球美元体系瘫痪。最终美国政府只能"国有化"金融系统——这是资本主义最讽刺的画面。
3. **身份的裂缝**——他不得不承认：没有外面的帮手，他一个人搞不定。他去G20峰会时，不再是"发号施令的村长"，而是"寻求合作的老大"。

### 欧洲（乡绅家族）——被波及的体面人

欧洲这家人，自己没干坏事（没搞次贷），却遭了殃。因为他们买了美国卖的"有毒资产"。

最惨的案例：**冰岛**——一个人口32万的小国，其三大银行的资产规模达到全国GDP的10倍。2008年10月，冰岛所有银行同时破产，国家"个人破产"。冰岛总理说的话很辛酸："我们准备好向世界借钱了——但世界没有钱借给我们。"

英国北岩银行挤兑事件（2007年9月）——这是150年来英国第一次银行挤兑。储户在银行门口排长队取钱，电视画面传遍全球。

德国、法国、意大利——无一幸免。欧洲的救市方案虽然规模也不小，但各国各自为战，缺乏协调。

### 中国（村东头老中家）——危机中的"稳定器"

2008年的中国，刚办了奥运会（8月），"同一个世界，同一个梦想"的口号还挂在鸟巢上空。两个月后，华尔街崩了。

老中家的反应是教科书级的——不是恐慌，不是退缩，而是**果断出击**：

- **40天内出方案**：从危机爆发（9月15日雷曼破产）到四万亿出台（11月5日），中国只用了不到50天。这在拥有复杂决策体系的治理体制中，堪称"极速反应"。
- **方向明确：向内求**：不能指望出口了（美国欧洲都在衰退），那就修自己的路、盖自己的房、补农村的课。
- **战略意义**：四万亿的本质不是"经济刺激"，而是"信心信号"——中国证明了自己是一个"负责任的大国"，有能力也有意愿为全球经济提供公共产品。

⏱ 15-25min ✋ **角色代入练习**
> **如果你是2008年11月的中国总理温家宝：在全球金融危机最黑暗的时刻，你要在G20峰会上发言。你可以承诺帮助世界，但也要守住中国自己的底线。请写一段150字的"G20峰会发言"——要做到三点：给世界信心、守住中国利益、体现大国担当。你觉得这三者之间有矛盾吗？怎么平衡？**

## 🍅 T4（25min）：刻意练习——三循环

🎯 **核心技能：分析金融危机传导链，标出中国的关键作用**

===== 循环1（模仿）：=====
**照着框架走一遍：** 画一条金融危机传导链，从起因到结果：

```
美国次贷危机 → 全球金融海啸 → 实体经济衰退 → G20机制化 → 中国四万亿 → 全球复苏
```

在纸上画出这条链，每个箭头旁边标注"为什么"。例如：
- 美国次贷危机 → 全球金融海啸：因为MBS/CDO卖给了全球投资者
- 全球金融海啸 → 实体经济衰退：因为银行不敢放贷，企业拿不到钱

===== 循环2（变式）：=====
**换角度再分析：** 现在换成"中国的视角"重画这条链。标出中国在每个环节：
- 危机前：中国持有大量美国国债（截至2008年约7000亿美元）——中国"被绑架"了？
- 危机中：四万亿出台——是"被迫救美国"还是"主动救自己"？
- 危机后：中国在世界经济中的话语权是如何提升的？

**关键思考：** 中国持有美国国债，美国倒了对中国有什么好处？一个需要中国帮助才能渡过难关的美国，才是对中国最有利的美国。这不是阴谋论，是理性博弈。

===== 循环3（创造）：=====
**用框架解释真实事件：** 基于你学到的金融危机传导机制，回答：

> 2020年新冠疫情期间，为什么美联储必须"无限量QE"（量化宽松）？2008年学到教训是什么？如果你是美国央行行长，2020年你会怎么做？请用"信贷冻结→流动性危机→实体经济衰退"的框架来分析。

## 🍅 T5（25min）：费曼输出 × 预习钩子

⏱ 0-10min ✋ **费曼输出**
> **"用最简单的语言向一个朋友讲今天学到的内容"**

试试这样说：2008年，美国银行乱放贷给买不起房的人，然后把烂账打包成复杂产品卖给全世界。房价一跌，整个系统炸了——美国最大投行雷曼兄弟破产，全球信贷冻结。救场的是各国政府——美国7000亿，中国四万亿。更重要的是危机催生了G20峰会，从此全球大事不再是G7说了算，发展中国家也上了桌。**这场危机暴露了西方金融体系的脆弱性，也宣告了中国作为全球经济"稳定器"角色的正式登场。**

⏱ 20-25min **预习钩子**
> 🌅 **明天预告：** 金融危机之后，美国在疗伤、欧洲在挣扎、全球在重组。但有一个故事，它的影响比金融海啸更深远——一个占世界人口五分之一的国家，在40年内完成了西方200年的工业化。2010年，它的GDP超越了日本，成为世界第二。然后，它提出了一个让世界既兴奋又警惕的计划——**一带一路。明天，我们讲中国崛起与一带一路。**
