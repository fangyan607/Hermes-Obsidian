# 表演的艺术：斯特拉·阿德勒的22堂表演课

> 表演不仅仅是关于你自己的事儿——演员要变得强大，去发现自我情感结构的本质，进而推及他人。

## 核心理念

本书由斯特拉·阿德勒（Stella Adler）撰写，核心方法论围绕**演员通过动作和想象力创造真实**。三个核心原则：
1. **表演就是去做**：演员通过行动而非空想来创造角色
2. **舞台世界不是你的世界**：演员必须进入剧本的规定情境
3. **演员是贵族也是战士**：演员需要高贵的内在品质和战斗的意志力

## 触发条件

当用户请求以下任务时自动激活：
- 学习表演方法论
- 准备角色和剧本分析
- 提升表演技巧和舞台表现力

## 执行流程

### Phase 1: 基础准备

**目标**：建立演员的基本素质和心态

**动作**：
1. 走上舞台的第一步——放下外部世界，聚焦自我
2. 理解舞台世界与真实世界的区别
3. 开发想象力——五感训练

**输出**：`演员状态设定：[专注度] + [想象力激活]`

### Phase 2: 动作研习

**目标**：建立动作的词汇表，使动作可行

**动作分类**：
- 研习动作的本质
- 使动作可行——从理解到执行
- 建立动作词汇表
- 将动作复杂化——赋予动作层次

**决策规则**：

| 条件 | 行动 |
|------|------|
| 动作不够清晰 | 拆分到最小的可行单元 |
| 动作缺乏层次 | 加入次级目标和障碍 |
| 动作过于外在 | 回到内在理由（瞬间及内在的理由） |

**输出**：`动作方案：[动作清单] + [执行层次]`

### Phase 3: 角色塑造

**目标**：从理解文本到塑造完整角色

**动作**：
1. 理解文本——分析剧本的结构和潜台词
2. 识别角色的元素——背景、动机、目标
3. 为角色着装——外部形象与内在的统一
4. 学习角色的节奏——语速、步态、呼吸

**输出**：`角色档案：[背景] + [动机] + [目标] + [外部特征]`

### Phase 4: 舞台实践

**目标**：将准备转化为舞台表演

**动作**：
1. 使装束真实——从内到外的统一
2. 获得表演的控制力
3. 在舞台上刻画阶层

**检查点**：
- [ ] 是否真正理解了角色的世界？
- [ ] 动作是否有内在理由支撑？
- [ ] 角色的外部特征是否与内在一致？
- [ ] 是否能在舞台上自如控制表演？

## 决策规则

| 条件 | 行动 | 说明 |
|------|------|------|
| 表演过于内敛 | 强化外部动作 | 用身体带动情感 |
| 表演过于外放 | 回归内在理由 | 找到动作的根源 |
| 角色缺乏深度 | 扩展角色的历史 | 挖掘角色的过去 |
| 台词缺乏力度 | 分析文本的节奏和韵律 | 找到语言的力量 |

## 局限性

- **不适用于**：纯影视表演的镜头前技术、商业表演速成
- **依赖条件**：需要基本的表演基础和理解力
- **信息截止**：基于斯特拉·阿德勒教学手稿和课堂笔记
