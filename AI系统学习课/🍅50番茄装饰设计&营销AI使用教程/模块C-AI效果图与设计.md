## 🍅 番茄21：AI绘图原理——Diffusion模型通俗版

### 📖 费曼三句话

1. **AI绘图 = "从一堆沙子慢慢雕出雕像"**——AI一开始看到的全是随机噪点（像电视雪花屏），然后一步一步"去噪"，每一步都让画面更清晰一点，最后变成一张完整的图。这个过程叫"扩散"（Diffusion）。
2. **AI画图需要两个东西**：①一个"画面描述"（你的提示词）；②一个"审美参考"（它在海量图片中学到的"什么是好看的客厅"）。两者结合，生成符合描述的图片。
3. **AI绘图的三个关键参数**：提示词（描述画面）、种子值（控制随机性，同一个种子+同一提示词=同一张图）、步数（去噪的精细度，步数越多越精细，但超过一定限度就没区别了）。

### 🛠 刻意练习

1. 打开你已有的AI绘图工具
2. 用同一个提示词生成两次，观察两张图的差异
3. 了解"seed"参数是什么（不同工具的名字可能不同）
4. 费曼检验：用自己的话解释AI绘图原理

### ✅ 完成标准

- [ ] 用同一提示词生成两次，观察到差异
- [ ] 知道"种子值"的作用
- [ ] 能用自己的话解释Diffusion原理

## 🍅 番茄22：装饰设计提示词——空间描述的核心词汇

### 📖 费曼三句话

1. **AI绘图的提示词 = "给画师的形容词清单"**——你不需要会画画，但需要知道哪些词能让AI画出你想要的效果。风格词（北欧/日式/工业）、材质词（原木/大理石/水泥）、光线词（自然光/暖光/氛围光）。
2. **装饰效果图提示词公式**：风格 + 空间 + 材质 + 颜色 + 光线 + 视角 + 氛围。缺少任何一项，AI就会"猜"，而它猜的未必是你想要的。
3. **最有用的三个魔法词**："极简"（减少杂乱元素）、"特写"（聚焦局部展现细节）、"日落暖光"（立即让画面有氛围感）。这三个词几乎适用于所有室内场景。

### 🛠 刻意练习

**创建你的 "装饰绘图提示词词库"**（笔记或文档保存）：

| 分类 | 词汇 | 效果说明 |
|:----|:-----|:--------|
| 风格 | 北欧简约、日式禅意、工业loft... | 确定整体调性 |
| 空间 | 客厅、卧室、厨房、卫生间、阳台... | 明确场景 |
| 材质 | 原木、大理石、水磨石、藤编、亚麻... | 决定质感 |
| 颜色 | 莫兰迪色系、暖白、深灰、原木色... | 定色调 |
| 光线 | 自然光、暖黄灯光、氛围灯带、日落光... | 造氛围 |
| 视角 | 广角、特写、俯视、人眼高度... | 定构图 |
| 氛围 | 温馨、宁静、通透、精致、自然... | 定情感 |

**练习**：用"风格+空间+材质+颜色+光线+视角+氛围"组合一个提示词，生成一张图

### ✅ 完成标准

- [ ] 创建了你的提示词词库（至少20个词）
- [ ] 用完整公式生成了一张不错的效果图
- [ ] 知道每个分类词汇对画面的影响

## 🍅 番茄23：Midjourney装饰设计实战（上）——基础操作

### 📖 费曼三句话

1. **Midjourney在装饰行业是"效果图神器"**——它对室内场景的理解远超其他AI绘图工具，尤其是光影和材质表现。虽然要付费（$10/月起），但效果值得。
2. **MJ的核心操作就四个命令**：`/imagine`（生成图片）、`/blend`（融合两张图）、`/describe`（从图反推提示词）、`/settings`（设置参数）。你80%的时间只用`/imagine`。
3. **MJ的重要参数**：`--ar 16:9`（横屏，适合方案展示）、`--ar 3:4`（竖屏，适合小红书）、`--v 6`（最新版本模型，效果最好）、`--s 250`（风格化程度，越高越艺术，越低越写实）。

### 🛠 刻意练习

**如果已订阅MJ**：

1. 输入 `/imagine`，写一个装饰效果图提示词：
   ```
   Scandinavian living room, white walls, light oak flooring,
   gray sofa, greenery, large windows, natural light,
   minimalist, cozy, interior design --ar 16:9 --v 6
   ```
2. 使用 U1-U4 放大选中的图，V1-V4 生成变体
3. 试着改参数 `--s 50` vs `--s 500` 感受风格化差异

**如果用国产工具**：

1. 用同样的中文描述在通义万相/文心一格生成
2. 注意国产工具的参数设置（通常叫"风格强度"或"创意度"）

## 🍅 番茄24：Midjourney装饰设计实战（下）——高级技巧

### 📖 费曼三句话

1. **MJ的"图生图"是装饰行业最强大的功能**——拍一张客户的旧房照片，让MJ"在保持结构不变的情况下换成北欧风格"。客户看到自己家的改造效果，比任何案例图都有说服力。
2. **操作方法**：用 `/imagine` + 上传原图 + 提示词。格式：`[图片链接] Scandinavian style renovation, same room layout, light colors, wooden flooring --ar 16:9`。MJ会尽量保留原图结构，只改变风格。
3. **"反推提示词"是偷师利器**——看到一张好看的装修图，用 `/describe` 上传，MJ会反推4个提示词。用这些提示词去生成，学别人的风格。

2. 用"图生图"方法让MJ生成改造效果图
3. 用 `/describe` 分析一张你喜欢的装修图，看看它的提示词是什么
4. 对比：提示词的"翻译版"和你自己写的有什么区别

## 🍅 番茄26：从方案文字到效果图的全流程

### 📖 费曼三句话

1. **"文字方案 → 效果图"的标准流程**：大模型写方案描述 → 提取关键词 → 组装成AI绘图提示词 → 生成效果图 → 选最好的那个 → 微调 → 发给客户。
2. **关键技巧：让大模型帮你写绘图提示词**——你不需要从零写绘图提示词。把方案文字发给大模型，说"请把这段方案描述转换成英文AI绘图提示词"，它就能帮你写好。
3. **同一方案至少生成4张图让客户选**——AI的成本接近零（电费/算力），多生成几张让客户挑，比只给一张"你最喜欢哪个风格"的参与感强10倍。

### 🛠 刻意练习

**完整跑通一次"文字→效果图"流程**：

```
用番茄17的方案→大模型转提示词→AI绘图→选图→发给客户的版本
```

1. 用模块B学到的技巧生成一段方案描述
2. 让大模型转成AI绘图提示词
3. 在AI绘图工具中生成至少4张效果图
4. 选1-2张最好的，微调后保存

### ✅ 完成标准

## 🍅 番茄27：效果图优化——文生图后的微调技巧

### 📖 费曼三句话

1. **第一版效果图很少完美——这不是你的问题，是AI的常态**。专业设计师用MJ也要生成5-10次才能选到满意的。关键是知道"往哪个方向调"。
2. **常见问题 + 解决方案**：
   - "颜色不对" → 加颜色词（"浅灰墙面，原木色地板"）
   - "光线太暗" → 加光线词（"大窗户，自然光，明亮"）
   - "太杂乱" → 加"极简""少装饰""大面积留白"
   - "不像我家" → 上传照片用图生图
3. **"局部重绘"是最省时间的优化方式**——画面80%满意，20%要改（比如换个沙发颜色），用局部重绘功能圈出要改的区域，只改那部分，不用全部重来。

### 🛠 刻意练习

1. 找一张你之前生成的"基本满意但不够好"的效果图
2. 分析问题：颜色/光线/风格/摆设？
3. 针对问题修改提示词关键词，重新生成
4. 对比优化前后的差距

### ✅ 完成标准

- [ ] 识别了至少1个之前效果图的问题
- [ ] 通过修改提示词优化了效果图
- [ ] 理解了"问题→解决方案"的对应关系

## 🍅 番茄28：适老化改造专项设计

### 📖 费曼三句话

1. **适老化改造AI出图的核心差异**——不是"好看"，而是"安全+方便"。提示词里要加入：扶手（grab bars）、防滑地面（non-slip flooring）、无障碍（wheelchair accessible）、宽敞（spacious for mobility）。
2. **适老化改造的三个关键空间**：卫生间（最危险，防滑+扶手+淋浴凳）、厨房（拉篮+可调节台面+防烫）、卧室（床两侧通道+夜灯+紧急呼叫）。
3. **AI在适老化上的独特价值**：你可以生成"同一空间、不同改造程度"的对比图——"最低成本版"vs"标准版"vs"全配版"，让客户根据预算选择。一图胜千言。

### 🛠 刻意练习

1. 生成一组适老化卫生间改造效果图（至少3张）

   ```

   不锈钢扶手，淋浴座椅，宽敞通道，暖光照明，
   北欧简约风格 --ar 16:9
   ```
3. 试做一组对比：改造前 vs 低成本改造 vs 全配改造

## 🍅 番茄29：方案展示板制作——Canva排版实战

### 📖 费曼三句话

1. **方案展示板（Mood Board）= 给客户的"视觉方案书"**——一张A4大小的图，左边放改造前照片，中间放效果图，右边放材料样板+文字说明。客户看一张图就理解全部方案。
2. **Canva排版的三原则**：左中右或上下结构清晰 + 不超过3种字体 + 留白30%。不要塞太满，客户需要"呼吸感"。
3. **AI排版提示词**："请为一套客厅改造方案设计一个展示板布局描述，包含改造前照片、效果图、材料清单和设计说明的位置。"然后按AI的建议在Canva里实现。

### 🛠 刻意练习

1. 在Canva搜索模板"装修方案"或"室内设计"
2. 用你之前生成的效果图+方案文字，制作一张方案展示板
3. 展示板应包含：标题、效果图、改造要点、材料清单
4. 导出为PDF或高清图片

### ✅ 完成标准
