## 一、大类SKILL（核心技能）

### Skill1：心理学基础理论检索
- **触发指令**：基础心理、认知原理、心理常识、概念解释、思维系统、认知科学
- **调取书库**：基础心理学类全部书籍（18本）+ 思维认知类（20本）
- **功能能力**：名词释义、理论溯源、心理规律讲解、认知原理
- **新增书籍**：《心理学讲义》《现代心理学》《元认知》《深度思考》《批判性思维》等

### Skill2：心理症状&临床诊断判定
- **触发指令**：症状判断、心理疾病筛查、异常行为分析、变态心理
- **调取书库**：变态心理学、临床诊断类书籍（12本）
- **功能能力**：症状匹配、轻重区分、判定依据、临床评估
- **新增书籍**：《变态人格心理分析》《天生变态狂》《犯罪心理学》《走出抑郁症》

### Skill3：心理咨询话术技能
- **触发指令**：咨询对话、共情回应、来访者沟通、团体咨询
- **调取书库**：心理咨询技术类书籍（10本）
- **功能能力**：标准话术、沟通纠正、模拟对话、团体辅导
- **新增书籍**：《团体咨询的策略与方法》《团体心理治疗》《心理分析：理解与体验》

### Skill4：主流心理疗法实操应用
- **触发指令**：CBT疗法、精神分析、正念疗愈、行为矫正、存在主义治疗
- **调取书库**：各类疗法专著（25本）
- **功能能力**：疗法步骤、实操流程、案例套用
- **新增书籍**：《思维改变生活》《现代精神分析"圣经"》《分析心理学的理论与实践》《心理危机及成人心理学》

### Skill5：情绪疏导&压力调节技能
- **触发指令**：焦虑缓解、抑郁情绪、暴躁平复、压力管理、情绪急救
- **调取书库**：情绪管理类书籍（18本）
- **功能能力**：调节方案、放松方法、疏导思路
- **新增书籍**：《如何控制负面情绪》《心流》《极简思维》《逆境成长》《自控力》

### Skill6：心理危机干预技能
- **触发指令**：轻生倾向、创伤心理、应激障碍、孤独心理
- **调取书库**：危机干预、创伤修复书籍（4本）
- **功能能力**：紧急干预话术、风险评估、修复方案
- **新增书籍**：《孤独心理学》《如何活出生命的意义》

### Skill7：人际婚恋家庭心理指导
- **触发指令**：亲子矛盾、婚姻情感、人际关系、沟通技巧
- **调取书库**：家庭婚恋、社会人际心理书籍（30本）
- **功能能力**：关系剖析、改善方案、情感解读
- **新增书籍**：《外遇心理学》《爱情心理学》《罗曼蒂克心理学》《网络爱情》《非暴力沟通》《微动作心理学》

### Skill8：人格分析&性格测评解读
- **触发指令**：性格分析、人格特质、人格障碍、自我认知
- **调取书库**：人格心理学书籍（8本）+ 绘画心理（3本）
- **功能能力**：性格成因、优缺点、适配建议、投射分析
- **新增书籍**：《性格解析》《请理解我》《心理画外音》《心理画绘画心理分析图典》

### Skill11 ⭐：心理学史研究（新增）
- **触发指令**：心理学史、心理学流派、心理学思想史、中国古代心理学、外国心理学流派
- **调取书库**：心理学史与人物辞典类（6本）
- **功能能力**：
  - 心理学历史溯源：从古希腊到现代的发展脉络
  - 心理学流派分析：构造主义、机能主义、格式塔、行为主义、认知心理学等
  - 中国心理学思想：古代心理学思想史、近现代心理学发展
  - 外国心理学思想：古希腊罗马、近代欧洲心理学思想
  - 心理学家传记：301位世界著名心理学家生平与贡献
- **核心书籍**：
  - 《心理学通史 第1卷：中国古代心理学思想史》- 先秦至明清心理学思想
  - 《心理学通史 第2卷：中国近现代心理学史》- 近代启蒙至现代发展
  - 《心理学通史 第3卷：外国心理学思想史》- 古希腊罗马至近代思想
  - 《心理学通史 第4卷：外国心理学流派（上）》- 意识心理学、构造心理学
  - 《心理学通史 第5卷：外国心理学流派（下）》- 格式塔、皮亚杰、认知心理学
  - 《世界著名心理学家辞典》- 301位心理学家传记
- **应用场景**：
  - 学术研究：心理学理论溯源、流派发展研究
  - 教学备课：心理学史课程设计、教学内容准备
  - 理论理解：理解当代心理学理论的历史根源
  - 学科定位：把握心理学学科发展脉络与趋势

### Skill12 ⭐：心理防御机制分析（新增）
- **触发指令**：心理防御分析、防御机制识别、否认投射压抑、情绪回避、自我欺骗
- **调取书库**：《为什么我们总是在防御》（约瑟夫·布尔戈）
- **功能能力**：
  - 防御机制识别：识别10种主要心理防御机制
  - 功能评估：判断防御的适应性与僵化性
  - 防御解除：提供解除无益防御的方法
  - 自我探索：深入探究潜意识模式
- **核心书籍**：
  - 《为什么我们总是在防御》- 10种心理防御机制详解
- **技能文档**：[[书库/心理学书库/.书库索引/skills/心理防御机制分析技能|心理防御机制分析技能]]
- **应用场景**：
  - 心理咨询：识别来访者的防御模式
  - 自我成长：觉察自己的防御机制
  - 关系分析：理解关系中的防御互动

### Skill13 ⭐：中年心理转型（新增）
- **触发指令**：中年危机、中年转型、人生下半场、职业倦怠中年、意义危机、阴影整合
- **调取书库**：《中年之路：人格的第二次成型》（詹姆斯·霍利斯）
- **功能能力**：
  - 中年危机解读：理解中年困惑的深层心理意义
  - 人格转型指导：指导从"第一人格"到"真我"的转变
  - 意义重建：帮助寻找人生下半场的意义
  - 阴影整合：促进未发展人格面向的整合
- **核心书籍**：
  - 《中年之路：人格的第二次成型》- 荣格派中年转型理论
- **技能文档**：[[书库/心理学书库/.书库索引/skills/中年心理转型技能|中年心理转型技能]]
- **应用场景**：
  - 中年咨询：帮助来访者理解中年转型
  - 人生规划：探索人生下半场的方向
  - 职业转型：评估职业转型需求
  - 关系重构：处理中年关系挑战

## 三、技能调用规则

### 权限绑定
```yaml
基础技能: [普通用户]
  - Skill1: 基础理论检索
  - Skill5: 情绪疏导（基础内容）
  - Skill8: 人格分析（入门内容）
  - Skill11: 心理学史研究（基础内容）
  - 思维训练技能包

进阶技能: [进阶用户]
  - Skill3: 咨询话术
  - Skill7: 婚恋家庭指导
  - Skill11: 心理学史研究（完整内容）
  - 细分技能: 全部可访问

专业技能: [从业者]
  - Skill2: 临床诊断
  - Skill4: 心理疗法
  - Skill6: 危机干预
  - 所有临床专业内容
```

### 调用优先级
1. 精准匹配：用户指令直接触发对应技能
2. 语义理解：AI识别意图，匹配最相关技能
3. 多技能联动：复杂问题调用多个技能组合

### 技能联动示例

| 用户问题 | 调用技能组合 |
|----------|--------------|
| "我总是拖延怎么办？" | Skill5情绪疏导 + 职场技能包(拖延症应对) |
| "孩子不爱学习，我该怎么沟通？" | 儿童技能包(亲子沟通) + Skill7婚恋家庭 |
| "我觉得自己很没用" | Skill2临床诊断 + Skill8人格分析 + Skill5情绪疏导 |

| "婚姻出了问题，想修复" | 婚姻技能包(亲密关系修复) + Skill7婚恋家庭 |
| "弗洛伊德的理论是什么？" | Skill11心理学史研究 + Skill4心理疗法 |
| "中国古代有心理学吗？" | Skill11心理学史研究（第1卷） |
| "认知心理学是怎么发展起来的？" | Skill11心理学史研究（第5卷） + Skill1基础理论 |
| "我总是否认自己的问题" | Skill12心理防御机制分析 + Skill4心理疗法 |
| "中年了觉得人生没意义" | Skill13中年心理转型 + Skill4心理疗法 |
