# 技能模块索引

## 道医全书技能系统

本文档定义了道医全书书库的所有技能模块及其触发词。

---

## 技能模块列表

### 技能1：道医文献查询

**技能ID**：`daoyi-query`
**触发词**：
- `/道医查询`
- `/道医全书`
- `/daoyi-query`

**功能描述**：
快速检索道医全书中的特定内容，支持按关键词、书名、作者等维度查询。

**使用示例**：
```
/道医查询 脉诊
/道医全书 黄帝内经
```

**适用范围**：
- 全部106册内容
- 支持模糊匹配和精确查询

---

### 技能2：中医古籍检索

**技能ID**：`tcm-classics`
**触发词**：
- `/古籍检索`
- `/中医文献`
- `/tcm-classics`

**功能描述**：
按书名、作者、朝代、部类等维度检索古籍信息。

**检索维度**：
| 维度 | 说明 | 示例 |
|------|------|------|
| 书名 | 古籍名称 | 《濒湖脉学》 |
| 作者 | 撰写者姓名 | 李时珍 |
| 朝代 | 成书时代 | 明代 |
| 部类 | 所属分类 | 医部·脉诊 |

**使用示例**：
```
/古籍检索 作者:李时珍
/中医文献 朝代:明代 部类:脉诊
```

### 技能3：道医知识问答

**技能ID**：`daoyi-qa`
**触发词**：
- `/道医问答`
- `/道医咨询`
- `/daoyi-qa`

**功能描述**：
基于道医全书内容的智能问答系统，提供专业的道医知识解答。

**问答范围**：
- 医学理论（阴阳五行、脏腑经络）
- 诊法技术（望闻问切、脉诊舌诊）
- 本草方剂（药物性味、方剂配伍）
- 养生功法（导引、吐纳、存想）
- 内丹修炼（炼精化气、炼气化神）

**使用示例**：
```
/道医问答 什么是三焦？
/道医咨询 如何理解"肾藏精"？
```

## 技能组合使用

多个技能可以组合使用，以获得更全面的信息：

```
/道医查询 肾虚 /方剂查询 /养生功法
```

这将返回：
1. 肾虚的理论解释
2. 治疗肾虚的方剂
3. 养肾的功法指导
