# 剧本：影视写作的艺术、技巧和商业运作

> 写作是扮演上帝——编剧是电影里排第一位的艺术创作者，仅仅因为他/她是：第一位。

## 核心理念

本书由理查德·沃尔特撰写，核心方法论围绕**剧本写作的三位一体：艺术、技巧、商业运作**。编剧不是天才的灵光一现，而是可以训练和学习的系统工艺。

## 执行流程

### Phase 1: 艺术——创意选择
七个被误解的词、观众定位、创意选择（点子/故事/主题/身份）、冲突设计（暴力与性）、幻觉（真实与电影）

### Phase 2: 技巧——执行方法
故事装配、角色塑造、对白写作、动作与场景设置、格式规范

### Phase 3: 商业运作
合作与协作、剧本销售策略、经纪人关系、市场定位

## 核心原理

1. 所有作家都憎恶写作
2. 写作是扮演上帝
3. 找到一个经纪人不难，难的是写出一部值得经纪人推销的剧本
4. 写作是精神分裂的过程

## 局限性

- **适用于**：电影和电视写作入门及进阶
- **不适用于**：实验性/先锋派影视创作
