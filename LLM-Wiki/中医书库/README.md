## 二、如何使用

### 自动触发

当你的问题包含以下关键词时，系统会自动激活中医知识库：

- 中医术语：中医、方剂、辨证、证型、中药、药材、本草
- 经典名称：伤寒、金匮、内经、温病
- 治疗方法：针灸、推拿、经络、穴位
- 方剂后缀：汤、丸、散、丹
- 具体方剂名：四君子汤、六味地黄丸等

### 直接调用

你也可以使用SKILL命令直接调用：

```
/tcm-knowledge-base [查询内容]
```

## 五、检索技巧

### 1. 精确查询

直接使用具体名称：
```
半夏泻心汤
消化性溃疡
太阳之为病
```

### 2. 组合查询

叠加多个条件：
```
消化性溃疡 脾胃虚寒 证型
四君子汤 加减
咳嗽 风寒 辨证
```

### 3. 按类别查询

指定书库范围：
```
在方剂大辞典中查补中益气汤
在消化病经验集成中查胃炎
在本草纲目中查人参
```
