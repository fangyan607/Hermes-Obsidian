## 三大方法论融合

| 方法论 | 作用 | 应用方式 |
|--------|------|----------|
| **番茄工作法** | 时间节奏管理 | 每日1-2个番茄钟，每钟25分钟专注学习 |
| **费曼学习法** | 理解深度保障 | 每个知识点后设「费曼自测」，用自己的话复述 |
| **刻意练习** | 技能固化提升 | 每天设置练习任务，走出舒适区 |

## 每周学习结构

### 一周时间分配（8个番茄钟）

| 日期 | 番茄钟 | 学习内容 |
|------|--------|----------|
| 周一 | 🍅🍅 | 概念理解：建立心智模型 |
| 周二 | 🍅🍅 | 原文精读：核心章节细读 |
| 周三 | 🍅🍅 | 思考深化：联系现实应用 |
| 周四 | 🍅🍅 | 复习输出：费曼自测 + 写作 |
| 周五 | 📝 | 周总结 + 刻意练习任务 |

### 单个番茄钟结构（25分钟）

```
📖 阅读理解（10分钟）
   └─ 精读书籍章节

🤔 费曼自测（8分钟）
   └─ 用自己的话复述核心概念

✍️ 笔记整理（5分钟）
   └─ 记录要点、疑问、感悟

⏸️ 休息（2分钟）
   └─ 短暂放松，准备下一个番茄钟
```
