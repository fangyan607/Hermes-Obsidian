# 易经杂说 · 一周学习计划

> **书名**：易经杂说
> **作者**：南怀瑾
> **学习周期**：1周（8个番茄钟，约3.5小时）
> **学习方法**：番茄时钟 + 费曼学习法 + 刻意练习

---

## 书籍概览

### 作者背景

南怀瑾先生对《易经》的通俗讲解，以浅显易懂的方式介绍《易经》的基本原理、卦象含义、占卜方法等。

### 核心主题

本书是《易经》入门的最佳读物，涵盖阴阳、八卦、六十四卦的基本知识，以及《易经》的哲学思想。

### 学习目标

完成本周学习后，你将能够：
- 理解《易经》的基本原理
- 掌握八卦与六十四卦的基础知识
- 用自己的话解释阴阳变化
- 将《易经》智慧应用于生活

## Day 1：易经概说

### 番茄钟1：易经是什么（25分钟）

#### 核心原文

> 「易者，变也。易经者，研究变化之书也。」

#### 南师解读要点

**1. 易经的地位**

南师介绍《易经》在中国文化中的崇高地位——群经之首。

**2. "易"的含义**

```
易 → 变化
简易 → 大道至简
变易 → 万物变化
不易 → 变中有不变
```

**3. 易经的构成**

南师介绍《易经》的构成：经、传。

> **费曼自测**
>
> 请解释"易"的三层含义。

---

### 番茄钟2：易经的起源（25分钟）

#### 核心原文

> 「伏羲画卦，文王演易，孔子作传。」

#### 南师解读要点

**1. 易经三圣**

南师介绍《易经》的形成过程：

| 伏羲 | 画八卦 |
|------|--------|
| 文王 | 演六十四卦 |
| 孔子 | 作十翼 |

**2. 伏羲画卦**

```
仰观天文 → 天象变化
俯察地理 → 地理形势
    ↓
始画八卦 → 乾兑离震巽坎艮坤
```

**3. 易经的演变**

南师讲解《易经》从八卦到六十四卦的演变。

> **费曼自测**
>
> 请讲述《易经》的形成过程。

## Day 2：阴阳之道

### 番茄钟3：阴阳的基本概念（25分钟）

#### 核心原文

> 「一阴一阳之谓道。」

#### 南师解读要点

**1. 阴阳的起源**

南师解释，阴阳源于古人对自然现象的观察。

**2. 阴阳的含义**

```
阳 → 天、日、男、刚、动
阴 → 地、月、女、柔、静
```

**3. 阴阳的关系**

南师讲解阴阳的对立统一关系。

> **费曼自测**
>
> 请举例说明阴阳的对立统一。

---

### 番茄钟4：阴阳的变化（25分钟）

#### 核心原文

> 「阳极阴生，阴极阳生。」

#### 南师解读要点

**1. 阴阳消长**

南师解释阴阳此消彼长的规律。

**2. 否极泰来**

```
泰卦 → 地天泰
否卦 → 天地否
    ↓
物极必反 → 阴阳转化
```

**3. 现代应用**

南师讲解阴阳变化在人生中的启示。

> **费曼自测**
>
> 请解释"阳极阴生"的含义。

## Day 3：八卦与六十四卦

### 番茄钟5：八卦的构成（25分钟）

#### 核心原文

> 「八卦者，乾坤震巽坎离艮兑也。」

#### 南师解读要点

**1. 八卦的名称**

南师讲解八卦的名称和象征：

| 乾 | 天 |
|----|-----|
| 坤 | 地 |
| 震 | 雷 |
| 巽 | 风 |
| 坎 | 水 |
| 离 | 火 |
| 艮 | 山 |
| 兑 | 泽 |

**2. 八卦的生成**

```
太极生两仪 → 阴阳
两仪生四象 → 太少阴阳
四象生八卦 → 八卦
```

**3. 先天与后天**

南师区分先天八卦与后天八卦。

> **费曼自测**
>
> 请默写八卦的名称和象征。

---

### 番茄钟6：六十四卦的形成（25分钟）

#### 核心原文

> 「八卦相荡，因而重之，六十四卦。」

#### 南师解读要点

**1. 六十四卦的形成**

南师解释，八卦两两相重，形成六十四卦。

**2. 卦的结构**

```
上卦 → 外卦
下卦 → 内卦
    ↓
六爻 → 从下至上
```

**3. 重要卦象**

南师简要介绍几个重要卦象的含义。

> **费曼自测**
>
> 请解释六十四卦是如何形成的。

## Day 4：易理与人生

### 番茄钟7：易理的智慧（25分钟）

#### 核心原文

> 「天行健，君子以自强不息。地势坤，君子以厚德载物。」

#### 南师解读要点

**1. 乾卦的精神**

南师解释"自强不息"的含义——刚健进取。

**2. 坤卦的精神**

```
乾 → 刚健有为
坤 → 柔顺包容
    ↓
乾坤并建 → 阴阳配合
```

**3. 人生的启示**

南师讲解《易经》对人生的指导意义。

> **费曼自测**
>
> 请解释"自强不息"与"厚德载物"的含义。

---

### 番茄钟8：知机与应变（25分钟）

#### 核心原文

> 「知几其神乎！几者，动之微，吉凶之先见者也。」

#### 南师解读要点

**1. 知机的智慧**

南师解释"知几"——预知事物变化的苗头。

**2. 应变的方法**

```
观察苗头 → 见微知著
提前准备 → 有备无患
顺应变化 → 因势利导
```

**3. 易经的核心**

南师指出，《易经》的核心是"趋吉避凶"。

> **费曼自测**
>
> 请解释"知几"的智慧。
