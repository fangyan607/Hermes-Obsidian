# 心理学123本MD书库AI检索开发利用方案
文档版本：V1.0
适用场景：本地AI知识库、智能心理咨询、心理测评诊断、专业理论检索、技能问答、实训答疑
书库基础：123本心理学MD文档，覆盖基础理论、发展心理、社会心理、人格心理、变态心理、临床诊断、心理咨询技术、危机干预、婚姻家庭、职场心理全类目

## 一、书库基础预处理（必做，适配AI检索）
### 1. 文档标准化统一规范
1. **统一MD格式标签**
每篇文档头部强制添加元数据标签，AI可精准分类检索
```md
# 书名：XXX
# 分类：基础理论/临床诊断/心理咨询/危机干预/人格分析
# 适用技能：心理评估、共情话术、障碍诊断、情绪疏导
# 关键词：3-8个核心词
# 难度：入门/进阶/专业临床
```
2. **内容切片拆分**
按知识点、案例、话术、诊断标准、疗法步骤拆分小块文本，单切片控制300-800字，适配向量检索粒度
3. **去重纠错**
剔除重复理论、老旧淘汰诊断标准，统一专业术语译名

### 2. 类目体系划分（对应123本书全覆盖）
1. 基础心理学：认知、感知、记忆、思维、动机
2. 发展心理学：儿童、青少年、中年、老年心理
3. 社会心理学：人际、从众、偏见、群体行为
4. 人格心理学：人格特质、人格障碍、性格解析
5. 变态心理学：心理异常分类、发病成因
6. 临床诊断：精神障碍筛查、症状判定、诊断依据
7. 心理咨询技术：谈话、倾听、共情、面质、释梦
8. 心理疗法：认知行为CBT、精神分析、人本、正念
9. 情绪压力管理：焦虑、抑郁、愤怒疏导调节
10. 危机干预：自杀干预、创伤应激PTSD
11. 婚恋家庭心理：亲子、婚姻矛盾、家庭疗愈
12. 职场心理：职场内耗、沟通、职场压力

## 二、核心技术架构：向量知识库+AI技能调用
整体架构：**MD文档→向量入库→分类索引→自定义SKILL技能→对话检索调用**
主流适配框架：LangChain、LLamaIndex、本地私有大模型、RAG检索增强

### 1. 向量库部署步骤
1. 批量读取全部123份MD文件，解析文本、标题、标签、关键词
2. 文本向量化嵌入，生成唯一向量索引
3. 按上方12大类目建立**分类检索索引**，缩小检索范围
4. 开启相似度阈值匹配，优先调取专业原文内容

### 2. 检索规则设定
- 基础检索：关键词模糊匹配、书名/作者精准查找
- 语义检索：理解问句含义，调取对应心理学理论
- 权限检索：入门问题调取基础书，临床问题调取诊断专著
- 溯源检索：回答结尾标注来源书籍，保证专业性

## 三、自定义SKILL技能模块开发
依托书库内容，封装8大可直接调用AI技能，每条技能绑定对应书籍知识库

### Skill1：心理学基础理论检索
**触发指令**：基础心理、认知原理、心理常识、概念解释
**调取书库**：基础心理学类全部书籍
**功能能力**：名词释义、理论溯源、心理规律讲解、知识点答疑

### Skill2：心理症状&临床诊断判定
**触发指令**：症状判断、心理疾病筛查、异常行为分析、诊断标准
**调取书库**：变态心理、临床诊断类书籍
**功能能力**：对照症状匹配病症、区分轻症/重症、给出临床判定依据

### Skill3：心理咨询话术技能
**触发指令**：咨询对话、共情回应、来访者沟通、咨询话术模板
**调取书库**：心理咨询技术、会谈技巧类书籍
**功能能力**：生成标准咨询话术、纠正不当沟通、模拟咨询对话

### Skill4：主流心理疗法实操应用
**触发指令**：CBT疗法、精神分析、正念疗愈、行为矫正
**调取书库**：各类疗法专著
**功能能力**：疗法步骤拆解、实操流程、案例套用、练习方法

### Skill5：情绪疏导&压力调节技能
**触发指令**：焦虑缓解、抑郁情绪、暴躁平复、内耗调节
**调取书库**：情绪管理类书籍
**功能能力**：给出科学调节方案、心理放松方法、情绪疏导思路

### Skill6：心理危机干预技能
**触发指令**：轻生倾向、创伤心理、应激障碍、危机安抚
**调取书库**：危机干预、创伤修复书籍
**功能能力**：紧急干预话术、风险评估、后续心理修复方案

### Skill7：人际婚恋家庭心理指导
**触发指令**：亲子矛盾、婚姻情感、人际关系、社交恐惧
**调取书库**：家庭婚恋、社会人际心理书籍
**功能能力**：关系问题剖析、相处改善方案、情感心理解读

### Skill8：人格分析&性格测评解读
**触发指令**：性格分析、人格特质、人格障碍、识人心理
**调取书库**：人格心理学书籍
**功能能力**：性格成因分析、人格优缺点、相处适配建议

## 四、三种落地使用模式
### 模式1：专业检索问答模式
用户提问→AI语义识别→匹配类目SKILL→检索123本书库原文→整合专业答案
适用：学生备考、咨询师查理论、专业知识查阅

### 模式2：模拟心理咨询对话模式
锁定心理咨询SKILL，AI以咨询师身份，调用书库会谈技术、案例、疏导方法，一对一模拟咨询

### 模式3：案例分析实训模式
输入心理案例，AI调取诊断、疗法、干预书籍内容，完成案例剖析、方案制定、问题总结

## 五、Prompt调用模板
### 通用检索调用提示词
```
你现在绑定123本心理学MD专业书库，严格依据库内内容作答。
1. 识别用户问题，匹配对应的心理学SKILL技能模块
2. 仅调取书库内理论、诊断标准、咨询技术、实操方法，不编造内容
3. 专业问题严谨客观，咨询类问题温和共情
4. 复杂问题分点作答，实操类给出清晰步骤
5. 禁止非库内玄学、偏方心理内容
```

### 心理咨询专属调用提示词
```
启用心理咨询SKILL技能，依托心理学书库会谈技术作答。
以专业心理咨询师身份沟通，耐心倾听、合理共情，
依据库内疏导方法、沟通话术、情绪干预思路回应来访者，
不做绝对病症定论，给出科学心理调整建议。
```

### 临床诊断分析提示词
```
启用临床诊断SKILL技能，参照书库变态心理、诊断标准内容，
客观分析症状表现，匹配对应心理异常类型，
说明判定依据、成因特点，给出专业评估方向。
```
