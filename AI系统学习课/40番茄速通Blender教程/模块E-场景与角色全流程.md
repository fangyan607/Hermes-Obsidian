## 🍅26 场景构建与构图（25分钟）

### 费曼输入（概念解释）

场景构建就是把"空场景"变成"完整世界"的过程。它不仅仅是用模型填满空间，更重要的是**讲故事——通过布局、灯光和色调传达氛围和情绪**。

**场景构建五步流程：**

```
参考收集 → 分镜布局 → 元素细化 → 材质光影 → 后期微调
```

1. **概念/参考收集**：找参考图（Pinterest、ArtStation、Google Images）
2. **分镜/布局**：用简单几何体（Cube/Sphere）快速确定场景构图
3. **元素细化**：逐一完善场景中的物体——细节密度：焦点区域多细节，非焦点区域简化
4. **材质与光影**：统一场景色调，建立材质层次
5. **后期微调**：合成器调色，做最后的润色

**构图原则（3D场景专用）：**

| 原则 | 说明 |
|------|------|
| **三分法** | 画面横竖三等分，重要元素放在交叉点或线上 |
| **引导线** | 利用场景中的线条（道路、墙沿、光线）引导观众视线到焦点 |
| **前后景** | 增加景深感——前景/中景/背景三个层次 |
| **黄金比例** | 非对称但平衡的布局，约 1:1.618 |

**场景氛围三要素：**

- **灯光**：决定场景的情绪——硬光 = 紧张/戏剧，柔光 = 温馨/平静
- **色调**：暖色调（橙/黄）= 温馨/紧张，冷色调（蓝/紫）= 冷静/悲伤
- **细节密度**：焦点区域多细节，非焦点区域简化——让观众知道看哪里

### 刻意练习（具体操作步骤 + 代码/命令）

**案例：创建室内小场景**

1. **布局阶段（10分钟）**
   - 用 Cube 搭建房间：墙壁 + 地板 + 天花板
   - 添加一个 Cube 做桌子，几个拉伸 Cube 做椅子
   - 添加一个 Cube 做窗户
   - 不要急着细化，先用几何体确定位置和比例

2. **三分法构图**
   - 打开相机安全框（Viewpoint > Camera > Safe Areas）
   - 三分法网格（Viewport Overlays > Guides > 勾选 Composition Guides > Thirds）
   - 将桌子和主要物体放在三分线交点

3. **添加细节**
   - 桌子：细化造型，加桌腿
   - 墙上：挂一幅画（面片 + 纹理）
   - 窗户外：加一个带有蓝色材质的背景面片

4. **灯光**
   - 窗户位置：加一盏 **Area Light**，模拟日光
   - 室内补光：一盏小 **Point Light** 模拟台灯
   - 调整强度/颜色：窗光偏蓝（冷），台灯偏橙（暖）→ 冷暖对比

5. **氛围微调**
   - 降低环境光强度
   - 给窗户光加一点点体积散射效果

### ✅ 完成标准

- [ ] 理解场景构建的五步流程（参考→布局→细化→材质→后期）
- [ ] 能用三分法和引导线规划场景构图
- [ ] 知道前后景布局增加景深感
- [ ] 理解灯光与色调共同决定场景氛围
- [ ] 能独立完成一个基础室内场景的搭建

### 📖 费曼三句话

1. 好场景 = 有层次（前中后）+ 有焦点（视觉中心）+ 有氛围（灯光色调）。
2. 三分法和引导线是构图的最基础、最有效工具——把重要元素放在交叉点，用线条引向焦点。
3. 场景构建是全流程中最需要"审美"的环节——技术可以学，审美靠大量看好的作品。

## 🍅27 角色建模基础（25分钟）

### 费曼输入（概念解释）

角色建模的两种主流方式：

| 方式 | 适合 | 特点 |
|------|------|------|
| **多边形建模（Box Modeling）** | 机械/硬表面角色 | 从基本体拉伸挤出，精确控制拓扑 |
| **雕刻建模（Sculpting）** | 生物/有机角色 | 从球体开始捏，直觉化，推荐入门 |

**Q版角色"从球开始"全流程：**

1. Shift + A → UV Sphere（细分 2-3 级）
2. 切换到 **Sculpting 工作区**
3. **Clay Strips** 笔刷：拉出身体基础形状
4. **Grab** 笔刷：调整整体比例（头/身体/四肢）
5. **Inflate** 笔刷：做胖乎乎的感觉（Q 版核心特征）
6. 手/脚/耳朵/鼻子：用 Sphere 添加，Merge 融合
7. 用 **Mask** 笔刷保护已完成的区域（画上绿色的地方不被修改）

**角色建模的核心原则：**

```
先大形后细节：比例比细节重要100倍
动态姿势比站姿好：即使静态也摆个S曲线
参考图必不可少：正面/侧面/背面三视图
```

**基础人体比例参考：**

| 风格 | 头:身比 | 说明 |
|------|---------|------|
| Q版 | 1:1 ~ 1:2 | 大头小身，可爱风 |
| 半写实 | 1:5 ~ 1:6 | 游戏角色风格 |
| 写实 | 1:7 ~ 1:8 | 真实人体比例 |

### 刻意练习（具体操作步骤 + 代码/命令）

**步骤1：准备三视图（可选）**

1. 在正交视图中（Numpad 1/3/7）添加参考图
2. Shift + A → Image → Reference → 选择正面/侧面/背面图
3. 把参考图分别放到 Front / Right / Back 视图

**步骤2：雕刻 Q 版角色（核心流程）**

1. Shift + A → UV Sphere → 切换到 Sculpting 模式
2. **Grab 笔刷**：拉出头部形状（圆圆的）
3. **Clay Strips**：在头部下方拉出身体（粗短）
4. **Grab**：拉出手臂和腿的雏形（上下左右四个方向拉出）
5. **Inflate**：在整个身体上刷一遍 → 胖乎乎的效果
6. **Smooth** 笔刷：把棱角刷圆
7. 添加眼球：两个小 Sphere 放在头部前方
8. 添加嘴巴：用 **Crease** 笔刷刻一条弧线

**步骤3：增加细节**

1. 多级细分（Remesh 或 Subdivision Modifier）
2. 用 **Clay Strips** 做衣服轮廓
3. 用 **Mask** + **Draw Face Sets** 区分不同颜色区域

### ✅ 完成标准

- [ ] 理解 Box Modeling 与 Sculpting 的区别及各自适用场景
- [ ] 掌握"从球开始"的 Q 版角色雕刻流程
- [ ] 理解"先大形后细节"的核心原则
- [ ] 掌握各人体风格的比例参考值
- [ ] 能用 Mask 保护已完成的雕刻区域

### 📖 费曼三句话

1. 角色建模从球开始——先捏整体比例，再逐步加细节，别一上来就扣细节。
2. 比例比细节重要 100 倍——比例不对，再多的细节也是丑；Q 版就是大头小身。
3. 三视图参考是角色建模的"导航地图"——没有参考就是在黑暗中盲目行走。

## 🍅28 绑定与权重（25分钟）

### 费曼输入（概念解释）

骨骼绑定（Rigging）就是给模型创建一套"骨架"，用骨架控制模型变形。**骨骼是模型的"木偶线"——移动骨骼带动模型动起来。**

**骨骼基础操作：**

| 操作 | 快捷键/方式 |
|------|------------|
| 添加骨骼 | Shift + A → Armature → Single Bone |
| 挤出延伸骨骼 | 选中骨骼末端 → **E** 挤出 |
| 自动绑定到模型 | 先选模型 → 再选骨架 → **Ctrl + P → With Automatic Weights** |

**绑定核心概念：**

- **骨骼链（Bone Chain）**：从父级到子级的层级结构——移动父骨，子骨跟着动
- **IK（反向动力学）**：定位手/脚的位置，关节自动弯曲——适合"把手放在桌上"
- **FK（正向动力学）**：逐个旋转关节——适合"挥手/摆臂"
- **Pole Target（极向目标）**：控制 IK 弯曲的方向（膝盖朝哪边弯）

**权重绘制（Weight Paint）——解决"弯曲处扭曲"的问题：**

每个顶点受哪些骨骼影响及影响程度：
- **红色** = 100% 受该骨骼影响
- **蓝色** = 0% 不受影响
- **绿色/黄色** = 中间过渡

笔刷模式：Add（加权重）/ Subtract（减权重）/ Smooth（平滑过渡）

### 刻意练习（具体操作步骤 + 代码/命令）

**案例：给 Q 版角色创建简单骨骼**

1. **创建骨架**
   - Shift + A → Armature → Single Bone
   - 切换到 Edit Mode
   - 选中骨骼顶端 → **E** 挤出：脊柱骨（从下往上）
   - 从脊柱上部挤出：头部骨
   - 从脊柱中部挤出：左手臂骨 → 再挤出前臂骨
   - 同理做右臂和双腿骨
   - 重命名骨骼（好习惯）：spine、head、arm_L、forearm_L...

2. **自动权重绑定**
   - 选择模型 → Shift + 选择骨架 → **Ctrl + P → With Automatic Weights**
   - 切换到 Pose Mode → 旋转骨骼 → 模型跟着动 ✅

3. **权重修正——解决膝盖弯曲问题**
   - 选中模型 → 切换到 **Weight Paint 模式**
   - 选择膝盖弯曲处的骨骼
   - 看到红色区域的分布
   - 用 **Subtract** 笔刷 → 减轻膝盖背面区域对大腿骨的影响
   - 用 **Add** 笔刷 → 增加对小腿骨的影响
   - 目标：膝盖弯曲时，膝盖正面皮肤自然折叠，不扭曲

4. **测试姿势**
   - Pose Mode → 旋转手臂骨骼 → 测试弯曲效果
   - 如果肘部扭曲 → 回到 Weight Paint 修正

### ✅ 完成标准

- [ ] 理解骨骼绑定的本质——用骨架控制模型变形
- [ ] 能在 Edit Mode 中用 E 挤出创建骨骼链
- [ ] 掌握 Ctrl + P → With Automatic Weights 自动绑定
- [ ] 理解 IK 与 FK 的区别及各自适用场景
- [ ] 能用 Weight Paint 修正弯曲处的权重问题

### 📖 费曼三句话

1. 骨骼是模型的"木偶线"——移动骨骼带动模型变形，理解层级结构是绑定的第一步。
2. IK 是"手放桌上"的方法（定位手肘自动弯），FK 是"挥手"的方法（逐节旋转），不同场景用不同模式。
3. 权重绘制解决"弯曲处不应该跟着动的区域跟着动了"的问题——红色跟着动，蓝色不动，绿色渐变过渡。

## 🍅29 角色动画入门（25分钟）

### 费曼输入（概念解释）

**动画十二原则（3D 版本）**——这是"动画界的物理定律"，适用于任何软件：

| # | 原则 | 通俗理解 |
|---|------|----------|
| 1 | 挤压与拉伸（Squash & Stretch） | 物体运动时形态变化，有"生命力" |
| 2 | 预备动作（Anticipation） | 动之前先反方向蓄力 |
| 3 | 表演布局（Staging） | 让观众一眼看懂在发生什么 |
| 4 | Pose to Pose | 先定关键姿势，再补中间过渡 |
| 5 | 跟随与重叠（Follow Through & Overlap） | 不同部分不同时间到达 |
| 6 | 慢入慢出（Slow In & Slow Out） | 运动开始和结尾平滑 |
| 7 | 弧线运动（Arcs） | 真实运动不是直线 |
| 8 | 次要动作（Secondary Action） | 主体动作之外的小动作 |
| 9 | 节奏（Timing） | 动作的快慢决定感觉 |
| 10 | 夸张（Exaggeration） | 比现实更强烈才有趣 |
| 11 | 立体感（Solid Drawing） | 即使在动态中也保持体积感 |
| 12 | 吸引力（Appeal） | 角色要讨喜/有趣 |

**角色动画工作流（Pose to Pose）：**

```
起始关键帧 → 中间关键姿势 → 结束关键帧
       ↓
用 Graph Editor 调整曲线
       ↓
添加中间帧（In-between）
       ↓
添加次要动作（头发/衣服/尾巴跟随动画）
```

**Blender 5.1 动画相关改进：**
- Shape Key 评估速度提升 4 倍（面部表情动画更流畅）
- Actions 评估大幅优化（多骨骼绑定场景尤为明显）
- Dope Sheet 新增彩色插值线（视觉化区分关键帧类型）
- 新增 Smooth (Gaussian) F-curve 修改器（一键平滑抖动曲线）
- 动画分层功能正在开发中（即将到来）

### 刻意练习（具体操作步骤 + 代码/命令）

**案例：角色挥手动画**

1. **Pose to Pose 关键姿势**
   - 第 1 帧：手臂自然放下 → I 插入位置/旋转关键帧
   - 第 10 帧：预备——手臂向后拉（蓄力）→ 插入关键帧
   - 第 15 帧：手臂抬到最高 → 插入关键帧
   - 第 20-30 帧：手腕上下摆动 3 次 → 插入关键帧

2. **调整 F-Curve**
   - 打开 **Graph Editor**
   - 选中手臂旋转的 F-Curve
   - 选择所有关键帧 → 设置缓入缓出（自动平滑）
   - 第 15 帧到第 20 帧之间的曲线：加一点"弹跳"效果

3. **增加弹性（利用 Smooth Gaussian 修改器 — Blender 5.1）**
   - 选中手腕的 F-Curve
   - Modifiers → Add Smooth (Gaussian)
   - 调整 Radius（半径）：让手腕抖动更自然

4. **添加次要动作**
   - 如果角色有头发/尾巴：在第 10-20 帧之间，让头发延迟 3-5 帧跟随摆动
   - 作用：让角色看起来"活"了

5. **循环播放测试**
   - 切换到 Animation 工作区
   - 播放测试 → 检查过渡是否自然
   - 回到第 1 步修正

### ✅ 完成标准

- [ ] 掌握动画十二原则的核心要点
- [ ] 能用 Pose to Pose 方法制作挥手动画
- [ ] 能在 Graph Editor 中调整 F-Curve 优化运动
- [ ] 理解次要动作（跟随与重叠）让角色"活"起来
- [ ] 知道 Blender 5.1 动画改进的关键条目（Shape Key 4 倍提速、Actions 优化等）

### 📖 费曼三句话

1. 动画十二原则是"动画界的物理定律"——适用于任何软件，跨软件通用的知识最有价值。
2. Pose to Pose 是最有效的工作流：先定关键姿势，再补中间帧——不要从第 1 帧开始一帧一帧做。
3. 次要动作（头发/尾巴/衣服晃动）是让角色"活"起来的关键——主体动作+跟随动作=真实感。

## 🍅30 商业出图与呈现（25分钟）

### 费曼输入（概念解释）

商业出图不只是"点渲染按钮"——**后期合成是让好变成更好的最后一步**。商业级作品需要关注分辨率、色彩空间、后期处理和呈现方式。

**商业出图标准：**

| 项目 | 标准 |
|------|------|
| 分辨率 | 4K (3840 × 2160) 为商业标准 |
| 格式 | PNG-16bit（静帧）/ OpenEXR（后期合成） |
| 色彩空间 | sRGB（网络）/ Rec.709（影视）/ DCI-P3（打印） |

**后期合成（Compositor）核心调色流程：**

```
原始渲染 → 颜色校正 → 景深 → 辉光 → 镜头特效 → 输出
```

1. **颜色校正**：曝光（Exposure）/ 对比度（Contrast）/ 饱和度（Saturation）
2. **景深（DOF）**：雾化背景突出主体
3. **辉光（Glare）**：高光区域产生光晕
4. **镜头光晕/色差（Chromatic Aberration）**：增加真实感
5. **Blender 5.1 新增 — Mask to SDF node**：将遮罩转为有符号距离场，实现更精确的边缘混合

**呈现技巧：**

| 技巧 | 说明 | 用途 |
|------|------|------|
| **工作室灯光** | 顶部柔光 + 两侧补光 | 产品展示的标准灯光 |
| **转盘展示** | 物体 360 度旋转动画 | 全方位展示模型 |
| **爆炸图** | 展示内部结构的分解图 | 产品功能说明 |
| **A/B 对比** | 不同材质/颜色的效果对比 | 设计决策展示 |

**作品集整理建议：**

- **Render**：最终渲染图（多角度 3-5 张）
- **Wireframe**：线框图展示拓扑结构
- **Breakdown**：分步骤展示制作过程
- **Turntable**：360 度展示视频（GIF 或 MP4）

### 刻意练习（具体操作步骤 + 代码/命令）

**步骤1：工作室灯光设置（产品展示）**

1. 三灯布局：
   - **主光（Key Light）**：Area Light，45 度俯角，强度 1000W
   - **补光（Fill Light）**：Area Light，另一侧 45 度，强度 300W，冷色
   - **背光（Rim Light）**：在物体后方，勾勒轮廓
2. 添加一个 HDRI（世界属性 → Environment Texture）
3. 渲染测试：物体应该有清晰的立体感

**步骤2：合成器调色**

1. 切换到 **Compositing 工作区**
2. 勾选 **Use Nodes**
3. 添加节点链：
   ```
   Render Layers → Color Balance → Glare → Composite
   ```
4. **Color Balance**：微调 Lift / Gamma / Gain
5. **Glare**：选择 Fog Glow → 调整 Mix 值控制强度
6. **Blender 5.1**：尝试新增的 **Mask to SDF node** → 连接遮罩 → 实现更精确的边缘过渡

**步骤3：转盘渲染输出**

1. 选中物体 → 添加空物体（Empty）在场景原点
2. 给物体添加 **Copy Rotation** 约束，目标为空物体
3. 空物体：第 1 帧旋转 Z=0 → 第 100 帧旋转 Z=360 → 插入关键帧
4. Output Properties：帧范围 1-100，格式 FFmpeg MP4
5. 渲染 → 自动生成 360 度旋转视频

**步骤4：整理作品集输出**

1. Render：渲染 3-5 个角度的静帧
2. Viewport Overlays → 勾选 Wireframe → 截线框图
3. 制作 Breakdown：用截图软件记录每一步的关键节点
4. 上传到 ArtStation / 个人网站
