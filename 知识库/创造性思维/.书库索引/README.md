## 使用模式

### 模式1：知识问答检索
```
提问 → 匹配目录/标签 → 调取笔记内容 → 整合回答（支持溯源）
```

### 模式2：创新方法指导
```
输入创新需求 → 匹配创造力/设计思维SKILL → 输出创新方法/工具推荐
```

### 模式3：AI算法学习
```
提出算法问题 → 匹配AI算法SKILL → 调取算法原理笔记 → 输出学习指导
```

## 快速检索指南

### 按标签检索
每本书配备标准YAML元数据，支持按 `category`、`skill_tag`、`level`、`keywords` 精准检索。

### 按技能检索
直接使用技能触发指令（如"创造力激发"、"认知科学"），自动匹配对应SKILL模块。

### 语意全文检索
跨笔记整合知识点，适合综合性问题。
