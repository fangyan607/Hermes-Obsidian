# Dream Cycle 梦境循环计划

> **让知识从"静态储存"进化为"持续生长的有机体"**
> Dream Cycle = 设想(Dream) → 规划(Plan) → 执行(Execute) → 复盘(Review) → 循环(Cycle)

---

## 一、核心理念

Dream Cycle 是基于 GSD（Goal-oriented Systematic Development）方法论的知识进化循环系统。每一次循环，知识库都完成一次"做梦→行动→反思"的完整迭代。

```
          ┌─────────────────────────────────┐
          │         Dream（设想）             │
          │    "知识应该长成什么样子？"         │
          └────────────┬────────────────────┘
                       │
          ┌────────────▼────────────────────┐
          │         Plan（规划）              │
          │    "下一步做什么？优先级是什么？"    │
          └────────────┬────────────────────┘
                       │
          ┌────────────▼────────────────────┐
          │        Execute（执行）            │
          │    "动手创建/更新知识库"          │
          └────────────┬────────────────────┘
                       │
          ┌────────────▼────────────────────┐
          │        Review（复盘）             │
          │    "学到了什么？怎么改进？"        │
          └────────────┬────────────────────┘
                       │
                  (回到 Dream)
```

---

## 二、四个阶段详解

### Phase 1: Dream（设想）

**目标**：为知识库设定愿景和方向

| 活动 | 说明 | 频率 |
|------|------|------|
| 知识库体检 | 检查哪些领域资料不足、哪些过时 | 每月 |
| 趋势扫描 | 搜索最新资讯，发现新知识领域 | 每周/按需 |
| 灵感捕捉 | 将零散想法记录到 Claude_Memory | 随时 |
| 愿景设定 | "这个知识库三个月后应该长成什么样？" | 每月 |

### Phase 2: Plan（规划）

**目标**：将设想转化为可执行的任务

| 活动 | 说明 | 输出 |
|------|------|------|
| 任务分解 | 将大目标拆为可执行子任务 | Todo list |
| 优先级排序 | 影响/ effort 矩阵 | 优先级表 |
| 资源评估 | 需要什么资料、工具、时间？ | 资源清单 |
| 依赖分析 | 哪些任务必须先做？ | 依赖图 |

### Phase 3: Execute（执行）

**目标**：实际创建/更新知识

| 活动 | 说明 | 对应的本仓库操作 |
|------|------|-----------------|
| 知识创建 | 新领域建库 | 创建 LLM-Wiki + GBrain |
| 知识迁移 | 从 Claude_Memory 到 Layer1 | 归档迁移 |
| 知识更新 | 补充新内容或修正 | 编辑已有文件 |
| 关系连接 | 建立跨域链接 | 编辑 GBrain relations |

### Phase 4: Review（复盘）

**目标**：从执行中学习，优化下一次循环

| 活动 | 说明 | 输出 |
|------|------|------|
| 成果检视 | 完成了什么？与目标差距？ | 完成度报告 |
| 质量评估 | 知识的深度、广度、准确性 | 质量评分 |
| 流程改进 | 哪些环节效率低？如何优化？ | 改进清单 |
| 经验沉淀 | 写入 MEMORY.md 经验教训 | MEMORY 更新 |

## 四、Dream Cycle 与三层记忆系统的关系

```
          Dream Cycle（元认知循环）
                 │
    ┌────────────┼────────────┐
    │            │            │
    ▼            ▼            ▼
  Layer 1     Layer 2     Layer 3
  永久记忆    会话记忆    大脑目录
  (知识库)   (Claude_Memory) (MEMORY.md)
    │            │            │
    └────────────┼────────────┘
                 │
         知识的"生长"方向：
         Layer2(新想法) → Layer1(沉淀) 
         Layer3(导航) ← 全库状态
```
