# 对白：文字·舞台·银幕的言语行为艺术

> 所有对白都带着目的——说话比任何特质都更能表达我们的人性。

## 核心理念

本书由罗伯特·麦基撰写，核心方法论围绕**对白作为言语行为（Verbal Action）**。对白不是日常闲聊，而是带有目的、推动冲突、揭示角色的半诗化语言。

## 执行流程

### Phase 1: 理解对白的艺术
对白的三个功能（暴露信息、塑造角色、推动情节）、表达三要素（内容、形式、技巧）

### Phase 2: 检查与修改瑕疵
可信性瑕疵、语言瑕疵、内容瑕疵、设计瑕疵、六项对白任务检查

### Phase 3: 创造角色专属对白
每个角色拥有独特的词汇库、语法、节奏、音调

### Phase 4: 对白场景设计
均衡冲突、喜剧冲突、不对称冲突、非直接冲突、自反性冲突、微限冲突

## 局限性

- **不适用于**：纯文学对白或诗歌写作
