# 🏛️ 哲学60番茄教程：两千六百年思想悬疑剧

> **风格声明**：这不是一门"哲学史课"。这是一场历时2600年的**思想探案**。每个哲学家都是嫌犯，每个命题都是线索，而"真相"——如果你找得到的话——会颠覆你习以为常的一切。

## 🎬 预告片

你每天早上醒来，睁开眼睛，看到一个"世界"。

但你有没有停下来问过——

**这个世界是真的吗？**（柏拉图在公元前380年就这么问了）

**你怎么知道你"知道"任何东西？**（笛卡尔说：有个恶魔可能在欺骗你）

**你所谓的"自由选择"，是不是只是你不了解的因果链条的一部分？**（斯宾诺莎和叔本华都说：是）

**"对"和"错"是客观存在的，还是我们编出来的？**（尼采说：上帝死了，是你自己编的）

**如果AI有了意识，它算不算"人"？**（当代心灵哲学家正在吵这个）

这些问题不是"学术问题"。它们是你每天早上醒来面对的世界时，**已经在处理的问题**——只不过你用的是"默认设置"，而哲学家花了一辈子试图破解这些默认设置。

**这门课的目的很简单：帮你破解默认设置。**

## 🍅 学习方法

沿用 [[社会学科番茄费曼教程创建SKILL]] 的三体合一法：

| 武器 | 在这套课里的角色 |
|------|------------------|
| **🍅 番茄工作法** | 每个番茄一个"思想单元"——刚好够读懂一个哲学家+他的核心命题+一个让你睡不着的追问 |
| **📝 费曼学习法** | 每番茄末尾"费曼三句话"——用你自己的话给一个外行讲清楚 |
| **🎯 刻意练习** | 每课的"思想实验"——你不再是被动读者，而是和哲学家辩论的对手 |

### 每个番茄的节奏

```
⏰ 0-20 分钟：哲学家的故事 + 核心思想 + 原文片段
⏰ 20-22 分钟：费曼三句话（闭眼给自己讲一遍）
⏰ 22-25 分钟：思想实验 + 写下你的反驳
⏰ 25-30 分钟：休息（让潜意识接着吵）
```

## 🧩 每天结构

```
🍅1 → 悬疑开场 + 核心哲学家故事
🍅2 → 核心理论拆解（"他在说什么？"）
🍅3 → 原文品读 + 思想实验（"他说得对吗？"）
🍅4 → 🧠 思维导图 + 费曼大复习
🍅5 → 刻意练习 + 跨界应用
```
