# 敷脐除湿药丸制作方法汇总

> 创建日期：2026-05-27
> 来源：中医书库检索整理

---

## 一、水泻除湿方

### 方剂1：木鳖子丁香丸

| 项目 | 内容 |
|------|------|
| **主治** | 水泻不止（脾虚湿盛） |
| **组成** | 木鳖子5个、母丁香5个、麝香1分 |
| **制作** | 研末，米汤调作膏 |
| **用法** | 纳脐中贴之，外以膏药护住 |
| **来源** | 《扶寿精方》 |

---

## 二、水肿除湿方

### 方剂2：大蒜田螺车前膏

| 项目 | 内容 |
|------|------|
| **主治** | 水气肿满 |
| **组成** | 大蒜、田螺、车前子各等分 |
| **制作** | 熬膏 |
| **用法** | 摊贴脐中，水从便漩而下，数日即愈 |
| **来源** | 《稗史》 |

---

## 三、泄泻痢疾除湿方

### 方剂3：七圣饼子

| 项目 | 内容 |
|------|------|
| **主治** | 泄泻痢疾 |
| **组成** | 黄连7寸（为细末）、巴豆仁7个 |
| **制作** | 为末，入蜜少许，和作饼子 |
| **用法** | 放脐中，用艾炷灸之。以利为度。急用水洗去，以防泻脱 |
| **来源** | 《医方类聚》引《经验秘方》 |

### 方剂4：大蒜贴脐方

| 项目 | 内容 |
|------|------|
| **主治** | 泄泻暴痢 |
| **组成** | 大蒜 |
| **制作** | 捣烂 |
| **用法** | 贴两足心，亦可贴脐中 |
| **来源** | 《千金方》 |

## 七、艾灸脐除湿方

### 方剂12：隔盐灸神阙

| 项目 | 内容 |
|------|------|
| **主治** | 慢性菌痢（脾虚寒湿型）、寒湿痢、泄泻 |
| **取穴** | 脐中（神阙穴） |
| **操作** | 在穴位处平铺一层很薄的食盐，将艾绒搓成绿豆大的艾炷，放在盐层中央，灸3~7壮 |
| **来源** | 《针灸临床经验辑要》 |

### 方剂13：妇人阴挺常湿灸方

| 项目 | 内容 |
|------|------|
| **主治** | 妇人胞胎门落颓不收，常湿 |
| **取穴** | 神阙、玉泉 |
| **操作** | 灸五十壮，三报（反复三次） |
| **来源** | 《针灸资生经》 |

## 八、敷脐药丸制作通用方法

```
┌─────────────────────────────────────────────────────────────┐
│                   敷脐药丸制作流程                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  第一步：选药                                                │
│  ├── 温阳散寒：丁香、肉桂、附子、干姜                        │
│  ├── 健脾利湿：白术、茯苓、车前子                            │
│  ├── 行气导滞：木香、沉香、麝香                              │
│  └── 清热燥湿：黄连、黄芩                                    │
│                                                             │
│  第二步：研末                                                │
│  └── 将药物研磨成极细粉末                                    │
│                                                             │
│  第三步：调和                                                │
│  ├── 蜜调：蜂蜜调和，滋润缓和                                │
│  ├── 醋调：米醋调和，增强渗透                                │
│  ├── 姜汁调：生姜汁调和，温散力强                            │
│  ├── 酒调：黄酒调和，活血通络                                │
│  └── 米汤调：米汤调和，健脾和胃                              │
│                                                             │
│  第四步：制形                                                │
│  ├── 饼状：敷贴面积大，适合脐部                              │
│  ├── 丸状：药量精确，便于保存                                │
│  └── 膏状：易于涂抹，渗透性好                                │
│                                                             │
│  第五步：敷贴                                                │
│  ├── 填脐：将药物填入脐窝                                    │
│  ├── 贴脐：将药饼贴于脐上                                    │
│  └── 固定：外用胶布或膏药固定                                │
│                                                             │
│  第六步：配合                                                │
│  ├── 艾灸：药上加灸，温阳化湿                                │
│  ├── 熨敷：热物熨之，助药力渗透                              │
│  └── 时间：一般敷贴6-12小时                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 九、敷脐除湿的药物选择原则

| 湿邪类型 | 首选药物 | 配伍药物 | 调和介质 |
|----------|----------|----------|----------|
| **寒湿** | 丁香、肉桂、干姜 | 附子、吴茱萸 | 姜汁、酒 |
| **湿热** | 黄连、黄芩 | 大黄、车前子 | 醋、蜜 |
| **脾虚湿盛** | 白术、茯苓 | 苍术、薏苡仁 | 米汤、蜜 |
| **水湿泛滥** | 大蒜、田螺 | 车前子、滑石 | 捣烂成膏 |

## 十、注意事项

1. **皮肤过敏**：敷脐前可在脐周涂抹少许油剂保护皮肤
2. **破溃禁用**：脐部有破损或感染者不宜敷贴
3. **孕妇慎用**：某些活血化瘀药物孕妇禁用
4. **时间控制**：一般敷贴6-12小时，不宜过长
5. **配合艾灸**：可增强除湿效果
6. **观察反应**：如出现红肿瘙痒应立即取下
