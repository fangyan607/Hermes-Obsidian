# Claude_Memory — 会话记忆层（Layer 2）

> **定位**：AI 的"工作记忆"与"草稿本"——每次会话的实时思路、决策、灵感临时存储区
> **对应视频**：Layer 2 = 前额叶（工作记忆）
> **规则**：会话中的临时内容写在此处；会话结束后，有价值的内容提炼迁移到 Layer 1（LLM-Wiki/ GBrain/）

## 使用规则

1. 当前会话的所有思路、决策、灵感**优先记录在此**
2. 文件名格式：`YYYY-MM-DD-<主题>.md`
3. 内容结构：`# 会话摘要 → ## 关键决策 → ## 灵感记录 → ## 待迁移内容`
4. 会话结束后执行归档：有价值内容 → Layer 1 对应文件夹；临时内容 → 清理或存档
