## 一、编剧创作类

### 1. save-the-cat-screenwriting（救猫咪编剧法）

**触发条件**
- 构思电影故事创意
- 写故事线（logline）
- 确定故事的题材类型
- 应用节拍表规划剧本结构
- 解决剧本创作中的常见问题

**使用方式**
```
Phase 1: 构思故事线 → 提炼意外元素、创造心理影像、界定观众成本、设计片名
Phase 2: 市场调研 → 街头测试、分析反馈
Phase 3: 题材分类 → 10种题材归位、研究同类影片
Phase 4: 应用15节拍结构表 → 填写节拍、填充内容
```

**使用场景**
- 剧本初稿创作
- 故事创意验证
- 商业类型片结构设计

**功能特征**
- 10种题材分类体系（鬼怪屋、金羊毛、如愿以偿等）
- 15节拍结构表（开场画面→终场画面）
- 故事线四要素（意外、心理影像、观众成本、片名）
- "救猫咪"场面设计法

**局限性**
- 不适用于艺术片、实验电影、非叙事性作品
- 需要创作者愿意接受结构化方法
- 过度依赖模板可能忽视故事独特性

### 2. save-the-cat-strikes-back（救猫咪反击战）

**触发条件**
- 改进失败的故事创意推销
- 解决剧本写作中的结构问题
- 处理剧本改写和反馈
- 准备销售会议和推介
- 管理编剧事业和心态

**使用方式**
```
Phase 1: 修复创意推销 → 诊断问题类型、修复故事线、重新测试
Phase 2: 解决写作卡壳 → 识别卡壳类型、应用结构图工具、执行修复策略
Phase 3: 处理剧本改写 → 分类改写意见、评估意见价值、执行改写
Phase 4: 销售与事业经营 → 准备销售会议、执行推介、事业经营
Phase 5: 心态管理 → 识别心理挑战、应用正面策略
```

**使用场景**
- 创意被拒绝后的修复
- 写作过程中的卡壳突破
- 改写意见处理
- 编剧职业生涯规划

**功能特征**
- 三大推销问题诊断（太平淡/太复杂/刻意保密）
- 四种卡壳类型识别（枝蔓过多/半步缓行/节拍模糊/脊柱侧凸）
- 改写意见分类评估体系
- 心态管理策略

**局限性**
- 不适用于非编剧领域的创意工作
- 需要创作者有实际的剧本或创意需要处理

## 二、导演与视听语言类

**触发条件**

**使用方式**
```
Phase 1: 影像化准备 → 剧本分析、美术设计规划、故事板制作
Phase 2: 镜头构图设计 → 选择景别、建立动作轴线、设计机位布局
Phase 3: 对话场景调度 → 选择调度系统、设计主镜头、设计覆盖镜头
Phase 4: 运动镜头设计 → 选择运动类型、设计运动路径、设计转场
```

**使用场景**
- 分镜设计
- 场面调度规划
- 对话场景拍摄方案

**功能特征**
- 动作轴线（180°规则）系统
- 三角机位系统
- 双人/三人/多人对话调度模式
- 运动镜头设计方法

**局限性**
- 不适用于纪录片即兴拍摄、实验电影非传统叙事

- 了解配乐确定配乐点（spotting）流程
- 学习音乐与画面同步技术
- 处理配乐合同、版税、版权等商业问题

**使用方式**
```
Phase 1: 理解影片核心 → 观看与感受、确定音乐功能、确定音乐风格
Phase 2: 确定配乐点 → 配乐点会议、计算时间、确定同步点
Phase 3: 音乐创作 → 创作主题素材、编写配乐段落、同步技术处理
Phase 4: 录制与混音 → 配器与准备、录音、混音
Phase 5: 商业运作 → 合同与酬金、版税登记、市场推广
```

**使用场景**
- 电影配乐创作
- 音乐编辑工作
- 配乐商业运作

**功能特征**
- 完整配乐流程
- 配乐点表模板
- 同步技术应用
- 商业运作指南

**局限性**
- 不适用于纯音乐创作（非影视配乐）
- 需要能观看粗剪版或完整剧本
