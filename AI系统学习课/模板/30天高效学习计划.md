# 30天高效学习计划：知乎大模型15期完整课程

## 核心理念（融合四部时间管理经典）

| 原则 | 来源 | 核心要点 |
|------|------|----------|
| **每日重点项（Highlight）** | 《创造时间》 | 每天选1-2个核心学习目标，写在便利贴上 |
| **番茄工作法（25+5）** | 《番茄工作法》 | 每个番茄=25分钟专注+5分钟休息，4个番茄后休息15-30分钟 |
| **极简·少即是多** | 《极简时间》 | 只做高杠杆率的事情，不贪多、不并行 |
| **持续行动·复利效应** | 《刻意学习》 | 每天4小时雷打不动，100天=二阶行动者，30天打好地基 |

### 日程框架（每日4小时 = 6个番茄）

```
25分钟学习 → 5分钟休息 → （重复4次）
→ 15-30分钟长休息 → 
25分钟学习 → 5分钟休息 → （重复2次）
```

| 时段 | 具体安排 | 番茄数 | 时长 |
|------|----------|--------|------|
| **早段** | 开篇定调：写今日重点项 + 回顾昨日难点 | 0 | 5分钟 |
| **核心段1** | 番茄1 → 番茄2 → 番茄3 → 番茄4 | 4 | 2小时 |
| **长休息** | 散步/拉伸/喝水/闭眼（严禁看手机） | 0 | 20分钟 |
| **核心段2** | 番茄5 → 番茄6 | 2 | 1小时 |
| **收尾** | 实践/写笔记/整理代码 + 记录反思 | 0 | 35分钟 |
| **总计** | | **6个番茄** | **4小时** |

## 时间管理工具箱（每日执行清单）

### 每天启动时（3分钟）

```
□ 写今日重点项（今天最想搞懂的1件事）
□ 复习昨日笔记（快速过一遍）
□ 确认今日番茄数量（默认6个，共4小时）
```

### 每个番茄结束后（10秒）

```
□ 在本子上画一个「×」
□ 立即离开座位
□ 不碰手机
```

### 每天结束时（5-10分钟反思）

```
□ 我今天学到了什么？（一句话）
□ 哪个地方卡住了？（标记回头补）
□ 明天重点是啥？
□ 精力评分（1-5分）
```

### 每周日额外30分钟大复盘

```
□ 查看完成率（应完成40个番茄，实完成____个）
□ 整理本周所有笔记
□ 补完未看视频的番茄
□ 调整下周计划
```

## 风险管理与纠偏

| 问题 | 解决方案 |
|------|----------|
| 某天实在没时间 | 至少完成2个番茄（1小时），标记为」最低配日「 |
| 某节视频太难看不懂 | 跳过不纠结，先往后学，周末回头补 |
| 学累了想放弃 | 执行「5分钟原则」：只做5分钟，5分钟后决定是否继续 |
| 长期卡在一个点上 | 设置「极限时间」：最多再花3个番茄，还搞不懂就放弃 |
| 周完成率<80% | 周日补课，下周降低期望值 |

## 番茄使用提示

- 1个番茄 = 看视频 + 实操 + 记笔记
- 视频倍速：简单内容1.5-2倍速，难的内容1-1.25倍速
- 长休息时：绝对不要看手机/刷视频，让大脑真正休息
- 每个番茄开始前：关闭所有通知、关掉无关网页、手机静音

> **记住**：持续行动30天，比完美执行3天更有价值。
> 完成比完美重要，持续比强度重要。
