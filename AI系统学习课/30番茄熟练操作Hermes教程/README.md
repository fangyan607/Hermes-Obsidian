## 🔧 前置准备

### 硬件要求

| 项目 | 最低配置 | 推荐配置 |
|:----|:--------|:--------|
| **本地电脑** | 4GB RAM, 10GB 磁盘 | 8GB+ RAM |
| **云服务器（可选）** | 2vCPU, 2GB RAM | 4vCPU, 8GB RAM |
| **操作系统** | Windows 10 / macOS 12+ / Ubuntu 20.04+ | 同上 |

### 需要注册的服务

1. **OpenRouter**（推荐）：https://openrouter.ai — 聚合多模型，充值 $5-10 可用很久
2. **GitHub 账号**：用于源码克隆和知识库同步
3. **云服务器**：阿里云/腾讯云（任选，部署模块需要）

### 开发环境

- Python 3.8+（推荐 3.11）
- Git
- Node.js 18+（Web 面板需要）

## 📚 教学法说明

### 🍅 番茄工作法

```
每4个番茄钟 = 1组学习单元
每个番茄钟 = 25分钟专注 + 5分钟休息
每组之间 = 15-30分钟长休息
```

### 🧠 费曼学习法

每个核心概念后嵌入 **✋ 费曼自测** 环节——用大白话复述、对比、应用，确保真正理解而非表面看过。

### 💪 刻意练习

每个模块末尾包含三级练习：
1. **基础练习**：模仿操作
2. **进阶练习**：独立完成
3. **挑战练习**：综合应用
