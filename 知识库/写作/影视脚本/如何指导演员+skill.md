# 如何指导演员：导演的必修课

> 没有不会"演"的演员，只有不会"导"的导演。

## 核心理念

本书由朱迪丝·韦斯顿撰写，核心方法论围绕**导演如何与演员合作创造精彩表演**。从避免"结果式指导"到帮助演员找到真实的情感表达。

## 执行流程

### Phase 1: 避免结果式指导
用动词替代形容词、给具体任务而不是情感指示、提出问题而非答案

### Phase 2: 创造时刻当下
营造安全环境、鼓励风险、专注与自由、特异性

### Phase 3: 引导演员选择
问题、逆向思考、评价、需求、意象、障碍、事实、调整、潜台词

### Phase 4: 剧本分析与排练
准备首次阅读、识别贯串线、发现转变时刻、排练技巧

## 局限性

- **适用于**：导演和演员的协作指导
- **不适用于**：演员单独的自修训练
