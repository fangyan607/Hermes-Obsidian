## 一、6大核心SKILL技能

### Skill1：创造力激发

**触发指令**：创造力激发、创新方法、创意生成、创新思维、创造力提升

**关联书籍**：
- [[创新简史：从石斧到爆品]]
- [[创造力曲线-艾伦·甘尼特]]
- [[创新者的基因]]
- [[创造与创新思维系列（6册装）]]
- [[企鹅与怪兽：互联时代的合作、共享与创新模式]]

**核心能力**：

| 能力 | 说明 |
|------|------|
| 创新史分析 | 梳理人类创新历史脉络，识别关键创新模式 |
| 创造力评估 | 基于创造力曲线理论评估个体/团队创造力水平 |
| 创新者思维训练 | 培养联想、质疑、观察、实验、社交五大创新基因 |
| 创新方法论应用 | 掌握系统性创新思维工具与方法论 |
| 合作创新模式 | 理解互联时代的合作共享创新机制 |

**使用示例**：
```
用户：如何提升团队的创造力？
→ 启用Skill1创造力激发
→ 调用《创造力曲线》《创新者的基因》笔记
→ 输出：创造力评估方法 + 五大创新基因训练方案 + 团队创新环境搭建建议
```

## 二、技能调用规则

### 调用优先级

```yaml
优先级1: 精准匹配
  - 用户指令直接触发对应技能

优先级2: 语义理解
  - AI识别意图，匹配最相关技能

优先级3: 多技能联动
  - 复杂问题调用多个技能组合
```

### 典型调用组合

| 场景 | 技能组合 |
|------|----------|
| 创新方案设计 | Skill1(创造力) + Skill5(设计思维) + Skill4(可视化) |
| AI产品创新 | Skill3(算法) + Skill5(设计思维) + Skill1(创造力) |
| 认知升级训练 | Skill2(认知科学) + Skill4(可视化思维) + Skill6(系统思考) |
| 复杂问题求解 | Skill6(系统思考) + Skill3(算法思维) + Skill4(可视化) |
| 学习方法优化 | Skill2(认知科学) + Skill4(可视化思维) + Skill3(算法思维) |

## 三、提示词模板

### 通用知识检索提示词
```
仅依据Obsidian LLM-WIKI内创造性思维MD笔记内容作答。
1. 识别问题匹配对应skill_tag与文件夹分类，调取库内原有知识点
2. 不编造库外理论内容
3. 复杂内容分层梳理，实操类给出清晰步骤
4. 关键内容标注对应笔记名称，支持溯源
```

### 创新方法指导提示词
```
启用创造力激发SKILL，调取LLM-WIKI创造力理论相关笔记。
结合用户创新需求，从创新史分析、创造力评估、创新者基因训练、
创新方法论等多角度给出专业建议与实操方法。
```

### AI算法学习提示词
```
启用AI算法思维SKILL，检索算法基础、机器学习、深度学习笔记。
依据用户当前水平（入门/进阶/专业），从算法原理、数学推导、
代码实现、应用场景分层讲解。
```
