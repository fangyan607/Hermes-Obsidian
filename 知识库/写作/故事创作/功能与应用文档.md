## 各 Skill 功能详解

### 1. creative-writing-fundamentals（创意写作基础）

**核心功能**：
- 6阶段写作训练流程（心灵解放→思维训练→写作训练→创意品位→创造性阅读→尽善尽美）
- 作家障碍识别与克服
- 创意培养方法（自由写作、自动书写、晨间pages）
- 文类成规理解与运用
- 修改技巧（结构修改、语言修改、细节修改）

**触发关键词**：
- "我想学习创意写作"
- "我有写作障碍"
- "帮我设计写作训练"
- "什么是创意写作"

**输出物**：
- 写作信心建立报告
- 创意素材库
- 类型化写作方案
- 创意品位提升方案
- 阅读笔记和分析
- 修改后的定稿

**局限性**：不适用于纯学术写作、技术文档

---

### 2. scene-setting（场景设定）

**核心功能**：
- 6阶段场景设计流程
- 225个故事舞台速查
- 情感连结场景设计
- 冲突场景设计（物理障碍、心理创伤、麻烦制造者）
- 气氛营造（天气、季节、光与影、叙事者视角）
- 比喻性词汇深化（直喻、隐喻、象征、拟人）
- 场景描写检查清单

**触发关键词**：
- "帮我设计一个场景"
- "故事发生在哪里"
- "如何描写场景"
- "225个故事舞台"

**输出物**：
- 情感连结场景方案
- 冲突场景方案
- 气氛场景方案
- 故事推进场景方案
- 比喻性场景描写
- 优化后的场景描写

**场景类型速查**：
- 田园篇：农场、牧场、海滩、森林、教堂、墓地等
- 都会篇：小巷、街道、夜店、法庭、地铁、机场等
- 学校篇：教室、校长室、宿舍、体育馆等
- 自然篇：山、溪、湖、海、沙漠、洞穴等

### 3. story-strategy-23-beats（故事策略23段落）

**核心功能**：
- 23个故事段落完整模板
- 三幕结构设计（建置→反应→行动→解决）
- 人物成长弧设计
- 冲突与压力递增系统
- 主题陈述方法（"为了___你必须___"）
- 人物共鸣9要素

**触发关键词**：
- "帮我设计电影剧本"
- "23个故事段落"
- "英雄目标序列"
- "电影剧本写作"

**输出物**：
- 主题陈述文档
- 主人公档案
- 23段落故事大纲
- 人物成长弧文档
- 冲突升级方案
- 修改后的完整剧本大纲

**23段落速查**：

| 幕 | 段落 | 名称 |
|----|------|------|
| 第一幕 | 1-6 | 开场画面、主题陈述、铺垫、触发事件、争辩、第一幕转折 |
| 第二幕上 | 7-12 | B故事、嬉戏时光、中点、反派逼近、一切失去、灵魂黑夜 |
| 第二幕下 | 13-17 | 第三幕转折、结局铺垫、执行计划、高潮前危机、高潮 |
| 第三幕 | 18-23 | 高潮后、结局画面、主题回归、人物变化、情感满足、余韵 |

## Skill 协同应用

### 完整小说创作流程

```
1. creative-writing-fundamentals → 克服写作障碍，激发创意
         ↓
2. 12-life-archetypes → 分析角色人格，确定主角原型
         ↓
3. 45-character-archetypes → 设计45种角色原型，完善人物
         ↓
4. classic-20-plots → 选择情节模式，构建因果链
         ↓
5. writers-journey-hero-journey → 设计英雄旅程12阶段
         ↓
6. story-strategy-23-beats → 细化23个故事段落
         ↓
7. scene-setting → 设计场景，营造气氛
         ↓
8. speculative-fiction-writing → 如为幻想类型，补充类型元素
         ↓
9. creative-writing-fundamentals → 最终修改润色
```

### 剧本创作流程

```
1. story-strategy-23-beats → 23段落剧本结构
         ↓
2. writers-journey-hero-journey → 英雄旅程三幕结构
         ↓
3. 45-character-archetypes → 角色原型设计
         ↓
4. scene-setting → 场景设计
         ↓
5. classic-20-plots → 情节模式选择
```

### 角色设计流程

```
1. 12-life-archetypes → 人格原型分析
         ↓
2. 45-character-archetypes → 45种角色原型选择
         ↓
3. writers-journey-hero-journey → 英雄旅程性格弧线
         ↓
4. scene-setting → 角色与场景的情感连结
```

## 使用场景指南

### 场景一：新手学习写作

**推荐 Skill**：`creative-writing-fundamentals`

**使用步骤**：
1. 先做"心灵解放"阶段，克服写作障碍
2. 进行"思维训练"，发展个人心思
3. 学习"写作训练"，理解文类成规
4. 通过"创造性阅读"提升品位
5. 最后"尽善尽美"修改作品

### 场景二：设计小说人物

**推荐 Skill**：`45-character-archetypes` + `12-life-archetypes`

**使用步骤**：
1. 用 `12-life-archetypes` 分析角色人格类型
2. 用 `45-character-archetypes` 选择具体原型
3. 设计性格弧线和人物旅程
4. 设计配角原型和角色关系

### 场景三：构建故事结构

**推荐 Skill**：`story-strategy-23-beats` + `writers-journey-hero-journey`

**使用步骤**：
1. 用 `writers-journey-hero-journey` 设计英雄旅程12阶段
2. 用 `story-strategy-23-beats` 细化23个故事段落
3. 设计人物成长弧
4. 设计冲突与压力递增

### 场景四：设计场景

**推荐 Skill**：`scene-setting`

**使用步骤**：
1. 确定场景的情感功能
2. 选择场景类型（田园/都会/学校/自然）
3. 设计冲突和气氛
4. 运用比喻性词汇深化
5. 检查感官描写完整性

### 场景五：幻想小说创作

**推荐 Skill**：`speculative-fiction-writing` + 其他相关 Skill

**使用步骤**：
1. 用 `speculative-fiction-writing` 选择类型（科幻/奇幻/恐怖）
2. 进行世界构建
3. 设计怪物/反派
4. 结合 `45-character-archetypes` 设计角色
5. 结合 `scene-setting` 设计场景

## 最佳实践

### 1. 循序渐进

- 新手从 `creative-writing-fundamentals` 开始
- 掌握基础后再学习结构和角色设计
- 最后学习类型化写作

### 2. 组合使用

- 角色设计：`12-life-archetypes` + `45-character-archetypes`
- 结构设计：`story-strategy-23-beats` + `writers-journey-hero-journey`
- 场景设计：`scene-setting` + `speculative-fiction-writing`

### 3. 反复迭代

- 完成初稿后用质量检查清单验证
- 根据检查结果修改
- 多次迭代直到满意

### 4. 灵活调整

- 各 Skill 的流程可根据需要调整顺序
- 不是所有步骤都必须执行
- 根据具体项目需求选择适用部分

### 5. 持续学习

- 阅读经典案例库中的范例
- 分析成功作品的结构
- 不断积累写作经验
