## 🍅 番茄31：内容营销策略——装饰行业获客底层逻辑

### 📖 费曼三句话

1. **内容获客 = "让客户在需要你的时候第一个想到你"**——不是发广告，是持续发有价值的内容。客户刷到你10次装修知识帖之后，自己家要装修时，第一个想到的就是你。
2. **装饰行业内容获客的"漏斗模型"**：
   - 顶部（曝光）：抖音/小红书 → 让1万人看到你
   - 中部（互动）：评论/私信 → 让100人咨询你
   - 底部（成交）：见面/量房 → 让10人选择你
   - 每个环节都靠内容驱动。
3. **AI让内容生产成本降低到传统方式的1/5**——一篇案例帖从4小时缩短到1小时，一条视频从6小时缩短到1.5小时。省下的时间就是你的"获客数量优势"。

### 🛠 刻意练习

1. 用AI分析你的目标客户：
   ```
   我是一名旧房翻新和适老化改造设计师，我的目标客户是
   35-60岁的城市居民，有老房需要翻新或家有老人需要适老改造。
   请帮我分析：
   1. 这类客户每天在手机上搜什么关键词？
   2. 他们最关心的5个问题是什么？
   3. 什么样的内容最容易打动他们？
   ```
2. 根据AI的分析，确定你本月的内容主题方向（3个）

### ✅ 完成标准

- [ ] AI输出了目标客户画像和内容方向
- [ ] 你确定了3个内容主题方向
- [ ] 理解了"曝光→互动→成交"的漏斗

## 🍅 番茄32：小红书AI内容生产（上）——案例帖

### 📖 费曼三句话

1. **小红书是装饰行业最好的获客平台**——用户主动搜"旧房翻新""装修避坑""客厅改造"，带着需求来的。一篇爆款改造案例帖，可以持续带来半年的咨询。
2. **爆款案例帖的公式**：
   - 封面：改造前后对比图（冲击力！）
   - 标题：数字+结果+悬念（"花3万把60平米老房变成这样，邻居都来问"）
   - 正文：改造前的问题 → 改造思路 → 改造后的效果 → 费用明细
   - 标签：#旧房翻新 #小户型改造 #装修案例 #省钱装修
3. **AI辅助案例帖写作流程**：你口述或写大概 → AI润色成小红书风格 → 你检查本地信息 → AI配图方案 → 发布。全程1小时。

### 🛠 刻意练习

1. 用AI写一篇案例帖初稿：
   ```
   请写一篇小红书画风的老房改造案例帖。
   基本信息：60平米老房，预算8万，北欧风格，主要改造了
   客厅和卧室，改造后非常温馨。
   请用小红书风格（口语化、emoji、分段明显）。
   标题要有数字+悬念，正文包含：改造前问题→改造思路→
   改造效果→费用清单。
   ```
2. 手动修改：加你的真实案例细节，去掉AI说错的
3. 用Canva做一张"改造前后对比"封面图

### ✅ 完成标准

- [ ] AI生成了符合小红书风格的文章
- [ ] 你手动修改了至少3处（加入真实细节）
- [ ] 准备好了封面图（改造前后对比）

## 🍅 番茄33：小红书AI内容生产（下）——避坑指南与知识帖

### 📖 费曼三句话

1. **避坑指南类比案例帖更好写、更容易爆**——因为"教你避免踩坑"天然有传播性。每个装修过的人都有槽要吐，你的指南帮他们说出来，他们就会转发。
2. **知识帖选题的三个来源**：客户问你的问题（"装修到底先买什么？"）、你自己的踩坑经历（"我第一次装修犯的5个错"）、行业变化（"2026年装修新规你必须知道"）。都是你最熟悉的素材。
3. **AI的"一次产出多平台分发"模式**：让AI把一个主题写成3个版本——小红书(短/有趣)、抖音(脚本)、公众号(深度)。一次思考，三倍产出。

### 🛠 刻意练习

1. 让AI生成一篇"避坑指南"：
   ```
   请写一篇小红书风格的"旧房翻新必看的5个避坑指南"。
   每个坑用"很多人不知道..."开头，
   最后给出解决方案。
   要求：口语化、用emoji、每段不超过3行。
   ```
2. 让AI把同一主题改成抖音脚本版本：
   ```
   请把以上避坑指南改成抖音短视频脚本。
   每条10-15秒，口播形式，需要开头3秒吸引注意力。
   ```
3. 对比两种风格的差异，哪个更适合你的表达习惯

### ✅ 完成标准

- [ ] 生成了一篇避坑知识帖
- [ ] 生成了对应的抖音脚本版本

## 🍅 番茄34：AI辅助短视频创作——改造过程记录

### 📖 费曼三句话

1. **改造过程视频 = 最真实的信任建立器**——客户看到你拆墙、布线、贴砖的全过程，比看100张精修效果图更有信任感。不完美才有真实感。
2. **AI在短视频中的作用不是"拍"，而是"剪"和"写"**：
   - AI写脚本（你口播什么）
   - AI剪映自动剪辑（智能去片头片尾）
   - AI自动字幕（识别语音生成字幕）
   - AI写标题和描述
3. **拍改造过程的最低配置**：手机支架+手机+每天拍3段10秒视频。开工前拍一段、中间拍一段、完工拍一段。三天素材就能剪一条"改造全过程"。

### 🛠 刻意练习

1. 用AI写一个"第一天拆除"的抖音口播脚本：
   ```
   请写一个30秒抖音口播脚本，主题是"老房改造第一天拆完了"。
   要求：开头3秒震撼画面+中间干货+结尾引导关注。
   语气：真实、不装、带点幽默。
   ```
2. 如果有已拍的装修视频素材，练习用剪映剪辑
3. 如果没有，用手机拍一段"你工作台的10秒"，练习剪映剪辑

### ✅ 完成标准

- [ ] AI生成了一个可用的口播脚本
- [ ] 剪映里完成了一段视频剪辑
- [ ] 你理解了"AI + 手机 + 剪映"的短视频工作流

## 🍅 番茄35：AI辅助短视频创作——知识干货类

### 📖 费曼三句话

1. **干货类短视频 = "用30秒回答客户一个问题"**——"装修到底先装什么？""旧房翻新多少钱一平米？""适老化改造政府有补贴吗？"每个问题就是一条视频，一个视频就是一个获客入口。
2. **批量生产的秘诀：一次录10条**——列出客户最常问的10个问题，用AI写好脚本，一次录完10条。每次3-5分钟，然后剪成10条30秒的视频，每周发2条，够用5周。
3. **AI辅助口播的两种方式**：①AI写脚本你照着读（需提词器）；②你口述大意让AI整理成文案（更自然，适合有经验的人）。新手推荐后者，更真实。

### 🛠 刻意练习

1. 用AI生产5条"旧房翻新常见问题"短视频脚本：
   ```
   请写出5条30秒抖音口播脚本，主题为旧房翻新常见问题：
   1. 旧房翻新到底要不要砸墙？
   2. 20年房龄的水电要不要全换？
   3. 旧房翻新预算怎么控制？
   4. 装修期间住哪里？
   5. 旧房翻新能省钱的3个地方
   每条包含：开头吸引+核心答案+结尾引导
   ```
2. 选其中1条，自己录一遍（手机录像+口播）
3. 用剪映加字幕，导出30秒成品

### ✅ 完成标准

- [ ] AI生成了5条短视频脚本
- [ ] 你实际录制了至少1条
- [ ] 完成了剪辑+字幕+导出

## 🍅 番茄36：公众号深度内容AI写作

### 📖 费曼三句话

1. **公众号是"信任深水区"**——小红书中了草，抖音看了热闹，但要客户真的掏钱找你，需要深度内容建立专业信任。一篇好的装修案例深度解析，抵得上10篇小红书。
2. **公众号深度内容的AI写作法**：先让AI写"长提纲"（不要直接写全文），你审提纲改方向，然后让AI按提纲一段段写。这样AI不会跑偏，你省80%的修改时间。
3. **AI写公众号的最优长度**：1500-2000字。太短没深度，太长没人看完。包含：一个完整案例（500字）+ 3个重点技巧（各300字）+ 总结（200字）+ 你的服务介绍（200字）。

### 🛠 刻意练习

1. 让AI写提纲：
   ```
   请为"80平米老房改造全记录"写一个公众号文章提纲，
   目标读者是准备装修的业主，需要体现专业性和可信度。
   提纲需包含：引人入胜的开头、改造前的问题描述、
   改造过程关键节点、改造后效果、费用明细、总结建议。
   ```
2. 调整提纲（加你实际的经验，删AI说的套话）
3. 让AI按调整后的提纲写出完整文章

### ✅ 完成标准

- [ ] AI生成了文章提纲
- [ ] 你调整了提纲（至少2处修改）
- [ ] AI按新提纲写出了完整文章

## 🍅 番茄38：AI驱动的评论区运营与私域转化

### 📖 费曼三句话

1. **评论区 = 免费的获客入口**——在小红书/抖音上，有100个人看到你的评论，就有10个人点进你的主页，1个人私信咨询。回复评论是最低成本的高效获客。
2. **AI帮你批量回复评论**——收集常见评论问题（"这个花了多少钱？""怎么联系你？""做了多久？"），让AI生成3种回复模板（真诚版、专业版、互动版），根据不同情况选择。
3. **从公域到私域的三步引导**：
   - 评论：回复专业+引导私信（"具体的材料清单我私信发您"）
   - 私信：提供价值+建立信任（不急着报价，先了解需求）
   - 微信：深度沟通+预约量房（加微信后发案例合集）

### 🛠 刻意练习

1. 让AI生成评论区回复模板：
   ```
   我是一名装修设计师，在小红书发装修案例。
   请为以下5种评论各生成3种回复风格(真诚/专业/互动)：
   1. "这个花了多少钱？"
   2. "在哪里找的设计师？"
   3. "老房子也能改这么好看吗"
   4. "求推荐施工队"
   5. "收藏了，以后装修找你"
   ```
2. 选择3个你认为最常用的回复风格
3. 模拟发一条内容+回复评论的完整流程

### ✅ 完成标准

- [ ] AI生成了15条评论回复（5问题×3风格）
- [ ] 你选了3个最贴近你说话风格的回复
- [ ] 理解了"评论→私信→微信"的转化路径

## 🍅 番茄39：AI数据分析——什么内容能带来客户

### 📖 费曼三句话

1. **内容获客的"北极星指标"不是点赞，是"私信咨询量"**——点赞多≠客户多。一条点赞500但没人私信的内容，不如一条点赞50但来了3个咨询的内容。
2. **AI帮你分析"什么内容能获客"**——把过去一个月的数据发给AI（点赞/收藏/评论/私信），让AI找出规律。"哪种标题类型的私信转化率最高？""哪种图片风格获客效果最好？"AI几分钟就能分析出你几个月才能总结出的规律。
3. **数据驱动的内容迭代**：做4种不同类型的内容各发2篇 → 2周后看哪种类型的私信最多 → 下个月多做这种类型 → 再测试新类型。每个月的获客效率都会提升。

### 🛠 刻意练习

1. 给AI发一段模拟数据做分析：
   ```
   以下是我上个月的内容数据，请分析哪种内容类型获客效果最好：
   - "3万改造60平老房"：点赞280，收藏150，私信8
   - "装修避坑5大要点"：点赞120，收藏300，私信3
   - "适老化改造全流程"：点赞80，收藏90，私信12
   - "日式风格客厅案例"：点赞200，收藏180，私信5
   - "旧房翻新预算表"：点赞60，收藏400，私信6
   请计算每个内容的"私信转化率"，告诉我还应该多做什么类型。
   ```
2. 根据分析结果，调整你的下月内容计划
3. 设定你的"私信转化率"基准线（比如≥5%就算好内容）

### ✅ 完成标准

- [ ] 用AI分析了内容数据
- [ ] 明确了哪种内容类型对你最有效
- [ ] 设定了你的内容效果评估标准
