# 电影批评

> 银幕世界是一处人为建构的幻想世界——电影研究、电影批评的意义，正在于揭示出电影的"书写"行为之所在。

## 核心理念

本书由戴锦华撰写，核心方法论围绕**以现代电影理论为工具进行影片精读**。每一章介绍一种电影理论流派，通过对一部影片的细读展示运用方法。

## 执行流程

### Phase 1: 选择理论框架

**可选框架**：视听语言与叙事分析、电影作者论、叙事学理论、精神分析女性主义、镜像理论、意识形态批评、第三世界寓言

### Phase 2: 影片细读

**分析维度**：
- 视听语言（机位、构图、场面调度、光与色彩）
- 叙事结构（组合段、时间、空间）
- 意识形态与社会语境
- 作者风格与主题

### Phase 3: 批评写作

**结构**：理论引入→影片细读→分析论证→结论

## 局限性

- **需要**：基本的电影理论知识和学术写作能力
- **不适用于**：即兴影评和商业电影推荐
