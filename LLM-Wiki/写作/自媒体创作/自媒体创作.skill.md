## 工作流程

1. **账号定位**：目标受众、内容方向、差异化策略
2. **选题策划**：热度评估、关键词挖掘、竞品分析
3. **内容创作**：标题设计、结构编排、语言风格
4. **平台适配**：各平台算法规则、发布时机、互动策略
5. **数据复盘**：阅读/完播/转化数据驱动优化

## 关键公式

- **爆款标题** = 痛点共鸣 + 意外反转 + 价值承诺
- **短视频脚本** = 黄金3秒钩子 + 内容主体 + 行动号召
- **文案转化** = 注意 → 兴趣 → 欲望 → 行动（AIDA）
