# 权限分级规则

> 定义书库内容的访问权限，保护专业临床内容

## 一、用户层级定义

### 层级1：普通用户 🔓
**开放内容**：基础心理学、情绪管理、自我成长

| 可访问类目 | 难度限制 | 标识 |
|------------|----------|------|
| 基础心理学 | 入门 | `[基础-入门]` |
| 社会心理学 | 入门 | `[基础-入门]` |
| 情绪压力管理 | 入门 | `[基础-入门]` |
| 人格心理学 | 入门部分 | `[基础-入门]` |
| 婚恋家庭心理 | 入门 | `[基础-入门]` |
| 自我成长类 | 入门 | `[基础-入门]` |

**开放技能**：
- Skill1：基础理论检索（部分）
- Skill5：情绪疏导（基础内容）
- Skill8：人格分析（入门内容）

---

### 层级2：进阶用户 🔐
**开放内容**：在普通用户基础上，增加进阶内容

| 可访问类目 | 难度限制 | 标识 |
|------------|----------|------|
| 发展心理学 | 入门/进阶 | `[进阶]` |
| 社会心理学 | 入门/进阶 | `[进阶]` |
| 人格心理学 | 入门/进阶 | `[进阶]` |
| 心理咨询技术 | 入门/进阶 | `[进阶]` |
| 婚恋家庭心理 | 入门/进阶 | `[进阶]` |
| 职场心理 | 全部 | `[进阶]` |

**开放技能**：
- Skill1-3：全部内容
- Skill5：全部内容
- Skill7-8：全部内容
- 儿童咨询技能：基础内容
- 职场咨询技能：全部内容
- 婚姻咨询技能：基础内容

---

### 层级3：从业者 🔒
**开放内容**：全部书库内容，包括临床诊断、危机干预

| 可访问类目 | 难度限制 | 标识 |
|------------|----------|------|
| **临床诊断** | 专业临床 | `[专业临床]` |
| **变态心理学** | 专业临床 | `[专业临床]` |
| **心理疗法** | 专业临床 | `[专业临床]` |
| **危机干预** | 专业临床 | `[专业临床]` |
| 其他所有类目 | 全部 | `[专业临床]` |

**开放技能**：
- Skill1-8：全部内容
- 所有细分技能：全部内容

**特别权限**：
- 诊断标准访问
- 危机干预话术
- 临床案例库
- 专业治疗技术

---

## 二、权限控制机制

### 内容标识规则
```yaml
# 书籍元数据中的权限字段
权限: [基础/进阶/专业临床]
```

### 访问检查流程
```
用户提问 → 识别用户层级 → 匹配问题所属内容权限
→ 权限匹配 → 返回答案
→ 权限不足 → 提示升级/返回基础版本
```

### 敏感内容保护
以下内容仅限从业者访问：
1. 精神障碍诊断标准
2. 自杀/自伤干预技术
3. 创伤治疗技术
4. 精神分析深度解读
5. 临床案例详情

---

## 三、权限升级路径

### 普通用户 → 进阶用户
- 完成心理学基础学习
- 通过基础知识测试
- 或：心理学相关专业学生

### 进阶用户 → 从业者
- 提供心理咨询师资质证明
- 或：临床心理学相关专业背景
- 或：完成平台专业培训认证

---

## 四、权限配置文件

```yaml
# 权限配置模板
user_levels:
  basic:
    name: 普通用户
    skills: [Skill1-basic, Skill5-basic, Skill8-basic]
    categories: [00, 02, 08, 10]
    max_difficulty: 入门
    
  advanced:
    name: 进阶用户
    skills: [Skill1, Skill2, Skill3, Skill5, Skill7, Skill8]
    categories: [00, 01, 02, 03, 06, 10, 11]
    max_difficulty: 进阶
    
  professional:
    name: 从业者
    skills: [Skill1, Skill2, Skill3, Skill4, Skill5, Skill6, Skill7, Skill8]
    categories: [all]
    max_difficulty: 专业临床

restricted_content:
  - category: 04 # 变态心理学
    level: professional
  - category: 05 # 临床诊断
    level: professional
  - category: 07 # 心理疗法
    level: professional
  - category: 09 # 危机干预
    level: professional
```
