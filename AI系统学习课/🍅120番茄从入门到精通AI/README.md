## 🎬 开幕词：一个关于"智能"的世纪悬案

1956年，达特茅斯学院的一个夏天，一群科学家聚在一起，讨论了一个听起来像科幻小说的想法：

**"能不能让机器像人一样思考？"**

他们不知道的是——这个问题的答案会在接下来的70年里，像多米诺骨牌一样倒下：

- 1997年，深蓝击败国际象棋世界冠军。人类说："但那只是计算，不是智能。"
- 2011年，Watson在《危险边缘》击败人类冠军。人类说："但那只是模式匹配，不是理解。"
- 2016年，AlphaGo击败围棋世界冠军。人类说："围棋虽然难，但它不会写诗。"
- 2023年，GPT-4通过律师资格考试。人类说："但……等等。"
- 2025年，Claude Code autonomously 修复了 Linux 内核的 bug。人类说："……"

**2026年，你坐在这里——而智能的定义，已经永远改变了。**

这不是一个"要不要学AI"的问题。这是一个"你如何在这场智能革命中不被淘汰"的问题。

这门课，就是你的入场券。

## 🧠 学习方法

| 武器 | 用法 | 在这套课里的角色 |
|------|------|------------------|
| **🍅 番茄工作法** | 25分钟专注 + 5分钟休息 | 每番茄就是一个"单元剂量"，刚好讲完一个悬疑故事+核心概念+动手实践 |
| **📝 费曼学习法** | 讲给外行听 | 每番茄末尾的费曼三句话逼你用自己的话复述复杂概念 |
| **🎯 刻意练习** | 针对弱点反复击打 | 每课的"探案练习"让你在真实代码/项目中应用概念 |
| **🔍悬疑叙事** | 用案件串联知识点 | 每个章节都是一个"悬疑案件"，驱动你往下读 |

### 120番茄怎么分配？

```
24 天 × 5 🍅/天 = 120 🍅
     ↑       ↑
   AI全景的  每个番茄25分钟专注，
  24个章节   刚好讲透一个核心命题
```

### 每番茄的节奏

```
⏰ 0-20 分钟：悬疑开场 → 核心概念 → 案例拆解
⏰ 20-22 分钟：费曼三句话（闭眼给自己讲一遍）
⏰ 22-25 分钟：刻意练习 + 记录困惑
⏰ 25-30 分钟：休息，让潜意识工作
```

## 🧩 每天的结构

每天5个番茄：

```
🍅1 → 悬疑开场 + 核心案例（"犯罪现场"引入）
🍅2 → 理论深度拆解（"作案手法"分析）
🍅3 → 代码/动手实践（"重演犯罪"）
🍅4 → 🧠 思维导图 + 费曼大复习
🍅5 → 刻意练习 + 跨界思考（"独立办案"）
```

每番茄末尾都有：
- ✅ **费曼三句话**（讲给自己听）
- ❓ **悬疑追问**（引发思考，不急于给答案）
- 📌 **连线思考**（和现实世界的连接点）
