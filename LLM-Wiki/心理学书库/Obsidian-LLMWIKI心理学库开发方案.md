# 适配Obsidian LLM-WIKI知识库｜123本心理学MD书库AI检索开发方案
版本：V1.1
适配环境：Obsidian + LLM-WIKI 本地知识库框架
存量资源：123本心理学MD笔记，覆盖基础理论、临床诊断、心理咨询全体系

## 一、Obsidian库内前置规整（贴合LLM-WIKI索引规则）
### 1. 统一笔记元数据Frontmatter
每本心理学笔记头部添加YAML元数据，LLM-WIKI可自动抓取分类、标签、技能关联
```yaml
---
title: 书名
category: 基础理论/临床诊断/心理咨询/危机干预
skill_tag: 心理评估,共情沟通,症状诊断,情绪疏导
level: 入门|进阶|临床专业
keywords: 核心关键词1,关键词2,关键词3
type: psychology
---
```

### 2. 库内文件夹目录分级（匹配LLM-WIKI检索路由）
```
LLM-WIKI/
├─心理学知识库/
   ├─01基础心理学
   ├─02发展心理学
   ├─03社会心理学
   ├─04人格心理学
   ├─05变态心理
   ├─06临床诊断
   ├─07心理咨询技术
   ├─08心理疗法实操
   ├─09情绪压力管理
   ├─10危机干预
   ├─11婚恋家庭心理
   └─12职场心理
```

### 3. 内容切片与内链优化
1. 按知识点、案例、诊断标准、咨询话术拆分段落，单片段适配LLM-WIKI嵌入检索窗口
2. 同类知识点双向内链关联，AI检索可联动调取关联笔记
3. 剔除重复内容，统一专业术语，保证库内语义一致性

## 二、LLM-WIKI架构适配检索逻辑
整体流程：Obsidian本地MD笔记 → LLM-WIKI索引扫描 → 向量缓存入库 → 标签/目录路由检索 → 绑定自定义SKILL技能 → 对话应答输出

### 1. LLM-WIKI索引配置要点
1. 指定**心理学知识库**文件夹为专属检索目录，限定扫描范围
2. 开启标签、分类、元数据联动索引，优先按skill_tag匹配技能模块
3. 设置文本嵌入阈值，适配123本笔记体量，平衡检索速度与精准度
4. 开启笔记溯源，回答可跳转对应Obsidian原笔记

### 2. 检索匹配规则适配
- 目录路由检索：按文件夹大类快速锁定领域笔记
- 标签技能检索：通过skill_tag直接触发对应SKILL功能
- 语义全文检索：理解问题语义，跨笔记整合知识点
- 层级权限检索：按level标签区分入门/专业内容输出

## 三、基于LLM-WIKI封装专属SKILL技能
技能直接绑定库内分类与标签，LLM-WIKI识别指令自动调取对应目录笔记

### Skill1 基础心理理论检索
触发词：心理概念、理论解析、基础常识
关联目录：01基础心理学
调取逻辑：匹配category、keywords标签，输出定义、原理、规律

### Skill2 临床症状与诊断判定
触发词：症状分析、心理障碍、临床判定
关联目录：05变态心理、06临床诊断
调取逻辑：对照笔记内诊断标准、症状特征做专业分析

### Skill3 心理咨询会谈技能
触发词：咨询对话、共情回应、来访沟通
关联目录：07心理咨询技术
调取逻辑：提取话术模板、会谈流程、沟通技巧作答

### Skill4 心理疗法实操应用
触发词：CBT、精神分析、正念、行为矫正
关联目录：08心理疗法实操
调取逻辑：拆解疗法步骤、实操案例、练习方式

### Skill5 情绪压力疏导调节
触发词：焦虑、抑郁、情绪内耗、压力缓解
关联目录：09情绪压力管理
调取逻辑：调用调节方法、放松方案、心理疏导思路

### Skill6 心理危机干预处置
触发词：轻生倾向、创伤应激、危机安抚
关联目录：10危机干预
调取逻辑：调取风险评估、干预话术、修复方案

### Skill7 人际婚恋家庭指导
触发词：亲子矛盾、婚姻情感、社交关系
关联目录：11婚恋家庭心理
调取逻辑：关系剖析、相处调整、情感心理解读

### Skill8 人格特质分析解读
触发词：性格分析、人格特质、性格判定
关联目录：04人格心理学
调取逻辑：性格成因、特质优缺点、相处适配建议

## 四、LLM-WIKI三种实际使用模式
### 1. 笔记知识检索问答
输入问题→LLM-WIKI识别标签目录→检索库内心理学笔记→整合内容回复，支持点击溯源原Obsidian笔记

### 2. 模拟心理咨询对话
锁定心理咨询SKILL，框架自动调取会谈技术笔记，以咨询师身份连贯对话疏导

### 3. 心理案例分析研判
上传/输入案例，跨目录调取诊断、疗法、干预笔记，完成案例剖析与方案制定

## 五、LLM-WIKI专属调用提示词
### 通用知识库检索提示词
```
基于Obsidian LLM-WIKI内123本心理学MD笔记作答，仅引用库内已有内容。
1. 识别问题匹配对应skill_tag与文件夹分类，调取对应笔记知识点
2. 不编造库外理论、偏方内容，专业表述贴合笔记原文
3. 复杂内容分层梳理，实操类给出清晰步骤
4. 关键内容可标注对应笔记名称，支持溯源查看
```

### 心理咨询对话提示词
```
启用心理咨询SKILL，调取LLM-WIKI心理咨询技术目录笔记。
以专业咨询师口吻沟通，共情倾听，依据库内会谈话术与疏导思路回应，
不随意下定疾病结论，给出科学心理调整建议。
```

### 临床诊断分析提示词
```
启用临床诊断SKILL，检索变态心理、临床诊断目录笔记。
对照笔记内诊断标准客观分析症状，说明判定依据与病症特征，给出专业评估方向。
```

## 六、Obsidian+LLM-WIKI优化迭代方案
1. 笔记增补：新增心理书籍转为MD存入对应目录，框架自动增量索引
2. 技能细分：新增儿童心理、职场咨询等子标签，细化SKILL调用粒度
3. 案例沉淀：从现有笔记提取经典案例，单独建目录纳入检索范围
4. 索引维护：定期刷新LLM-WIKI缓存，同步笔记修改、新增、删除内容
5. 上下文联动：开启会话记忆，连续咨询可关联历史检索笔记内容

## 七、落地部署操作步骤
1. 按既定文件夹结构，整理123本心理学笔记归类归档
2. 批量补全所有笔记YAML元数据标签
3. 在LLM-WIKI中选定心理学总目录为检索数据源，完成索引构建
4. 绑定8项自定义SKILL技能与对应目录、标签
5. 导入专属提示词模板，配置检索应答规则
6. 测试问答、咨询、案例分析功能，微调索引匹配精度
7. 日常更新笔记，定期刷新知识库索引即可
