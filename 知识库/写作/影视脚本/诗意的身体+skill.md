# 诗意的身体

> 一切都在动。一切都在演变与前进。一切都在弹跳与反射。——雅克·勒考克

## 核心理念

本书由雅克·勒考克撰写，核心方法论围绕**通过身体动作探索戏剧的本质**。中性面具、元素旅程、动作分析三大支柱构成从身体认知到戏剧创作的完整路径。

## 执行流程

### Phase 1: 身体认知
中性面具、元素旅程（水/火/风/土）、成为大自然、戏剧转化

### Phase 2: 动作技术
赋予动作意义、戏剧式特技体操、动作分析、日常动作出发、寻找经济性

### Phase 3: 空间与结构
一页白纸的空间意识、表演的结构、风格约束、造型与角色

### Phase 4: 创作生成
从即兴到文本、从身体到诗意、从个人到集体

## 局限性

- **适用于**：剧场表演者和导演
- **不适用于**：纯影视表演技术
