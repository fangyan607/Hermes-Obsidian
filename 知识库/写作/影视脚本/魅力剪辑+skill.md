# 魅力·剪辑：影视剪辑思维与技巧

> 剪辑不只是剪掉不好的东西——它是关于对结构、成色、运动及时间的处理。

## 核心理念

本书由傅正义主编，核心方法论围绕**从剪辑思维到剪辑技巧的完整工作流**。涵盖剪辑史、情节结构、剪辑流程、跳切技法等。

## 执行流程

### Phase 1: 建立情节走势结构图
读剧本、分析叙事结构、确定情节点、建立结构图

### Phase 2: 粗剪
完成每场戏的戏剧任务、确保叙事连贯性

### Phase 3: 精剪
调整细节、完善结构、确立叙事节奏、形成风格

### Phase 4: 跳切与风格化
理解跳切的叙事功能、有意识地运用跳切创造节奏

## 局限性

- **适用于**：影视剪辑实践和理论学习
- **不适用于**：纯数字后期技术
