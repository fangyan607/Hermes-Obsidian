# 🍅 40番茄速通 Blender 教程

> **从零到一掌握 Blender 全流程，从建模到 AI 整合，40个番茄打通 3D 创作**
> 适用人群：零基础入门者 / 3D 爱好者 / 独立创作者 / AI+3D 探索者

## 🧠 核心教学法

**番茄工作法 × 费曼学习法 × 刻意练习** 三体合一：

| 方法 | 作用 | 在本教程中的体现 |
|:----|:----|:--------------:|
| 🍅 番茄工作法 | 保持节奏 | 每🍅=25分钟专注+5分钟休息，4个一组后15分钟长休 |
| 📖 费曼学习法 | 深度学习 | 每番茄末尾写"费曼三句话"，用最简单的话解释核心概念 |
| 🎯 刻意练习 | 技能内化 | 每番茄有具体操作任务 + 明确的完成标准 checklist |

## 📊 模块总览

```
模块0：基础入门与界面      (5🍅)  安装部署/界面/基础操作/快捷键/基础建模
模块A：硬表面建模与修改器   (5🍅)  硬表面/修改器堆栈/拓扑/布尔/补面
模块B：材质渲染与UV烘培     (5🍅)  材质节点/UV展开/烘培/Cycles/EEVEE
模块C：几何节点与雕刻       (5🍅)  几何节点程序化/雕刻/Dyntopo/纹理绘制
模块D：特效动画与插件工作流  (5🍅)  粒子/物理/动画/相机/插件生态
模块E：场景与角色全流程     (5🍅)  场景构建/角色建模/绑定/权重/商业出图
模块F：AI+Blender整合       (5🍅)  MCP协议/AI建模/代码生成/工作流
模块G：综合实战与项目规范化  (5🍅)  全流程实战/规范/输出/总结

Total: 8模块 × 5🍅 = 40🍅
```

## 🧭 学习路径

### 路径A：零基础新手
```
模块0(全) → 模块A(🍅21-23) → 模块B(🍅26-28) → 模块C(🍅31-33)
→ 模块D(🍅36-38) → 模块G(🍅51-53)
```
~25🍅，掌握核心建模渲染能力

### 路径B：有建模基础想学 Blender
```
模块0(🍅4-5快速过) → 模块A → 模块B → 模块C → 模块E(🍅41-45)
```
重点在 Blender 特有的几何节点和全流程

### 路径C：想学 AI+3D
```
模块0(🍅1-3) → 模块B(🍅28-29) → 模块D(🍅36-37) → 模块F(全)
```
先搭环境再直冲 AI 整合

### 路径D：商业创作向
```
模块0(🍅1-5) → 模块A → 模块B → 模块E(🍅41-45) → 模块G(全)
```
覆盖商业出图全流程

## 🛠 环境要求

| 工具 | 版本要求 | 说明 |
|:----|:--------|:-----|
| Blender | **5.1+**（教程基于 5.1） | 免费开源，[blender.org](https://www.blender.org) 下载 |
| Python | 3.13（Blender 内置） | 无需额外安装 |
| uv | 最新版（MCP 用） | `pip install uv` |
| Claude Desktop | 最新版（可选，MCP 用） | Claude AI 桌面端 |
| Git | 可选 | 版本管理项目文件 |

## 📁 推荐项目结构

```
my_blender_project/
├── project.blend        # 主文件
├── assets/              # 外部资产
│   ├── textures/        # 贴图
│   ├── hdri/            # 环境贴图
│   └── models/          # 引用模型
├── renders/             # 渲染输出
├── exports/             # 导出文件
│   ├── fbx/
│   ├── obj/
│   └── glb/
├── scripts/             # Python 脚本
└── references/          # 参考图
```
