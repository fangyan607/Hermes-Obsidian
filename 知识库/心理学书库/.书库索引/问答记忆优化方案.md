# 问答记忆优化方案

> 实现上下文关联检索，支持连续咨询场景

## 一、问题分析

### 当前痛点
1. 每次问答独立，缺乏上下文记忆
2. 连续咨询时需要重复描述背景
3. 无法关联之前的对话内容
4. 用户体验不连贯

### 目标
- 上下文关联检索
- 连续咨询连贯调取书库内容
- 多轮对话记忆保持

---

## 二、技术方案

### 方案1：会话记忆机制

```yaml
# 会话记忆结构
session:
  id: SESSION-XXX
  user_id: USER-XXX
  start_time: YYYY-MM-DD HH:MM
  context:
    - topic: 当前话题
    - history: 对话历史
    - books_accessed: 已调用书籍
    - skills_used: 已触发技能
```

### 方案2：上下文向量存储

```
用户提问 → 生成问题向量
         → 检索相似历史问题
         → 关联历史答案
         → 结合当前问题作答
         → 存储新问答对
```

### 方案3：书库内容关联网络

```yaml
# 内容关联索引
content_links:
  "焦虑症定义":
    related:
      - "焦虑症诊断标准"
      - "焦虑症治疗"
      - "焦虑与抑郁区分"
    books:
      - [[变态心理学 第6版]]
      - [[伯恩斯新情绪疗法]]
```

## 三、实现步骤

### Phase 1：基础记忆
1. 记录每次对话的主题
2. 存储调用过的书籍和技能
3. 保持当前会话的上下文

### Phase 2：关联检索
1. 建立内容关联索引
2. 问题自动关联相关知识
3. 推荐后续可能的问题

### Phase 3：长期记忆
1. 用户画像建立
2. 常问问题记录
3. 偏好学习

---

## 四、对话流程设计

### 单轮问答
```
用户：什么是焦虑症？
AI：[调取Skill5 + 《变态心理学》]
    焦虑症是...（来源：《变态心理学》第X章）
    
    相关问题推荐：
    - 焦虑症如何治疗？
    - 焦虑症和抑郁症有什么区别？
```

### 多轮连续对话
```
用户：我最近总是很焦虑
AI：[记录主题：焦虑]
    理解您的感受。能具体说说焦虑的表现吗？

用户：晚上睡不着，心跳加速
AI：[上下文关联：焦虑 + 睡眠问题]
    [调取Skill5 + 《情绪急救》焦虑章节]
    这些可能是焦虑的躯体化表现...
    建议尝试：1. 深呼吸练习 2. 睡前放松技巧

用户：有什么放松技巧推荐？
AI：[上下文关联：焦虑 + 睡眠 + 放松]
    [调取Skill5 + 《正念》相关内容]
    推荐以下放松技巧：
    1. 渐进式肌肉放松（来源：《XXXX》）
    2. 4-7-8呼吸法...
```
