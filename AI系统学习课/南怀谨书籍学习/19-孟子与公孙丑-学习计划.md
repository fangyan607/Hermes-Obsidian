### 核心主题

《公孙丑篇》是孟子思想的重要篇章，讨论了"浩然之气"、仁政王道、人性本善等关键问题，是理解孟子哲学的核心文本。

完成本周学习后，你将能够：
- 理解"浩然之气"的含义与培养方法
- 掌握"四端之心"的内容

- 将孟子的智慧应用于现代生活

## Day 1：浩然之气

### 番茄钟1：我善养吾浩然之气（25分钟）

#### 核心原文

> 「我知言，我善养吾浩然之气。」「敢问何谓浩然之气？」曰：「难言也。其为气也，至大至刚，以直养而无害，则塞于天地之间。」

#### 南师解读要点

**1. 浩然之气的含义**

南师解释，"浩然之气"是孟子独创的概念——一种充盈天地、至大至刚的精神力量。

**2. 气的特点**

```
至大 → 无限广大
至刚 → 无比刚强
直养 → 正直培养
塞天地 → 充盈宇宙
```

**3. 培养方法**

南师强调，浩然之气需要"直养"——用正直、真诚的心来培养。

> **费曼自测**
>
> 请用自己的话解释"浩然之气"是什么。

---

### 番茄钟2：养气的方法（25分钟）

#### 核心原文

> 「其为气也，配义与道；无是，馁也。」

#### 南师解读要点

**1. 配义与道**

南师指出，浩然之气需要"义"与"道"来配合。

| 配义 | 配道 |
|------|------|
| 正义感 | 真理追求 |
| 道德勇气 | 价值坚守 |

**2. 无是馁也**

```
没有义与道 → 气就"馁"（空虚无力）
    ↓
有义与道 → 气才充实有力
```

**3. 集义所生**

南师强调，浩然之气是"集义所生"——长期积累正义行为而产生。

> **费曼自测**
>
> 请解释"集义所生"在日常生活中的应用。

## Day 2：四端之心

### 番茄钟3：恻隐与羞恶（25分钟）

#### 核心原文

> 孟子曰：「人皆有不忍人之心……恻隐之心，仁之端也；羞恶之心，义之端也。」

#### 南师解读要点

**1. 四端之心**

南师解释，"四端"是四种道德的萌芽。

| 端 | 含义 | 德 |
|----|------|-----|
| 恻隐之心 | 同情心 | 仁 |
| 羞恶之心 | 羞耻感 | 义 |
| 辞让之心 | 谦让心 | 礼 |
| 是非之心 | 判断力 | 智 |

**2. 恻隐之心**

```
见人受苦 → 心里不忍
    ↓
恻隐之心 → 仁的萌芽
    ↓
扩充 → 仁德养成
```

**3. 羞恶之心**

南师指出，羞恶之心是对不义之事感到羞耻。

> **费曼自测**
>
> 请举一个"恻隐之心"在现代社会中的例子。

---

### 番茄钟4：辞让与是非（25分钟）

#### 核心原文

> 「辞让之心，礼之端也；是非之心，智之端也。」

#### 南师解读要点

**1. 辞让之心**

南师解释，"辞让"是谦逊退让的美德。

**2. 是非之心**

```
是非之心 → 判断对错的能力
    ↓
智之端 → 智慧的萌芽
    ↓
扩充 → 明辨是非
```

**3. 四端扩充**

南师强调，四端只是萌芽，需要"扩充"才能成为完整的德行。

```
恻隐 → 扩充 → 仁
羞恶 → 扩充 → 义
辞让 → 扩充 → 礼
是非 → 扩充 → 智
```

> **费曼自测**
>
> 请解释为什么"四端"需要扩充。

## Day 3：王道政治

### 番茄钟5：保民而王（25分钟）

#### 核心原文

> 孟子曰：「保民而王，莫之能御也。」

#### 南师解读要点

**1. 王道的核心**

南师解释，"保民"即保护百姓，这是王道的根本。

**2. 王道与霸道**

| 王道 | 霸道 |
|------|------|
| 以德服人 | 以力服人 |
| 保民安民 | 征服统治 |
| 长久稳定 | 短暂强盛 |

**3. 保民的具体措施**

```
不违农时 → 发展生产
薄税敛 → 减轻负担
养生丧死无憾 → 社会保障
```

> **费曼自测**
>
> 请比较王道与霸道在现代领导中的表现。

---

### 番茄钟6：仁政的措施（25分钟）

#### 核心原文

> 「五亩之宅，树之以桑，五十者可以衣帛矣。」

#### 南师解读要点

**1. 具体的仁政措施**

南师详细讲解了孟子的仁政蓝图：

| 措施 | 效果 |
|------|------|
| 五亩之宅树桑 | 老人有衣穿 |
| 鸡豚狗彘之畜 | 老人有肉吃 |
| 百亩之田勿夺时 | 家有余粮 |

**2. 养教并重**

```
先养 → 满足物质需求
后教 → 施行道德教育
```

**3. 现代启示**

南师认为，现代社会也需要先保障民生，再推行教育。

> **费曼自测**
>
> 请分析现代社会中"养教并重"的政策。

## Day 4：大丈夫之道

### 番茄钟7：富贵不能淫（25分钟）

#### 核心原文

> 孟子曰：「富贵不能淫，贫贱不能移，威武不能屈，此之谓大丈夫。」

#### 南师解读要点

**1. 大丈夫的标准**

南师解释，这是孟子对理想人格的定义。

| 富贵不能淫 | 不被财富地位腐蚀 |
|------------|------------------|
| 贫贱不能移 | 不因贫穷改变志向 |
| 威武不能屈 | 不为权势低头 |

**2. 大丈夫的境界**

```
外物诱惑 → 不动心
    ↓
困境磨难 → 不移志
    ↓
权势压力 → 不屈节
```

**3. 现代意义**

南师认为，这是人格修养的最高境界。

> **费曼自测**
>
> 请举一个现代"大丈夫"的例子。

### 番茄钟8：居天下之广居（25分钟）

#### 核心原文

> 「居天下之广居，立天下之正位，行天下之大道。」

#### 南师解读要点

**1. 三句话的含义**

南师解释：

| 广居 | 仁 |
|------|-----|
| 正位 | 礼 |
| 大道 | 义 |

**2. 大丈夫的实践**

```
居仁 → 住在仁爱之中
立礼 → 站在礼义之上
行义 → 走在正义之路上
```

**3. 得志与不得志**

南师指出，无论得志与否，大丈夫都坚守原则。

> **费曼自测**
>
> 请解释"居天下之广居"的含义。
