# 📚 社会学科番茄&费曼教程创建SKILL

> **触发词**：当你需要创建**人文学科/社会科学**（心理学、社会学、哲学、历史、政治学、人类学等）的系统性学习教程时，加载此 SKILL。
> 
> **使用场景**：将某个社会科学主题转化为**N个番茄钟**的结构化悬疑风格学习文档。

---

## 📋 快速参数

| 参数 | 默认 | 说明 |
|------|------|------|
| 总番茄数 | 按用户要求 | 推荐10/20/40/60个 |
| 每日番茄 | 5🍅/天 | 刚好符合人脑每日学习极限 |
| 每个番茄时长 | 25分钟专注+5分钟休息 | 标准番茄钟 |
| 叙事风格 | 悬疑故事 + 幽默直白 | 社会学科特别适配 |
| 学习法组合 | 番茄×费曼×刻意练习 | 三体合一 |

---

## 🧭 核心方法论

### 1. 人文社科专属三体学习法

```
🍅 番茄工作法 → 控制节奏（每个番茄一个小单元）
📝 费曼学习法 → 确保理解（每番茄末尾自我复述）
🎯 刻意练习   → 深度加工（针对弱点反复击打）
```

**关键区分（为什么社会学科和其他学科不同）：**

| 维度 | 理工科教程 | 社会科学教程 |
|------|-----------|-------------|
| 核心载体 | 代码/公式/操作步骤 | 人物/故事/案例/观点冲突 |
| 理解方式 | 逻辑推导 | 叙事共鸣+思辨 |
| 练习方式 | 动手操作 | 悬疑推理+情景分析 |
| 风格需要 | 精确简洁 | 有温度有态度 |
| 思维导图 | 流程/架构图 | 人物关系/理论谱系 |

### 2. 悬疑叙事驱动法（社会学科特供）

社会知识的最大敌人是**无聊**。社会学科学的是"人自己"——而"人"本身就是最大的悬疑。这套方法论的核心是**每个番茄 = 一个悬疑单元**：

```
每个番茄的结构：
┌──────────────────────────────────┐
│  🎬 悬疑钩子（开头30秒）         │
│  "你有没有想过……为什么……"      │
│  "想象一下这样的场景……"          │
├──────────────────────────────────┤
│  📖 叙事主体（15-18分钟阅读量）  │
│  人物故事 → 核心概念 → 实验/案例 │
│  ↓ 费曼三句话（2分钟闭眼复述）    │
├──────────────────────────────────┤
│  ❓ 悬疑追问（引发深度思考）      │
│  📌 连线笔记（链接到生活）        │
├──────────────────────────────────┤
│  ⏰ 休息5分钟                     │
└──────────────────────────────────┘
```

### 3. 悬疑钩子类型库

以下悬疑钩子可任意组合使用：

| 类型 | 公式 | 示例（心理学） |
|------|------|---------------|
| **颠覆常识型** | "你相信X，但真相是Y" | "你觉得自己在自由选择？你的大脑在7秒前就做了决定" |
| **黑暗秘密型** | "有一个你不知道的事" | "弗洛伊德和荣格决裂的真正原因，比你知道的复杂得多" |
| **道德困境型** | "如果是你，你会怎么做？" | "如果权威让你电击陌生人到450伏，你会照做吗？" |
| **身份颠覆型** | "你其实不是你以为的那种人" | "你'爱'你的母亲——弗洛伊德说，那可能不只是爱" |
| **历史悬案型** | "有一个实验再也没有被重复" | "小阿尔伯特后来怎么样了？没人知道——华生没有追踪他" |
| **细思极恐型** | "这个日常现象的背后……" | "你每天解锁手机200次——你知道斯金纳箱就在你口袋里吗？" |

### 4. 费曼三句话模板（社会学科适配版）

```markdown
🧠 **费曼三句话**
1. [这个概念的本质是什么？一句话说清——用比喻]
2. [它和我/人类有什么关系？——举一个生活案例]
3. [我还有什么不懂的/想反驳的？——记下来思考]
```

## 🏗️ 教程架构设计指南

### 步骤1：确定总番茄数

社会学科推荐的学习路径：

| 总番茄 | 适合 | 覆盖深度 |
|--------|------|---------|
| 10🍅 (2天) | 单一流派/人物 | "认识一个人"的水平 |
| 20🍅 (4天) | 单一学科分支 | "入门到能聊"的水平 |
| 40🍅 (8天) | 跨分支系统了解 | "能给别人讲"的水平 |
| 60🍅 (12天) | 学科全景图 | "建立自己的知识体系" |

### 步骤2：划分每日主题（12天5🍅范例）

```
Day 01 → 起源（从何而来）
Day 02 → 第一流派/人物
Day 03 → 第二流派/人物（常为对立派）
Day 04 → 第三流派/人物（常为折衷/超越派）
Day 05 → 方法论革命
Day 06 → 人本转向
Day 07 → 认知转向
Day 08 → 社会/群体层面
Day 09 → 发展/生命周期
Day 10 → 个体差异
Day 11 → 应用/实践
Day 12 → 前沿+总复习
```

### 步骤3：填充每日5🍅

每天结构固定：

```
🍅1 → 悬疑开场 + 核心人物/事件故事（叙事驱动）
🍅2 → 核心理论/概念拆解（深度理解）
🍅3 → 关键实验/案例/文本分析（"证据"环节）
🍅4 → 🧠 思维导图 + 费曼大复习
🍅5 → 刻意练习 + 跨界思考（应用到生活）
```

### 步骤4：每日思维导图

使用 **mermaid mindmap** 格式。社会学科的思维导图应包含：

- 人物谱系（师承、恩怨、流派）
- 理论谱系（从哪个问题出发？发展到哪？）
- 核心概念树
- 关键实验/案例
- 当代影响

### 步骤5：刻意练习设计

```markdown
### 刻意练习三件套

1. **案例分析**：给一个真实/虚构情境，让用户用刚学的理论分析
   - "用XX理论解释这个现象"
   - "XX和YY会分别如何看待这个案例？"

2. **情景推理**：让用户在两个竞争理论之间做选择
   - "如果你是XX学派的治疗师，你会怎么处理？"

3. **反身性思考**：让用户把理论应用到自身
   - "你的出生顺序是否符合阿德勒的描述？"
   - "你的依恋类型是什么？它如何影响你的关系？"
```

## ✍️ 写作风格手册

### 语调原则

```
✅ 要做：
- 像在酒吧里和聪明朋友聊天
- 有立场，有态度，敢毒舌
- 用故事串理论
- 先给结论再给证据（悬疑手法）
- 尊重读者智力，不要过度解释

❌ 不要做：
- 教科书式的"首先……其次……最后"
- 虚假客观（社会学科没有纯客观）
- 鸡汤/正能量
- 过度使用术语
- 长篇大论不分段
```

### 幽默工具箱

| 手法 | 说明 | 示例 |
|------|------|------|
| **反差** | 严肃内容+意外吐槽 | "弗洛伊德说梦是愿望的满足。他梦到了什么？可能是一根没人打断他抽的雪茄" |
| **现代类比** | 用当代事物解释古老概念 | "荣格的集体潜意识 ≈ 人类共有的操作系统，个体潜意识 ≈ 你安装的应用程序" |
| **自我揭短** | 先承认理论的荒谬再解释 | "第一次听到'俄狄浦斯情结'你的反应是什么？正常人都会说：什么玩意儿？" |
| **夸张对比** | 用一个极端的比喻 | "弗洛伊德和荣格的关系，相当于乔布斯和沃兹尼亚克——只不过他们吵的不是Apple II，是人类的灵魂" |

### 悬念设计流程

```
问题1：这个故事最反常识的点是什么？→ 这就是你的开场
问题2：读者会问什么"然后呢"？→ 这就是你的节奏
问题3：读者会在哪里觉得"这和我有什么关系"？→ 在那之前插入连线笔记
问题4：哪里适合让读者停下来思考？→ 放费曼三句话
```

## 🚀 创作流程速查

### 启动模板

当用户说"我要学XX"时：

```markdown
1. 确定总番茄数（10/20/40/60）
2. 拆分为N天（总番茄÷5）
3. 规划每日主题（覆盖从起源到当代）
4. 为每个番茄设计悬疑钩子
5. 挖掘每个理论的"人物故事"（理论需要血肉）
6. 每番茄写完后停顿，问：这个知识点，读者能在生活中用吗？
7. 每天末尾画mermaid思维导图
8. 设计刻意练习（案例→推理→反身）
```

### 参考文献处理

引用书籍时，使用 Wikilinks：

```markdown
参考了 [[书库/心理学书库/梦的解析]] 中关于潜意识的理论
```

### 文件命名规范

```
{Day编号}-{主题缩写}.md
示例：
Day01-心理学的诞生·从哲学到科学的惊险一跃
Day02-弗洛伊德与潜意识的暗黑大陆
```

## ⚖️ 社会学科创作伦理检查

社会科学涉及真实的人、真实的历史伤害、真实的痛苦。创作时自查：

- [ ] 是否尊重历史人物的复杂性（不把他们简化为标签）？
- [ ] 是否交代了争议/批评（不只讲一面之词）？
- [ ] 是否避免了"受害者心理学"（不把创伤浪漫化）？
- [ ] 是否区分了"科学结论"和"理论推测"？
- [ ] 是否对非西方文化有足够尊重（避免西方中心主义）？
- [ ] 幽默是否建立在论点之上，而非取笑真实受害者？
