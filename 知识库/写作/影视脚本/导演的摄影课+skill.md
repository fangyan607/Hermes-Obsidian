# 导演的摄影课

> 导演和摄影师都在不断地探寻着一起行动，去实现那些他们任何一个人都不可能单独完成的目的。

## 核心理念

本书由杰奎琳·弗洛斯特撰写，核心方法论围绕**导演与摄影师的创造性协作**。以7位ASC顶级摄影师的经验为基础，提供导演与摄影师沟通合作的完整体系。

## 执行流程

### Phase 1: 选择摄影师
评估摄影风格、沟通方式、合作历史

### Phase 2: 确定视觉风格
色彩基调、照明风格、镜头语言、借鉴参考片

### Phase 3: 技术前期筹备
镜头选择、灯光方案、拍摄计划、测试

### Phase 4: 实拍协作
现场沟通、创意调整、效率平衡

### Phase 5: 后期确认
调色方案、DI审查、最终画面确认

## 决策规则

| 条件 | 行动 |
|------|------|
| 预算有限 | 优先保证照明质量 |
| 时间紧迫 | 提前做技术预演 |
| 风格不确定 | 制作视觉参考板 |

## 局限性

- **不适用于**：纯技术摄影操作手册、数字摄影机操作指南
