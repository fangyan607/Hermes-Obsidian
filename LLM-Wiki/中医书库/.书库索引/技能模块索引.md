## 二、技能调用规则

### 调用优先级

```yaml
优先级1: 精准匹配
  - 用户指令直接触发对应技能
  
优先级2: 语义理解
  - AI识别意图，匹配最相关技能
  
优先级3: 多技能联动
  - 复杂问题调用多个技能组合
```

### 典型调用组合

| 场景 | 技能组合 |
|------|----------|
| 感冒问诊 | Skill2(辨证) + Skill3(药材) + Skill4(方剂) |
| 体质调理 | Skill6(体质) + Skill8(食疗) |
| 针灸治疗 | Skill5(穴位) + Skill2(辨证) |
| 慢病调理 | Skill7(病症) + Skill3(药材) + Skill8(康养) |
