## 🍅 番茄41：内容营销策略——装饰行业获客底层逻辑

### 📖 费曼三句话

1. **内容获客 = "让客户在需要你的时候第一个想到你"**——不是发广告，是持续发有价值的内容。客户刷到你10次装修知识帖之后，自己家要装修时，第一个想到的就是你。
2. **装饰行业内容获客的"漏斗模型"**：
   - 顶部（曝光）：抖音/小红书 → 让1万人看到你
   - 中部（互动）：评论/私信 → 让100人咨询你
   - 底部（成交）：见面/量房 → 让10人选择你
   - 每个环节都靠内容驱动。
3. **AI让内容生产成本降低到传统方式的1/5**——一篇案例帖从4小时缩短到1小时，一条视频从6小时缩短到1.5小时。省下的时间就是你的"获客数量优势"。
4. **LLMWIKI知识库 = 你的"内容弹药库"**——在Obsidian中建立一个装饰行业LLM-WIKI知识库，把每个项目、每种材料、每个风格都写成结构化笔记。AI生成内容时不再"凭空创作"，而是从知识库中提取真实案例和数据，产出内容更专业、更有说服力。知识库结构：`案例库/`、`材料知识/`、`风格指南/`、`内容日历/`。

### 🛠 刻意练习

1. 用AI分析你的目标客户：
   ```
   我是一名旧房翻新和适老化改造设计师，我的目标客户是
   35-60岁的城市居民，有老房需要翻新或家有老人需要适老改造。
   请帮我分析：
   1. 这类客户每天在手机上搜什么关键词？
   2. 他们最关心的5个问题是什么？
   3. 什么样的内容最容易打动他们？
   ```
2. 根据AI的分析，确定你本月的内容主题方向（3个）
3. **在Obsidian中创建你的LLMWIKI知识库结构**：
   ```
   装饰行业LLMWIKI/
   ├── 案例库/          # 每个真实项目一条笔记
   │   ├── 案例模板.md
   │   └── ...
   ├── 材料知识/        # 地板/涂料/瓷砖
   ├── 风格指南/        # 北欧/日式/现代
   ├── 内容日历/        # 月度内容规划
   └── 客户画像/        # 目标客户分析
   ```

### ✅ 完成标准

- [ ] AI输出了目标客户画像和内容方向
- [ ] 你确定了3个内容主题方向
- [ ] 理解了"曝光→互动→成交"的漏斗

## 🍅 番茄42：小红书AI内容生产（上）——案例帖

### 📖 费曼三句话

1. **小红书是装饰行业最好的获客平台**——用户主动搜"旧房翻新""装修避坑""客厅改造"，带着需求来的。一篇爆款改造案例帖，可以持续带来半年的咨询。
2. **爆款案例帖的公式**：
   - 封面：改造前后对比图（冲击力！）
   - 标题：数字+结果+悬念（"花3万把60平米老房变成这样，邻居都来问"）
   - 正文：改造前的问题 → 改造思路 → 改造后的效果 → 费用明细
   - 标签：#旧房翻新 #小户型改造 #装修案例 #省钱装修
3. **AI辅助案例帖写作流程**：你口述或写大概 → AI润色成小红书风格 → 你检查本地信息 → AI配图方案 → 发布。全程1小时。
4. **用写作SKILL + 宝玉插图SKILL生成高质量案例帖**——写作SKILL负责文案（小红书风格、口语化、有节奏），宝玉插图SKILL(baoyu-article-illustrator)自动为文章配图。在Obsidian中：`/write 案例帖 参数` → AI生成文案 → `/illustrate 参数` → AI配图 → 存入LLMWIKI → n8n发布。

### 🛠 刻意练习

1. 用AI写一篇案例帖初稿：
   ```
   请写一篇小红书画风的老房改造案例帖。
   基本信息：60平米老房，预算8万，北欧风格，主要改造了
   客厅和卧室，改造后非常温馨。
   请用小红书风格（口语化、emoji、分段明显）。
   标题要有数字+悬念，正文包含：改造前问题→改造思路→
   改造效果→费用清单。
   ```
2. 手动修改：加你的真实案例细节，去掉AI说错的
3. 用Canva做一张"改造前后对比"封面图
4. （可选）用宝玉插图SKILL为文章生成配图建议

### ✅ 完成标准

## 🍅 番茄44：AI辅助短视频创作——改造过程记录

### 📖 费曼三句话

1. **改造过程视频 = 最真实的信任建立器**——客户看到你拆墙、布线、贴砖的全过程，比看100张精修效果图更有信任感。不完美才有真实感。
2. **AI在短视频中的作用不是"拍"，而是"剪"和"写"**：
   - AI写脚本（你口播什么）
   - AI剪映自动剪辑（智能去片头片尾）
   - AI自动字幕（识别语音生成字幕）
   - AI写标题和描述
3. **拍改造过程的最低配置**：手机支架+手机+每天拍3段10秒视频。开工前拍一段、中间拍一段、完工拍一段。三天素材就能剪一条"改造全过程"。

### 🛠 刻意练习

1. 用AI写一个"第一天拆除"的抖音口播脚本：

   ```
   要求：开头3秒震撼画面+中间干货+结尾引导关注。
   语气：真实、不装、带点幽默。
   ```
2. 如果有已拍的装修视频素材，练习用剪映剪辑
3. 如果没有，用手机拍一段"你工作台的10秒"，练习剪映剪辑

## 🍅 番茄45：AI辅助短视频创作——知识干货类

### 📖 费曼三句话

1. **干货类短视频 = "用30秒回答客户一个问题"**——"装修到底先装什么？""旧房翻新多少钱一平米？""适老化改造政府有补贴吗？"每个问题就是一条视频，一个视频就是一个获客入口。
2. **批量生产的秘诀：一次录10条**——列出客户最常问的10个问题，用AI写好脚本，一次录完10条。每次3-5分钟，然后剪成10条30秒的视频，每周发2条，够用5周。
3. **AI辅助口播的两种方式**：①AI写脚本你照着读（需提词器）；②你口述大意让AI整理成文案（更自然，适合有经验的人）。新手推荐后者，更真实。

   ```
   每条包含：开头吸引+核心答案+结尾引导
   ```

3. 用剪映加字幕，导出30秒成品

## 🍅 番茄46：公众号深度内容AI写作

### 📖 费曼三句话

1. **公众号是"信任深水区"**——小红书中了草，抖音看了热闹，但要客户真的掏钱找你，需要深度内容建立专业信任。一篇好的装修案例深度解析，抵得上10篇小红书。
2. **公众号深度内容的AI写作法**：先让AI写"长提纲"（不要直接写全文），你审提纲改方向，然后让AI按提纲一段段写。这样AI不会跑偏，你省80%的修改时间。
3. **AI写公众号的最优长度**：1500-2000字。太短没深度，太长没人看完。包含：一个完整案例（500字）+ 3个重点技巧（各300字）+ 总结（200字）+ 你的服务介绍（200字）。
4. **LLMWIKI + 写作SKILL = 深度内容流水线**——从LLMWIKI案例库中选一个真实项目 → 写作SKILL生成初稿 → 你审核修改 → huashu-design SKILL排版 → 发布。全程45分钟，产出带真实案例的深度文章。

## 🍅 番茄48：AI驱动的评论区运营与私域转化

### 📖 费曼三句话

1. **评论区 = 免费的获客入口**——在小红书/抖音上，有100个人看到你的评论，就有10个人点进你的主页，1个人私信咨询。回复评论是最低成本的高效获客。
2. **AI帮你批量回复评论**——收集常见评论问题（"这个花了多少钱？""怎么联系你？""做了多久？"），让AI生成3种回复模板（真诚版、专业版、互动版），根据不同情况选择。
3. **从公域到私域的三步引导**：
   - 评论：回复专业+引导私信（"具体的材料清单我私信发您"）
   - 私信：提供价值+建立信任（不急着报价，先了解需求）
   - 微信：深度沟通+预约量房（加微信后发案例合集）

## 🍅 番茄49：AI数据分析——什么内容能带来客户

### 📖 费曼三句话

1. **内容获客的"北极星指标"不是点赞，是"私信咨询量"**——点赞多≠客户多。一条点赞500但没人私信的内容，不如一条点赞50但来了3个咨询的内容。
2. **AI帮你分析"什么内容能获客"**——把过去一个月的数据发给AI（点赞/收藏/评论/私信），让AI找出规律。"哪种标题类型的私信转化率最高？""哪种图片风格获客效果最好？"AI几分钟就能分析出你几个月才能总结出的规律。
3. **数据驱动的内容迭代**：做4种不同类型的内容各发2篇 → 2周后看哪种类型的私信最多 → 下个月多做这种类型 → 再测试新类型。每个月的获客效率都会提升。

### 🛠 刻意练习

1. 给AI发一段模拟数据做分析：
   ```
   以下是我上个月的内容数据，请分析哪种内容类型获客效果最好：
   - "3万改造60平老房"：点赞280，收藏150，私信8
   - "装修避坑5大要点"：点赞120，收藏300，私信3
   - "适老化改造全流程"：点赞80，收藏90，私信12
   - "日式风格客厅案例"：点赞200，收藏180，私信5
   - "旧房翻新预算表"：点赞60，收藏400，私信6
   请计算每个内容的"私信转化率"，告诉我还应该多做什么类型。
   ```
2. 根据分析结果，调整你的下月内容计划
3. 设定你的"私信转化率"基准线（比如≥5%就算好内容）

### ✅ 完成标准

- [ ] 明确了哪种内容类型对你最有效

## 🍅 番茄50：n8n内容自动化工作流搭建

### 📖 费曼三句话

1. **n8n内容自动化 = "一次配置，永不手动"**——内容获客最花时间的不是"写"，而是"分发"和"监控"。n8n可以把以下流程自动化：监控RSS/社交媒体 → AI生成摘要 → 生成多平台内容 → 排期发布 → 收集评论数据 → AI分析反馈。你的工作从"执行者"变成"审核者"。
2. **内容自动化的核心n8n工作流**：Trigger（定时触发/事件触发）→ AI节点（用Ollama/Hermes或云端API生成内容）→ 处理节点（格式化/适配平台）→ 输出节点（保存/通知/发布）。每个节点可插拔替换。
3. **装饰行业内容自动化的三个典型工作流**：①每周自动生成一篇小红书装修案例帖 ②监控装修话题下的热门问题，自动生成回答草稿 ③客户评论自动分类（咨询/吐槽/表扬）并分配给不同的处理流程。
4. **n8n + Obsidian = 自动化内容工厂**——n8n从Obsidian的LLMWIKI中读取案例笔记 → 调用Hermes生成多平台内容 → 保存回Obsidian内容日历 → 通知你审核。你只需维护LLMWIKI知识库，n8n负责生产。

   ```
   Schedule Trigger（每周一9:00）

       Prompt: "请生成一篇小红书风格的旧房翻新案例帖..."
     → 格式化节点（转为Markdown+图片占位）
     → 保存到本地文件
     → 发送通知到你的手机
   ```

2. **具体操作步骤**：
   - 打开 n8n (http://localhost:5678)
   - 新建工作流，添加 "Schedule Trigger" 节点（设置每周触发）
   - 添加 "HTTP Request" 节点，调用 Ollama API：
     ```
     URL: http://localhost:11434/api/generate
     Method: POST
     Body (JSON): {
       "model": "hermes3:8b",
       "prompt": "写一篇小红书风格的装修案例帖：60平米老房翻新，预算8万，北欧风格。包含标题、正文、标签。",
       "stream": false
     }
     ```
   - 添加 "Code" 节点，从API响应中提取生成的文本
   - 添加 "Write Binary File" 节点，保存到本地

3. **测试运行**：手动触发一次，检查内容是否成功生成
