## 🎯 学习目标

### 核心能力（番茄目标）

完成本教程后，你将掌握：

1. **云服务器部署能力**：在阿里云Ubuntu 24.04上从零搭建Hermes Agent服务
2. **多端同步能力**：实现本地Obsidian ← → GitHub ← → 服务器Hermes 三端同步
3. **LLM-WIKI知识库构建**：掌握Obsidian知识库设计原则与落地方法
4. **Agent实战能力**：用Hermes部署心理咨询、法律咨询、写作创作等AI Agent
5. **生产级运维能力**：安全配置、性能优化、日志监控

### 费曼检验标准

学完后，你能用通俗语言向他人解释：
- "什么是Hermes Agent？它怎么工作？"
- "知识库如何与AI Agent结合？"
- "Obsidian如何成为AI的知识来源？"

## 📐 教学法说明

### 🍅 番茄工作法

```
每番茄 = 25分钟专注 + 5分钟休息
每4番茄 = 一组 + 15分钟长休息
30番茄 = 7组 + 2个单番茄 ≈ 15小时
```

**学习节奏建议**：
- **天模式**：每天4-6番茄 → 5-7天完成
- **周模式**：每周10番茄 → 3周完成
- **沉浸模式**：周末集中10-15番茄 → 2个周末完成

### 📖 费曼学习法

每个知识点遵循四步循环：

```
① 学习概念 → ② 用自己的话复述 → ③ 找出理解漏洞 → ④ 简化并类比
```

教程中每个模块末尾都有**费曼检验**环节，请务必口述或写下来。

### 💪 刻意练习

每个模块配有：
- **基础练习**：验证性操作（确保理解）
- **挑战练习**：组合性任务（提升应用能力）
- **项目练习**：开放性课题（培养创新能力）
