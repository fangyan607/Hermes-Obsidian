## 🎬 开幕词：每个症状都是一个谜题

想象一下——

公元前2000年，一个商朝的巫师对着甲骨刻下："王疾齿，唯父乙祟"——王的牙疼，是祖先在作祟。这是中国人最早的"病历"。

公元前500年，扁鹊路过齐国，看了一眼齐桓侯的脸色，说："你有病，在皮肤里。"桓侯不信。五天后扁鹊又说："病到血脉了。"桓侯还是不信。又五天："病到肠胃了。"桓侯怒了。再五天：扁鹊远远看见桓侯，转身就跑——已经没救了。

公元205年，东汉大疫，"白骨露于野，千里无鸡鸣"。一个叫张仲景的长沙太守，在死人堆里写了一本改变东亚医学史的书。

公元2026年，你坐在屏幕前，浑身哪儿都不舒服，但体检报告说"正常"。

**你确定"正常"吗？**

中医说：你已经"失衡"了——只是还没到仪器能检测出来的程度。

## 🧠 学习方法：番茄×费曼×刻意练习的三体合一

| 武器 | 用法 | 在这套课里的角色 |
|------|------|------------------|
| **🍅 番茄工作法** | 25分钟专注 + 5分钟休息 | 每个番茄就是一个"单元剂量"，刚好够讲完一个悬疑故事+核心概念+费曼自测 |
| **📝 费曼学习法** | 假装你要讲给一个外行听 | 每个番茄末尾的"费曼三句话"逼你用自己的话说出来 |
| **🎯 刻意练习** | 针对弱点反复击打 | 每课的"探案练习"让你在真实案例中应用概念 |

### 100番茄怎么分配？

```
20 天 × 5 🍅/天 = 100 🍅
     ↑       ↑
 中医通史的   每个番茄25分钟
 20个章节     专注，刚好讲透一个命题
```

### 每个番茄的节奏

```
⏰ 0-20 分钟：悬疑开场 + 核心概念 + 案例拆解
⏰ 20-22 分钟：费曼三句话（闭眼给自己讲一遍）
⏰ 22-25 分钟：刻意练习 + 记录困惑
⏰ 25-30 分钟：休息，喝口水，让潜意识工作
```

### 刻意练习三大循环（贯穿全程）

| 循环 | 阶段 | 内容 |
|:-----|:-----|:-----|
| 🔄 循环1 | 模仿 | Day01-07：用中医框架分析给定的案例（历史上的医案） |
| 🔄 循环2 | 变式 | Day08-14：在不同病种间切换应用辨证方法 |
| 🔄 循环3 | 创造 | Day15-20：自主完成从望诊到处方的完整临床思维链 |

## 🧩 每天的结构

每天 5 个番茄，结构如下：

```
🍅1 → 悬疑开场 + 核心人物/事件（叙事驱动）
🍅2 → 核心理论拆解（概念深度）
🍅3 → 经典案例/实验（"证据"环节）
🍅4 → 🧠 思维导图/体系总结 + 费曼大复习
🍅5 → 刻意练习 + 跨界思考（"应用到临床"）
```

每个番茄末尾都有：
- ✅ **费曼三句话**（讲给自己听）
- ❓ **悬疑追问**（引发思考，不急于给答案）
- 📌 **连线思考**（和现实身体/生活的连接点）

## 📖 如何使用本教程

### 学习节奏建议

| 节奏 | 时间 | 适合人群 |
|------|------|----------|
| 🐢 悠闲模式 | 每天1天（5🍅≈2.5h） | 20天学完全部 |
| 🐇 标准模式 | 每天2天（10🍅≈5h） | 10天学完全部 |
| 🚀 密集模式 | 每天4天（20🍅≈10h） | 5天急行军（不推荐！） |

### 费曼输出模板（每个番茄结束时用）

```markdown
🧠 **费曼三句话**
1. [这个概念的本质是什么？一句话说清]
2. [它和我的身体/生活有什么关系？举一个例子]
3. [我还有什么不懂的/想反驳的？记下来]
```

### 必备工具

- 📓 笔记本（纸质的更好——费曼学习法要求手写）
- 🕐 番茄钟计时器（手机App或实体钟）
- 💧 一杯水（每番茄喝一口，保持大脑湿润）
