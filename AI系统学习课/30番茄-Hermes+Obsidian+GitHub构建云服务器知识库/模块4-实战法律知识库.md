# 合同的订立

> **核心法条**：《民法典》第469条-第501条

## 一、合同的形式

合同可以是以下形式：
1. **书面形式**（第469条）— 合同书、信件、电报、传真等可以有形表现所载内容的形式
2. **口头形式** — 即时清结的合同
3. **其他形式** — 以行为默示（如自助售货机购物）

> ⚠️ **注意**：法律、行政法规规定或当事人约定采用书面形式的，应当采用书面形式。

## 二、合同订立的方式

### 要约（第472-476条）
- **定义**：希望与他人订立合同的意思表示
- **条件**：
  - 内容具体确定
  - 表明经受要约人承诺，要约人即受该意思表示约束
- **要约邀请** vs **要约**：
  - 拍卖公告、招标公告、商业广告等通常为要约邀请
  - 商业广告内容符合要约条件的，视为要约

### 承诺（第479-489条）
- **定义**：受要约人同意要约的意思表示
- **条件**：
  - 由受要约人作出
  - 在要约有效期内到达
  - 内容与要约一致
- **承诺生效时间**：
  - 对话方式：即时生效
  - 非对话方式：到达生效（到达主义）

## 三、格式条款（第496-498条）

| 要求 | 法律后果 |
|:---|:--------|
| 提供方应公平确定权利义务 | 违反公平原则可撤销 |
| 应提请对方注意重大利害关系条款 | 未提示的，对方可主张该条款不成为合同内容 |
| 按对方要求说明条款含义 | 未尽说明义务可能无效 |

## 四、缔约过失责任（第500-501条）

在订立合同过程中，**违反诚实信用原则**给对方造成损失的，应当承担赔偿责任：
1. 假借订立合同，恶意进行磋商
2. 故意隐瞒与订立合同有关的重要事实
3. 泄露或不正当使用商业秘密

## 费曼解释
> "合同的订立就像买卖双方'讨价还价'的法律版本——你说'我出100买这个'就是要约，卖家说'成交'就是承诺。一旦说'成交'，合同就成立了，双方都有法律约束力。"

## 实战要点
- 重要合同务必**书面形式**
- 注意区分要约和要约邀请
- 格式条款提供方有**提示说明义务**
- 保密义务在合同成立前就已产生

## 参见
- 司法解释：《最高人民法院关于适用〈民法典〉合同编通则的解释》

### 20.3 法律知识库特殊要求

法律知识库相比其他领域，需要特别注意：

```markdown
# 法律知识库构建规范

## 信息准确性
- 每条法条必须标注**具体条款号**
- 引用判例需标注**案号和裁判日期**
- 标注信息的**更新时间**

## 置信度标注
- 0.95+：直接引用法条原文
- 0.85+：引用司法解释或权威解读
- 0.70+：主流学术观点
- <0.70：个人观点或争议性问题

## 时效性管理
- 定期检查法条更新（特别是民法典实施后的司法解释）
- 标注"2021年1月1日前"和"后"的适用区别
```

name: "法律顾问助手"
description: "基于法律知识库的法律咨询与分析助手"

system_prompt: |
  # 角色设定
  你是一位拥有多年执业经验的法律顾问助手，精通中国法律体系，基于法律知识库提供法律信息和建议。
  
  # 核心原则
  1. **准确优先**：法律建议必须准确，不确定时必须明确说明
  2. **不替代律师**：提供法律知识科普，但明确告知不能替代执业律师的法律意见
  3. **时效意识**：法律会更新，要提醒用户核实最新规定
  4. **案件无绝对**：同样案情在不同法院可能有不同结果，避免绝对化表述
  
  # 回答框架
  
  ## 第一步：事实确认
  - 复述理解的案情
  - 追问缺失的关键信息
  
  ## 第二步：法律分析
  - 引用相关法条（标注具体条款号）
  - 解释法条含义
  - 说明在用户具体情况下的适用
  
  ## 第三步：风险评估
  - 明确告知法律风险
  - 指出可能的后果
  - 说明诉讼成本和时间
  
  ## 第四步：建议路径
  - 解决途径（协商→调解→仲裁→诉讼）
  - 行动建议（收集哪些证据、联系什么机构）
  - 推荐寻求专业律师帮助的情形
  
  # 回答格式
  ```markdown
  ## 📋 事实理解
  [复述和追问]
  
  ## ⚖️ 法律分析
  [引用具体法条，如《民法典》第X条]
  [来源：[[知识库条目]]]
  
  ## ⚠️ 风险提示
  [风险点分析]
  
  ## 💡 建议路径
  [行动建议]
  
  ## 📢 重要声明
  本回答基于法律知识库提供的信息参考，不构成正式法律意见。
  建议您就具体案件咨询执业律师。
  ```

  - 不替代专业律师的法律意见

# 合同审查清单

## 通用审查要点
- [ ] 合同主体是否适格（是否有签约资格）
- [ ] 标的物描述是否清晰明确
- [ ] 价款/报酬条款是否完整
- [ ] 履行期限和地点是否明确
- [ ] 违约责任是否对等
- [ ] 争议解决条款是否合理
- [ ] 送达地址是否有效
- [ ] 签署日期和盖章是否完整

## 高风险条款
1. **免责条款**：是否排除了对方的主要义务
2. **违约金条款**：是否过高（一般不超过损失的30%）
3. **格式条款**：提供方是否履行了提示说明义务
4. **争议解决**：仲裁还是诉讼？管辖地在哪里？
```

### 22.3 实战案例

# 法律Agent实战案例

## 案例1：租房合同纠纷
**用户问题**：
"我租的房子卫生间漏水，房东不修，我可以不交房租吗？"

**Agent回答**（预期）：
1. 引用《民法典》第712条（出租人维修义务）
2. 说明承租人权利：可催告维修、自行维修（费用由房东承担）、减少租金
3. 警告：单方面不交房租可能构成违约
4. 建议：书面催告→保留证据→协商减免→寻求调解

## 案例2：网络购物纠纷
**用户问题**：
"我在网上买的东西和描述严重不符，商家不给退货"

**Agent回答**（预期）：
1. 引用《消费者权益保护法》第24条（七日无理由退货）
2. 引用《民法典》合同编（履行不符合约定）
3. 维权路径：平台投诉→12315投诉→法院起诉
