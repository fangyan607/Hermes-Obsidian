# 中国佛教发展史略述 · 一周学习计划

> **书名**：中国佛教发展史略述
> **作者**：南怀瑾
> **学习周期**：1周（8个番茄钟，约3.5小时）
> **学习方法**：番茄时钟 + 费曼学习法 + 刻意练习

---

## 书籍概览

### 作者背景

南怀瑾先生对佛教在中国发展历史的简明介绍，梳理佛教从传入到本土化的演变过程。

### 核心主题

本书简述佛教在中国的传播与发展，包括佛教传入、宗派形成、重要人物、思想演变等内容。

### 学习目标

完成本周学习后，你将能够：
- 了解佛教传入中国的历史
- 掌握中国佛教的主要宗派
- 认识佛教的重要人物
- 理解佛教中国化的过程

## Day 1：佛教传入

### 番茄钟1：佛教初传中国（25分钟）

#### 核心原文

> 「佛教传入中国，一说汉明帝永平年间，一说西汉末年。」

#### 南师解读要点

**1. 传入时间**

南师介绍佛教传入中国的时间争议。

**2. 传入路线**

```
丝绸之路 → 陆路传入
南海航线 → 海路传入
    ↓
西域 → 中原
```

**3. 早期特点**

南师分析早期佛教的特点——依附神仙方术。

### 番茄钟6：天台与华严（25分钟）

#### 核心原文

> 「天台智者，创一心三观；华严法藏，明事事无碍。」

#### 南师解读要点

**1. 天台宗**

南师介绍天台宗的核心思想：

```
一心三观 → 空假中
三谛圆融 → 真俗中
    ↓
中国佛教的独创体系
```

**2. 华严宗**

南师介绍华严宗的核心思想：

```
四法界 → 事无碍、理无碍、事事无碍、理事无碍
    ↓
圆融无碍的最高境界
```

**3. 中国特色**

南师指出天台、华严是中国佛教最具原创性的宗派。

> **费曼自测**
>
> 请介绍天台宗或华严宗的核心思想。
